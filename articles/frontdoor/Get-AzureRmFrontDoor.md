# Get-AzureRmFrontDoor
Abrufen der Front Door-Konfiguration

## SYNTAX

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Beschreibung
Mit dem Cmdlet **Get-AzureRmFrontDoor** werden alle vorhandenen Front Door-Instanzen im aktuellen Abonnement abgerufen, die den angegebenen Informationen entsprechen.
