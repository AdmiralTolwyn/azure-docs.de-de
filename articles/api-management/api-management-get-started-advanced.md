# Erste Schritte mit der erweiterten Azure API Management-Konfiguration

Die Lernprogrammen in diesem Bereich beschreiben erweiterte Lösungen und Aufgaben für die API Management-Konfiguration.

-	[Anpassen des Entwicklerportals in Azure API Management][] – Dieses Lernprogramm beschreibt, wie Sie Aussehen und Verhalten des Entwicklerportals an Ihre Marke anpassen können.
-	[Verfolgung von Aufrufen in Azure API Management mit dem API-Inspektor][] – Dieses Lernprogramm beschreibt die Verfolgung und Informationssammlung für Operationsaufrufe mithilfe des API-Inspektors.
-	[Schützen Ihrer API mit Durchsatzgrenzen unter Verwendung von Azure API Management][] – Dieses Lernprogramm zeigt, wie Sie Durchsatzgrenzen und Kontingentrichtlinien konfigurieren. Außerdem wird anhand einer Demonstration gezeigt, wie Sie Vorgänge auslösen, bis die Durchsatzgrenze erreicht ist.
-	[Delegieren von Benutzerregistrierung und Produktabonnierung][] – Dieses Lernprogramm beschreibt, wie Sie Anmeldung, Registrierung und Produktabonnierung von Entwicklern mit Ihrer vorhandenen Website umsetzen können, anstatt die integrierte Funktion im Entwicklerportal zu verwenden.
