<h1 id="vnettut1">Azure Active Directory Connect</h1>

Der Azure Active Directory Connect-Assistent stellt das einzige Tool und die Anleitung zum Verbinden Ihrer lokalen Verzeichnisse mit Azure Active Directory dar.  Der Assistent implementiert und konfiguriert alle Komponenten, die erforderlich sind, um Ihre Verzeichnisintegration einzurichten und auszuführen, einschließlich Synchronisierungsdienste, Kennwortsynchronisierung oder Active Directory-Verbunddienste (AD FS) sowie Voraussetzungen wie das Azure AD PowerShell-Modul.

>[AZURE.NOTE] **Azure Active Directory Connect umfasst Funktionen, die zuvor als "DirSync" und "AAD Sync" freigegeben wurden. Diese Tools werden zukünftig nicht mehr einzeln veröffentlicht.** 

> **Azure Active Directory Connect ist jetzt Ihre zentrale Anlaufstelle für die Synchronisierung, das einmalige Anmelden und alle anderen Aspekte der lokalen Azure AD-Integration.**


Wenn Sie das Azure Active Directory-Synchronisierungstool (DirSync), Azure Active Directory-Synchronisierungsdienste (AAD Sync) oder Forefront Identity Manager 2010 R2 testen oder verwenden, finden Sie weitere Informationen unter [Verzeichnisintegrationstools](http://msdn.microsoft.com/library/azure/dn757582.aspx "Directory Integration Tools").


## Azure Active Directory Connect - Öffentliche Vorschau 

Die aktuelle öffentliche Vorschau von Azure Active Directory Connect bietet eine Anleitung zum Integrieren mindestens einer Windows Server Active Directory-Gesamtstrukturen mit Azure AD. 

[Herunterladen der öffentlichen Vorschau von Azure AD Connect](http://connect.microsoft.com/site1164/program8612 "Azure Active Directory Connect")

Mit dieser Vorschauversion von Azure Active Directory Connect haben Sie folgende Möglichkeiten: 

- Wählen Sie die Express-Einstellungen für eine schnelle und einfache Konfiguration einer einzelnen Gesamtstruktur mit vier Klicks.
- Wählen Sie benutzerdefinierte Einstellungen aus, um mehrere Gesamtstrukturen zu konfigurieren oder AD FS für einmaliges Anmelden (SSO) auszuwählen.
- Konfigurieren Sie zusätzliche Synchronisierungsoptionen wie den Exchange-Hybrid-Modus, das Zurückschreiben von Kennwörtern und alternative ID-Attribute.
