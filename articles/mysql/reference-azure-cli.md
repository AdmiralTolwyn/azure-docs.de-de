# <a name="commands"></a>Befehle
|||
|---|---|
|az mysql server|Verwaltet Server. Server umfassen Datenbanken, Data Warehouses und elastische Pools.|
|az mysql server create|Erstellt einen MySQL-Server.|
|az mysql server delete|Löscht einen MySQL-Server.|
|az mysql server list|Ruft eine Liste aller MySQL-Server in einer Ressourcengruppe ab.|
|az mysql server show|Zeigt Details zu einem MySQL-Server an.|
|az mysql server update|Aktualisiert einen Server.|
|az mysql server restore|Stellt einen Server auf den Zustand zu einem früheren Zeitpunkt wieder her.|
|az mysql server firewall-rule|Verwaltet die Firewallregeln eines Servers.|
|az mysql server firewall-rule create|Erstellt eine Firewallregel.|
|az mysql server firewall-rule delete|Löscht eine Firewallregel.|
|az mysql server firewall-rule list|Listet die Firewallregeln auf.|
|az mysql server firewall-rule show|Zeigt die Details einer Firewallregel.|
|az mysql server firewall-rule update|Aktualisiert eine Firewallregel.|
|az mysql db|Verwaltet Datenbanken.|
|az mysql db list|Listet alle MySQL-Datenbanken auf einem Server auf.|
|az mysql server-logs list|Listet die Protokolldateien für einen Server auf.|
|az mysql server-logs download|Lädt Protokolle vom Server auf den lokalen Computer herunter.|
|az mysql server configuration list|Listet Serverkonfigurationsparameter auf.|
|az mysql server configuration show|Zeigt Details zu Serverkonfigurationsparametern an.|
|az mysql server configuration update|Ändert den Wert von Serverkonfigurationsparametern.|
|||
