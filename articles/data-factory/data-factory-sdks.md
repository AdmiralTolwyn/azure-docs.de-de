# Azure Data Factory SDKs

Sie können die Factorys mithilfe des Azure-Portals, von Azure PowerShell, der Klassenbibliothek von .NET oder REST-API erstellen, überwachen und verwalten.

Methode | Resource Location | Entwicklerreferenzen
---------------------------------------------------- | ------------------------------ | -----------
Azure-Portal | [https://portal.azure.com/](https://portal.azure.com) | 
Azure PowerShell | Die neueste [Azure PowerShell](http://go.microsoft.com/?linkid=9811175&clcid=0x409)-Version herunterladen | [Cmdlet-Referenz](https://msdn.microsoft.com/library/dn820234.aspx)<p>[Referenz zur JSON-Skripterstellung](https://msdn.microsoft.com/library/azure/dn835050.aspx) </p> 
Klassenbibliothek von .NET | Das Azure Data Factory .NET SDK dient zum Erstellen, Überwachen und Verwalten von Azure Data Factorys und Erweitern einer Data Factory mithilfe einer .NET-Aktivität. In den Artikeln [Verwenden von benutzerdefinierten Aktivitäten in einer Azure Data Factory-Pipeline](data-factory-use-custom-activities.md) und [Erstellen, Überwachen und Verwalten von Azure Data Factorys mithilfe des Data Factory .NET SDK](data-factory-create-data-factories-programmatically.md) finden Sie Hilfe für den Einstieg.<p><b>Das neueste Nuget-Paket herunterladen</b><br/>Sie können das neueste Azure Data Factory Management Library Nuget-Paket hier herunterladen: [https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/](https://www.nuget.org/packages/Microsoft.Azure.Management.DataFactories/)</p><p>**Verwenden der Paket-Manager-Konsole in Visual Studio**<br/>Sie können den folgenden Befehl in der Paket-Manager-Konsole in Visual Studio ausführen, um die neueste Azure Data Factory Management Library zu erhalten</p>Install-Package Microsoft.Azure.Management.DataFactories</p> | [Referenz zum .NET SDK](https://msdn.microsoft.com/library/dn883654.aspx)
REST-API | Zum Erstellen, Überwachen und Verwalten von Azure Data Factorys können Sie die Data Factory-REST-API verwenden. | [REST-API-Referenz](https://msdn.microsoft.com/library/dn906738.aspx)
