# <a name="azure-cosmos-db-sql-api-documentation"></a>Azure Cosmos DB: SQL-API-Dokumentation

Dieser Artikel enthält Links zu allen Azure Cosmos DB-Inhalten, die für die SQL-API gelten.

Diese Artikel gelten nicht für die Graph-, Tabellen- oder MongoDB-API.

## <a name="quickstarts"></a>Schnellstarts

Anhand der Schnellstartthemen können Sie besonders schnell eine funktionierende Anwendung mit Azure Cosmos DB erstellen. In jeder Schnellstartanleitung erfahren Sie, wie Sie mithilfe des benutzeroberflächenbasierten Azure-Portals und Ihrer bevorzugten Programmiersprache Datenbanklösungen mit Azure Cosmos DB erstellen. Für jede Schnellstartanleitung sind klonbare Web-Apps in GitHub verfügbar. 

- [Azure Cosmos DB: Erstellen einer Web-App mit einer DocumentDB-API mit .NET und dem Azure-Portal](create-sql-api-dotnet.md)
- [Azure Cosmos DB: Erstellen einer Dokumentdatenbank mit Java und dem Azure-Portal](create-sql-api-java.md)
- [Azure Cosmos DB: Erstellen einer Web-App der DocumentDB-API mit Node.js und dem Azure-Portal](create-sql-api-nodejs.md)
- [Azure Cosmos DB: Erstellen einer Web-App von DocumentDB-API mit Python und dem Azure-Portal](create-sql-api-python.md)
- [Azure Cosmos DB: Erstellen einer Web-App mit .NET, Xamarin und Authentifizierung mit Facebook](create-sql-api-xamarin-dotnet.md)
