# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualisieren Sie Ihre Diagrammdaten in Azure Cosmos DB Gremlin API mit Datenvisualisierungslösungen

Die folgenden Lösungen werden von der [Apache Tinkerpop Community](http://tinkerpop.apache.org/#poweredby) für die Gremlin-Grafikvisualisierung empfohlen.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Linkurious-Logo](./media/graph-visualization/linkurious-logo.jpg)

Linkurious Enterprise nutzt Grafiktechnologie und Datenvisualisierung, um komplexe Datensätze in interaktive visuelle Netzwerke zu verwandeln. Die Plattform verbindet sich mit Ihren Datenquellen und ermöglicht es den Ermittlern, nahtlos über Milliarden von Einheiten und Beziehungen zu navigieren. Das Ergebnis ist eine neue Fähigkeit, verdächtige Beziehungen zu erkennen, ohne mit Abfragen oder Tabellen zu jonglieren.

Die interaktive Visualisierungsoberfläche von Linkurious Enterprise bietet eine einfache Möglichkeit, komplexe zusammenhängende Daten zu untersuchen. Suchen Sie nach bestimmten Entitäten, erweitern Sie Verbindungen, um versteckte Beziehungen aufzudecken, und wenden Sie Layouts Ihrer Wahl an, um komplexe Netzwerke zu entwirren. Linkurious ist jetzt mit Azure Cosmos DB Gremlin-API kompatibel. Es eignet sich für End-to-End-Grafikvisualisierungsszenarien und unterstützt Lese- und Schreibfunktionen über die Benutzeroberfläche. 

![Linkurious-Demo](./media/graph-visualization/linkurious-demo.gif)

**Nächste Schritte:** Sie können eine [Demo von Linkurious mit Azure Cosmos DB anfordern](https://linkurio.us/contact/).

## <a name="cambridge-intelligence"></a>Cambridge Intelligence

![Cambridge Intelligence Logo](./media/graph-visualization/ci-logo.png)


Die Diagrammvisualisierungs-Toolkits von Cambridge Intelligence unterstützen nun Azure Cosmos DB:
- [KeyLines für JavaScript-Entwickler](https://cambridge-intelligence.com/keylines/) 
- [Re-Graph für React-Entwickler](https://cambridge-intelligence.com/regraph/)

![Cambridge Intelligence-Demo](./media/graph-visualization/ci-demo-2.gif)

Mit den Toolkits können Sie leistungsstarke Anwendungen zur Visualisierung und Analyse von Graphen für Ihren Anwendungsfall entwickeln. Sie nutzen leistungsstarkes WebGL-Rendering und sorgfältig erstellten Code, um den Benutzern eine schnelle und aufschlussreiche Visualisierungserfahrung zu bieten.

Sie sind mit jedem Browser, Gerät, Server oder jeder Datenbank kompatibel und werden mit Schritt-für-Schritt-Anleitungen, vollständig dokumentierten APIs und interaktiven Demos geliefert.
