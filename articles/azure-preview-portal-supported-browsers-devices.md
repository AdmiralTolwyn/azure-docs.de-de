# <a name="supported-browsers-and-devices-for-the-azure-portal"></a>Unterstützte Browser und Geräte für das Azure-Portal
Das [Azure-Portal](https://portal.azure.com) kann auf allen modernen Desktop- und Tabletgeräten und mit allen Browsern ausgeführt werden.

### <a name="supported-devices"></a>Unterstützte Geräte
Das Azure-Portal funktioniert einwandfrei auf modernen PCs, Macs und Tablets. Wenn Sie Ihre Azure-Ressourcen über ein mobiles Gerät verwalten möchten, testen Sie die mobile Azure-App, die für iPhone und Android verfügbar ist. Weitere Informationen finden Sie unter [Azure Mobile App](https://azure.microsoft.com/features/azure-portal/mobile-app/).

### <a name="supported-browsers"></a>Unterstützte Browser
Es wird empfohlen, den neuesten Browser zu verwenden, der mit Ihrem Betriebssystem kompatibel ist. Die folgenden Browser werden unterstützt:

* Microsoft Edge (neueste Version)
* Internet Explorer 11
* Safari (neueste Version, nur auf Mac)
* Chrome (neueste Version)
* Firefox (neueste Version)
