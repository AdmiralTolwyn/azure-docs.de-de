Bevor Sie das Beispiel erstellen, müssen Sie die [Anomalieerkennungs-API](https://labs.cognitive.microsoft.com/en-us/project-anomaly-finder) abonnieren, die Teil der Microsoft Cognitive Services ist. Klicken Sie auf die Schaltfläche „Subscribe“ (Abonnieren), und melden Sie sich dann mit Ihrem Microsoft-Konto (Hotmail usw.) an, um die API zu abonnieren. Anschließend finden Sie Ihre kostenlosen Abonnementschlüssel [hier](https://labs.cognitive.microsoft.com/en-us/subscriptions?productId=/products/AnomalyDetection.Preview&source=labs). Sowohl Schlüssel 1 als auch Schlüssel 2 kann verwendet werden.
