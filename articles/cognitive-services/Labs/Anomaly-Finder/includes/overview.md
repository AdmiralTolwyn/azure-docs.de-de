## <a name="overview"></a>Übersicht
Mithilfe der API für die Suche nach Anomalien können Sie Daten im Zeitverlauf überwachen und Anomalien mit dem Machine Learning-Dienst ermitteln, der sich durch die automatische Anwendung des richtigen Statistikmodells an Ihre Daten anpasst. Diese API ermöglicht es Organisationen, die Datenqualität zu gewährleisten, zuverlässige Dienste bereitzustellen, Vorfälle im Unternehmen zu erkennen und die Unternehmensstrategie zu optimieren, ohne dass dabei unnormale Daten mühsam manuell ermittelt werden müssen.
