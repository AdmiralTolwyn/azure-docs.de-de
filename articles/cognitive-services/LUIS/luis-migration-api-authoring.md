# <a name="authoring-api-changes"></a>Änderungen in der Erstellungs-API 

Die Erstellungs-APIs werden für V3 aktualisiert, um die [Modellzerlegung mit durch maschinelles Lernen erworbenen Entitäten](luis-concept-data-extraction.md) zu ermöglichen. 

Sie können in den V2- oder Vorschau-V3-APIs erstellen. Dieselbe App kann auch wahlweise V2- oder V3-Vorhersageendpunkt-Antworten verwenden. Sie müssen nicht dieselbe API-Version für Erstellung und Vorhersageendpunkt-Antwort verwenden.

Das LUIS-Portal verwendet exklusiv die V3-Erstellungs-APIs.

## <a name="next-steps"></a>Nächste Schritte
