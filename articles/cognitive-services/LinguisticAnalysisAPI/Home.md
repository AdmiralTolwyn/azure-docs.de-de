# <a name="what-is-the-linguistic-analysis-api-preview"></a>Worum handelt es sich bei der API für linguistische Analyse?  (Vorschau)

> [!IMPORTANT]
> Die Vorschauversion für die linguistische Analyse wurde am 9. August 2018 außer Betrieb genommen. Es wird empfohlen, [Azure Machine Learning-Textanalysemodule](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics) für die Textverarbeitung und -analyse zu verwenden.

Die API für linguistische Analyse ermöglicht den Zugriff auf Tools für die Verarbeitung von natürlicher Sprache (Natural Language Processing, NLP), mit denen die Struktur von Text identifiziert wird. Die aktuelle Version verfügt über drei Analysearten:

- [Trennung und Tokenisierung von Sätzen](Sentences-and-Tokens.md)
- [Satzteilmarkierung](POS-tagging.md)
- [Konstituentenanalyse](Constituency-Parsing.md)

## <a name="getting-started"></a>Erste Schritte

Nachdem Sie sich mit den oben aufgeführten Hauptarten der Analyse vertraut gemacht haben, können Sie sich über den Endpunkt für das [Abrufen der Liste mit den verfügbaren Analysetools](AnalyzersMethod.md) und deren [Benennung](Analyzer-Names.md) und dann über den Endpunkt für das [Analysieren der neuen Eingabe](AnalyzeMethod.md) informieren.
