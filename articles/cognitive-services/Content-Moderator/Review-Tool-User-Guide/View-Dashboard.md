# <a name="view-dashboard"></a>Dashboards anzeigen

Klicken Sie auf die Registerkarte **Dashboard**, um wichtige Metriken für alle Inhaltsprüfungen anzuzeigen, die im Tool durchgeführt wurden. Es werden die gesamten, abgeschlossenen und laufenden Prüfungen für Bilder und Text angezeigt. Sie können Metriken für Ihr Standardteam und Ihre untergeordneten Teams anzeigen.
