# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Registrieren für die Sprachübersetzungs-API

Führen Sie diese Schritte aus, um sich für die Sprachübersetzungs-API zu registrieren.

## <a name="1-sign-into-azure"></a>1: Anmelden an Azure

- Sie haben kein Konto? [Registrieren Sie sich für ein Microsoft Azure-Konto](http://azure.com/).
- Sie verfügen bereits über ein Konto? [Anmelden](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: Abonnieren von Microsoft Translator

- Navigieren Sie nach der Anmeldung bei Azure zum Abschnitt [Azure Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices).
- Wählen Sie unter „API-Typ“ die Option „Sprachübersetzungs-API“.
- Wählen Sie im Abschnitt „Tarif“ den Tarif aus, der Ihren Anforderungen am besten entspricht.
    - Jedes Abonnement verfügt über einen Free-Tarif. Der Free-Tarif bietet dieselben Features und Funktionen wie die kostenpflichtigen Tarife und hat kein Ablaufdatum.
    - Sie können nur ein kostenloses Abonnement für Ihr Konto verwenden.
- Füllen Sie die restlichen Felder des Formulars aus, und wählen Sie die Schaltfläche „Erstellen“.
- Das Abonnement ist sofort einsatzbereit.

## <a name="3-authentication-key"></a>3: Authentifizierungsschlüssel

- Rufen Sie Ihren Authentifizierungsschlüssel ab, indem Sie im Menü „Alle Ressourcen“ auf Ihr Abonnement klicken.
- Der Schlüsselwert wird für die Authentifizierung verwendet. Geben Sie ihn daher nicht an andere Personen weiter. Sie benötigen den Schlüsselwert für die Entwicklung Ihrer App.
