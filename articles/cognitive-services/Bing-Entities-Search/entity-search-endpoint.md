# <a name="entity-search-endpoints"></a>Entität Suche Endpunkte
Die **Entität-Suchdienst-API** einen Endpunkt enthält.

## <a name="endpoint"></a>Endpunkt
Um Suchergebnisse Entität anfordern, wird senden Sie eine Anforderung an den folgenden Endpunkt. Verwenden Sie die Header und die URL-Parameter, um Spezifikationen genauer zu definieren.

Endpunkt `GET`: 
``` 
https://api.cognitive.microsoft.com/bing/v7.0/entities
```

Die folgende URL-Parameter sind erforderlich:
- mkt. Dem Markt, in dem die Ergebnisse stammen. 
- q. Die Suchabfrage für die Entität.
