# <a name="share-your-custom-search-instance"></a>Freigeben der Instanz für die benutzerdefinierte Suche
Sie können Ihre Instanz für die benutzerdefinierte Suche für alle Benutzer freigeben, indem Sie einfach die entsprechende E-Mail-Adresse verwenden. Durch die Freigabe können andere Benutzer Ihre Custom Search-Instanzen ändern. Melden Sie sich für [Custom Search](https://customsearch.ai) an, wählen Sie eine Custom Search-Instanz aus, und klicken Sie auf das Symbol „Einstellungen“ (Zahnrad). Geben Sie im Abschnitt **Share Your Instance** (Instanz freigeben) die E-Mail-Adresse der Person ein, für die Sie die Instanz freigeben möchten, und klicken Sie auf **Share** (Freigeben). Die E-Mail-Adresse wird in der Liste mit dem Namen **Instance shared with** (Instanz freigegeben für) angezeigt. Wiederholen Sie dies für jede Person, für die Sie Ihre benutzerdefinierte Suche freigeben möchten. 

Wenn Sie der Liste die E-Mail-Adresse einer Person hinzufügen möchten, muss diese Person nicht über ein Custom Search-Konto verfügen. Sie muss sich für Custom Search registrieren, bevor sie Konfigurationsänderungen vornehmen kann. Nachdem Sie eine Instanz freigegeben haben, wird sie für die betreffende Person in der Liste mit den Custom Search-Instanzen angezeigt. Eine Instanz kann nur von jeweils einer Person geändert werden. Wenn Sie versuchen, eine Instanz zu ändern, die bereits von einem anderen Benutzer bearbeitet wird, wird eine Warnung angezeigt. Eine Instanz kann für maximal zehn Benutzer freigegeben werden.

## <a name="stop-sharing"></a>Beenden der Freigabe
Wenn Sie die Freigabe einer Instanz für eine Person beenden möchten, können Sie das Symbol „Entfernen“ verwenden, um die E-Mail-Adresse aus der Liste zu entfernen.
