# <a name="open-source-big-data-applications-that-work-with-azure-data-lake-store"></a>Open Source-Big Data-Anwendungen, die mit Azure Data Lake Store funktionieren
In diesem Artikel werden Open Source-Big Data-Anwendungen aufgelistet, die mit Azure Data Lake-Speicher funktionieren. Für die Anwendungen in der Tabelle unten werden nur die Versionen unterstützt, die mit der aufgelisteten Verteilung verfügbar sind. Weitere Informationen dazu, welche Versionen dieser Anwendungen mit HDInsight zur Verfügung stehen, finden Sie unter [HDInsight-Komponentenversionen](../hdinsight/hdinsight-component-versioning.md).

| Open Source-Software | Verteilung |
| --- | --- |
| [Apache Sqoop](http://sqoop.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [MapReduce](http://hadoop.apache.org/docs/r1.0.4/mapred_tutorial.html) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Storm](https://storm.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Hive](http://hive.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [HCatalog](https://cwiki.apache.org/confluence/display/Hive/HCatalog) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Mahout](http://mahout.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Pig/Pig Latin](http://pig.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Oozie](http://oozie.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Zookeeper](http://zookeeper.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Tez](http://tez.apache.org/) |HDInsight 3.2, 3.4 und 3.5 |
| [Apache Spark](http://spark.apache.org/) |HDInsight 3.4 und 3.5 |
