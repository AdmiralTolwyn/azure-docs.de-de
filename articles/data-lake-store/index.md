# <a name="data-lake-store-documentation"></a>Dokumentation zu Data Lake Store

Azure Data Lake Store ist ein riesiges Repository für Big Data-Analyseworkloads. Azure Data Lake bietet Ihnen die Möglichkeit, Daten von beliebiger Größe, Art und Erfassungsgeschwindigkeit zur Durchführung operativer und explorativer Analysen an einem einzigen Ort zu erfassen.  Auf Azure Data Lake Store kann über Hadoop (verfügbar mit HDInsight-Clustern) mithilfe der WebHDFS-kompatiblen Szenarien zugegriffen werden. Es bietet umfassende professionelle Funktionen für Sicherheit, Verwaltbarkeit, Skalierbarkeit, Zuverlässigkeit und Verfügbarkeit.
