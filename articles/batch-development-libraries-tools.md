# Entwicklungsbibliotheken und Tools für Azure Batch 
<p> Erhalten Sie diese Bibliotheken und Tools zum Entwickeln von Azure Batch- und Batch Apps-Anwendungen.

+ [Batch]
+ [Batch Apps]
+ [Zusätzliche Ressourcen]

## Batch
 
+ [Batch-Clientbibliothek für .NET](http://www.nuget.org/packages/Azure.Batch/) (NuGet) - Entwickeln von Clientanwendungen zum Ausführen von Aufträgen mit dem Batch-Dienst
+ [Batch-Verwaltungsbibliothek](http://www.nuget.org/packages/Microsoft.Azure.Management.Batch/) (NuGet) - Verwalten von Batch-Dienstkonten
+ [Batch-Explorer](https://code.msdn.microsoft.com/windowsazure/Azure-Batch-Explorer-c1d37768) (MSDN) - Erstellen einer GUI-Anwendung zum Anzeigen und Verwalten von Batch-Aufträgen

## Batch Apps

+ [Cloud-SKD für Batch Apps](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps.Cloud/) (NuGet) - Auslagern von Aufträgen an den Batch-Dienst durch Anwendungen ermöglichen
+ [Visual Studio-Erweiterung Batch Apps](https://visualstudiogallery.msdn.microsoft.com/8b294850-a0a5-43b0-acde-57a07f17826a) (Visual Studio Gallery) - Aktivieren von Anwendungen in Visual Studio für Clouds mithilfe des Cloud-SDKs für Batch Apps
+ [Client-SDK für Batch Apps](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps/) (NuGet) - Interagieren mit Anwendungen, denen das Auslagern von Aufträgen an den Batch-Dienst ermöglicht wurde
