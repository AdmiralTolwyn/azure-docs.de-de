# <a name="use-the-storsimple-device-manager-service-to-log-a-support-request-for-the-storsimple-virtual-array"></a>Verwenden Sie den StorSimple-Geräte-Manager-Dienst, um eine Supportanfrage für das virtuelle StorSimple-Array zu protokollieren

## <a name="overview"></a>Übersicht

Der StorSimple-Geräte-Manager bietet die Möglichkeit, **protokollieren eine neuen Supportanfrage** innerhalb der Zusammenfassung Blatt "Dienst". In diesem Artikel wird erläutert, wie Sie eine neue Supportanfrage zu protokollieren und Verwalten des Lebenszyklus von innerhalb des Portals.

## <a name="new-support-request"></a>Neue Supportanfrage

Abhängig von Ihrer [Supportplan](https://azure.microsoft.com/support/plans/), können Sie Support-Tickets für ein Problem für Ihr StorSimple Virtual Array direkt über die Zusammenfassung-Blatt für die StorSimple-Geräte-Manager-Dienst erstellen.

#### <a name="to-log-a-new-request"></a>Um eine neue Anforderung zu protokollieren.

1. Wechseln Sie zu Ihrer StorSimple-Geräte-Manager-Dienst. Wechseln Sie in die Zusammenfassung Blade-diensteinstellungen zu **Unterstützung + Problembehandlung** Abschnitt, und klicken Sie dann auf **neu unterstützen Anforderung**.
   
    ![Neue Supportanfrage](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket1.png)

2. In der **Grundlagen** Blatt, gehen Sie folgendermaßen vor:

    1. Aus der **ausstellen Typ** Dropdownliste **technische**. 
    
    2. Die aktuelle **Abonnement**, **Service** Typ, und die **Ressource** (StorSimple-Geräte-Manager-Dienst) werden automatisch ausgewählt. 

    3. Geben Sie ein oder mehrere Geräte registriert werden, mit dem Dienst, bei denen Probleme aufgetreten sind.

    4. Wählen Sie ein geeignetes **Supportplan** mehrere Pläne, die Ihrem Abonnement zugeordnet ist. Sie benötigen einen kostenpflichtigen Supportplan, um technischen Support zu aktivieren.

3. In **Schritt2**, wählen Sie die **Schweregrad** und angeben, wenn das Problem auf das Array oder den StorSimple-Geräte-Manager-Dienst verknüpft ist. Außerdem wählen Sie eine **Kategorie** dafür geben, und bieten eine höhere **Details** zu diesem Problem.
   
    ![Neue Supportanfrage](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket2.png)

4. In **Schritt 3**, geben Sie Ihre Kontaktinformationen. Microsoft-Support verwendet diese Informationen, um Sie für Weitere Informationen zu erhalten, Diagnose und Behebung um eine Verbindung.
   
    ![Neue Supportanfrage](./media/storsimple-virtual-array-log-support-ticket/log-support-ticket3.png)

## <a name="manage-a-support-request"></a>Verwalten Sie eine Supportanfrage

Nach dem Erstellen ein Support-Ticket, können Sie den Lebenszyklus des Tickets aus im Portal verwalten.

#### <a name="to-manage-your-support-requests"></a>Zum Verwalten Ihrer Supportanfragen

Um zur Seite Hilfe und Support zu erhalten, navigieren Sie zu **Durchsuchen > Hilfe und Support**.

![Verwalten von Supportanfragen](./media/storsimple-virtual-array-log-support-ticket/manage-support-tickets.png)
