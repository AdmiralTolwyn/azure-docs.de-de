# Was ist Azure Active Directory?

Azure Active Directory ist ein Dienst, mit dem Sie ganz einfach alle Ihre Anforderungen an die Identitäts- und Zugriffsverwaltung erreichen können. Mit Azure AD können Sie ein privates Identitätsverzeichnis in der Cloud erstellen. Auf ähnliche Weise wie bei Windows Server Active Directory können Sie mit Azure AD den Zugriff auf Anwendungen und Ressourcen zentral steuern. Sie können problemlos vorhandene Ressourcen hinzufügen (Cloud-Dienste oder lokale Anwendungen) sowie von Ihnen entwickelte Anwendungen integrieren. Mit Azure AD erhalten Ihre Benutzer Zugriff durch einmalige Anmeldung auf alle ihre Anwendungen, wodurch sichergestellt wird, dass sie nicht wissen müssen, wo sich die einzelnen Apps befinden. Zudem müssen sie sich kein separates Kennwort merken.

Azure AD kann als eigenständiges Cloud-Verzeichnis oder zur Erweiterung der lokalen Identitäts- und Zugriffsverwaltunglösungen in der Cloud verwendet werden. In diesem Fall können Sie weiterhin von Ihren vorhandenen Investitionen und lokalen Funktionen profitieren, während Sie gleichzeitig Azure AD zum Erreichen der Identitäts- und Zugriffsverwaltung in der Cloud nutzen. [Weitere Informationen](http://msdn.microsoft.com/library/jj573653).

Darüber hinaus können Sie Regeln und Richtlinien festlegen, die steuern, wer Zugriff hat und unter welchen Bedingungen dieser Zugriff gestattet wird. Sie können z. B. die Multi-Factor Authentication (MFA) erfordern und den Zugriff auf Basis des Geräts oder des Speicherorts verwalten. [Weitere Informationen zu Azure-MFA](http://azure.microsoft.com/services/multi-factor-authentication/)

Alle diese Regeln und Richtlinien, einschließlich der MFA, können für den Zugriff auf Cloud-Anwendungen oder lokale Legacy-Anwendungen erzwungen werden, wobei der integrierte Anwendungsproxy verwendet wird. Dabei ist nicht erforderlich, dass die Anwendungen umgeschrieben oder direkt im Internet verfügbar gemacht werden müssen. [Erfahren Sie mehr über den Azure AD-Anwendungsproxy](https://msdn.microsoft.com/library/azure/dn768219.aspx).

Schließlich gibt Ihnen Azure AD Informationen über die Abläufe in Ihrer Organisation an die Hand. Mithilfe der erweiterten Berichterstellung und durch Analysen erhalten Sie eindeutige Informationen über den Zugriff Ihrer Benutzer. Mithilfe der Anwendungsermittlung können Sie z. B. herausfinden, welche Anwendungen in Ihrer Organisation aktiv verwendet werden. [Erfahren Sie mehr über Azure AD Cloud App Discovery](https://appdiscovery.azure.com/).

Da es sich um einen umfassenden Dienst handelt, bietet Azure AD unterschiedliche Vorteile für die verschiedenen Personen innerhalb einer Organisation:

- Entscheidungsträger können mit Azure AD eine mobile Belegschaft erreichen und die Leistungsfähigkeit der Cloud-Anwendungen mit der Gewissheit nutzen, dass die Governance-Anforderungen erfüllt werden.
- Dienstanbieter können mit Azure AD problemlos die Identitäts- und Verwaltungsanforderungen bewältigen, die die Dienste mit den vorhandenen Identitätslösungen der Kunden verbinden, während der Zugriff auf Microsoft Azure- und Office 365-Kunden ermöglicht wird. Azure AD ist Ihre zentrale Anlaufstelle, um es Unternehmen und Kunden zu ermöglichen, Ihre Lösung problemlos zu übernehmen. Azure AD kann auch alle Anforderungen an den Backoffice-Zugriff bewältigen. Sie können sicher sein, dass die richtigen Personen sowohl intern als auch extern über die richtigen Zugriffsrechte verfügen.
- Für IT-Experten verbessert Azure AD die Steuerung und Sichtbarkeit von Vorgängen, während die Geschwindigkeit und Flexibilität der Cloud erreicht wird. Mit Azure AD können Sie ermitteln, was die Personen verwenden und es stärkt sie durch den Self-Service.

Das beste ist, dass mit Azure AD alles vorab integriert ist. Multi-Factor Authentication (MFA), SaaS-Anwendungen, Hybrid-Tools und Self-Service-Funktionen sind einsatzbereit. Administratoren haben für die umfassende Verwaltung Zugriff auf das Azure AD-Verwaltungsportal und Windows PowerShell. Entwickler verfügen über einen konsistenten Satz von RESTful-APIs und über den einfachen Zugriff auf die Veröffentlichung und Nutzung von Anwendungsschnittstellen. Endbenutzer erhalten eine einfache Benutzeroberfläche. Zudem erhalten sie Profile, Anwendungen und Funktionen zum zentralen Verwalten des Zugriffs auf Ressourcen, ohne spezielle Schulungen zu erfordern.

Azure AD weist eine Reihe von Funktionsebenen auf, die von einem Cloud-Verzeichnis mit Anwendungszugriff bis zu einer umfassenden, regelgesteuerten und verwalteten Self-Service-Lösung reichen. Azure AD ist für alle Benutzer kostenlos. Registrieren Sie sich jetzt für eine Azure-Testversion. Ihr Verzeichnis wartet bereits auf Sie. Das Verzeichnis gehört Ihnen, unabhängig davon, ob Sie etwas kaufen. Aktivieren Sie Ihre Azure AD Premium-Testversion, um die hier beschriebenen Funktionen in vollem Umfang zu nutzen. [Erfahren Sie mehr über die Azure AD Editionen](https://msdn.microsoft.com/library/azure/dn532272.aspx).
