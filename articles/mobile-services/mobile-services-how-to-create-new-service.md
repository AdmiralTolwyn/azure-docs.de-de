# Erstellen eines neuen mobilen Diensts

In diesem Thema erfahren Sie, wie Sie mit dem [Azure-Verwaltungsportal][Management Portal] ein neues JavaScript-Back-End in Azure Mobile Services erstellen, das JavaScript für die serverseitige Geschäftslogik verwendet. Informationen zum Erstellen des Back-Ends für den mobilen Dienst sowie einer mobilen Client-App, die den neuen mobilen Dienst nutzt, finden Sie unter [Erste Schritte mit Mobile Services].

>[AZURE.NOTE]Sie benötigen ein Azure-Konto mit aktivierter "Azure Mobile Services"-Funktion, um dieses Lernprogramm auszuführen. Wenn Sie noch kein Konto haben, können Sie in nur wenigen Minuten ein kostenloses Testkonto erstellen. Einzelheiten finden Sie unter <a href="http://www.windowsazure.com/pricing/free-trial/?WT.mc_id=A30A4DDE2&amp;returnurl=http%3A%2F%2FFde-de%2Fdocumentation%2Farticles%2Fmobile-services-how-to-create-new-service%2F" target="_blank">Kostenlose Azure-Testversion</a>.
