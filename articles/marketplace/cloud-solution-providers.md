# <a name="cloud-solution-providers"></a>Cloud Solution Provider (CSP)

Softwareangebote können, zusätzlich zur öffentlichen Verfügbarkeit von Angeboten über die [Microsoft-Webstorefronts](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace), Millionen von berechtigten Microsoft-Kunden erreichen, die von Partnern des Cloud Solution Provider-Programms (CSP-Programm) betreut werden.

Verleger konfigurieren Angebote für die Verfügbarkeit im CSP-Programm auf Abonnementbasis (Standard ist ausgeschlossen) für ein neues oder vorhandenes Angebot, sodass Partner Ihre Produkte verkaufen und gebündelte Lösungen für Kunden erstellen können.

Die Herausgeber sind verantwortlich für die Bereitstellung des Supports zum Beheben von Problemen für den Endkunden. Außerdem müssen sie einen Mechanismus für CSP-Partner und/oder Kunden zur Kontaktaufnahme mit dem Support bereitstellen. Es hat sich bewährt, den CSP-Partnern Benutzerdokumentationen, Schulungen und Benachrichtigungen über Dienstintegrität/Dienstausfall (wie zutreffend) bereitzustellen, sodass die CSP-Partner für die Behandlung von Supportanforderungen der Ebene 1 von Kunden ausgerüstet sind.

Die folgende Tabelle enthält die Angebotsarten und Preismodelle, für die das CSP-Programm abonniert werden kann:

| **Angebotstyp**    | **BYOL**  |  **Free** | **Nutzungsbasierte Zahlung**   |
| :---------------- | :---------|:----------|:-----------|
| VMs  | Ja | Ja | Ja |
| Container | – | Ja | – |
| Lösungsvorlagen | – | Ja | – |
| Verwaltete Anwendungen | – | Ja | Ja |
| Software-as-a-Service (SaaS) | – | Ja | Ja |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>Konfigurieren ein Angebots

Die „Abonnieren“-Einstellung für das CSP-Programm wird im Partner Center oder im Cloud-Partnerportal im Bereich für die Erstellung von Angeboten konfiguriert. ([Erfahren Sie mehr über den Wechsel der Umgebung für Herausgeber.](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293))

Im Cloud-Partnerportal wird die Einstellung auf der Registerkarte „Marketplace“ oder „Digitale Ladenzeile“ festgelegt. Im Partner Center wird sie auf der Registerkarte „Angebotseinrichtung“ unter „Einrichtungsdetails“ festgelegt.

![Umgebung zum Abonnieren des CSP-Partnerkanals im CPP](media/marketplace-publishers-guide/csp-opt-in.png)

Wie nachstehend zu sehen ist, können Herausgeber ihre Angebote im CSP-Programm verfügbar machen, indem sie das Kontrollkästchen **Mein Angebot für Partner im CSP-Programm (Cloud Solution Provider) verfügbar machen** aktivieren.

![Umgebung zum Abonnieren des CSP-Partnerkanals im Partner Center](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>Bereitstellen von Verkaufs- und Begleitmaterialien für CSP-Partner

Um es Partnern im Cloud Solution Provider-Programm zu ermöglichen, bestmöglich Ihr Angebot zu präsentieren und mit Ihrem Unternehmen in Kontakt zu treten, übermitteln Sie Verkaufs- und Begleitmaterialien, die den Handelspartnern zur Verfügung stehen. Diese Ressourcen werden Kunden in den digitalen Marketplace-Ladenzeilen nicht verfügbar gemacht.

Wie nachstehend zu sehen ist, können Herausgeber im Cloud-Partnerportal eine URL eingeben, unter der die relevanten Marketingmaterialien und Kanalkontaktinformationen für den CSP-Kanal gehostet werden:

![CSP-Zusatzinformationen](media/marketplace-publishers-guide/cpp-csp-information.png)
