# <a name="saas-application-channel-info-tab"></a>Registerkarte „Kanalinformationen“ für eine SaaS-Anwendung

Verwenden Sie die Registerkarte „Kanalinformationen“, um Kanalinformationen für Ihre App bereitzustellen, einschließlich Markteinführungsmaterialien (Go-to-Market, GTM) und Kontaktinformationen.

![Formular für Kanalinformationen](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>So konfigurieren Sie Kanalinformationen

1. Geben Sie unter **Kanalinformationen > GTM-Materialien** die URL für die Website ein, auf der Ihre GTM-Materialien gehostet werden.
2. Geben Sie unter **Kanalmanagerkontakt** eine Managerkontaktperson an, die Ansprechpartner für Microsoft bei Support- und Geschäftsproblemen ist. Geben Sie Informationen in diese Felder ein: Name, E-Mail-Adresse und Telefonnummer.
3. Wählen Sie **Speichern** aus, um das Einrichten der Kanalinformationen zu beenden.
