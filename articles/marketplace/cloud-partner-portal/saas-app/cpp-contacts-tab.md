# <a name="saas-application-contacts-tab"></a>Registerkarte „Kontakte“ für eine SaaS-Anwendung

In diesem Artikel wird beschrieben, wie Sie die Registerkarte „Kontakte“ verwenden, um Kontakte für Kunden zu identifizieren, die Ihr Angebot verwenden.


## <a name="identify-contacts"></a>Identifizieren von Kontakten

Verwenden Sie die nächste Bildschirmaufnahme als Leitfaden für die Bereitstellung der erforderlichen Informationen. Alle Felder, die mit einem Sternchen (*) gekennzeichnet sind, müssen ausgefüllt werden.

![Registerkarte „Kontakte“](./media/saas-offer-contacts.png)

1. Geben Sie unter „Engineering-Kontakt“ Informationen in die folgenden Felder ein:

   - **Name\*** : Name der Kontaktperson für technische Fragen für Ihre App. Diese Kontaktperson erhält technische Mitteilungen von Microsoft.
   - **E-Mail\*** : E-Mail-Adresse der Kontaktperson für technische Fragen.
   - **Telefon\*** : Telefonnummer der Kontaktperson für technische Fragen. Telefonnummern im ISO-Format werden unterstützt. Weitere Informationen finden Sie unter [E.123](https://en.wikipedia.org/wiki/E.123).
  
2. Geben Sie unter „Supportkontakt“ Informationen in die folgenden Felder ein:

   - **Name\*** : Name der Kontaktperson für den Support für Ihre App an. Diese Kontaktperson erhält Mitteilungen im Zusammenhang mit dem Support von Microsoft.
   - **E-Mail\*** : E-Mail-Adresse der Kontaktperson für den Support für Ihre App.
   - **Telefon\*** : Telefonnummer der Kontaktperson für den Support. Telefonnummern im ISO-Format werden unterstützt. Weitere Informationen finden Sie unter [E.123](https://en.wikipedia.org/wiki/E.123).
   - **Support-URL\*** : URL Ihrer Supportseite.

3. Wählen Sie **Speichern** aus, wenn Sie die Formulare abgeschlossen haben.
