# <a name="power-bi-apps-contacts-tab"></a>Registerkarte „Kontakte“ für Power BI-Apps

Verwenden Sie auf der Seite **Neues Angebot** die Registerkarte **Kontakte**, um Ressourcen für technischen und Benutzersupport für Ihr Angebot anzugeben. Sie ist in zwei Abschnitte unterteilt: **Engineering** und **Kundensupport**.

![Registerkarte „Kontakte“](media/contacts-tab.png)


## <a name="contacts-fields"></a>Felder „Kontakte“ 

In der folgenden Tabelle sind Name und Zweck dieser Felder aufgeführt. 

|   Feld               |   BESCHREIBUNG                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *Engineering-Kontakt* | Dient als technischer Kontakt zwischen AppSource und Ihrer Organisation    |
| **Name**              | Name der Person oder Gruppe, die als technischer Support/Engineering-Support fungiert |
| **E-Mail**             | E-Mail-Adresse dieses technischen Ansprechpartners                                  |
| **Telefon**             | Telefonnummer für den technischen Support                                       |
| *Supportkontakt*     | Empfängt die Supporttickets, die von Kunden in AppSource eröffnet werden            |
| **Name**              | Name der Person oder Gruppe, die als Kundensupport fungiert              |
| **E-Mail**             | E-Mail-Adresse des Kundensupports                                        |
|  |  |


## <a name="next-steps"></a>Nächste Schritte

Nachdem Sie das Angebot der Power BI-App fertig gestellt haben, müssen Sie bestätigen, dass Sie alle zugehörigen [technischen Ressourcen](./cpp-create-technical-assets.md) für dieses Angebot erstellt haben.
