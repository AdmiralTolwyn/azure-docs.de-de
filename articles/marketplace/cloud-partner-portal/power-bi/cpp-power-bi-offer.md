# <a name="power-bi-application-offer"></a>Power BI-Anwendungsangebot

|                  |                                |
|------------------|--------------------------------|
| In diesem Abschnitt finden Sie Informationen zum Veröffentlichen einer Power BI-Anwendung im [Marketplace von Microsoft AppSource ](https://appsource.microsoft.com/). Eine Power BI-Anwendung bündelt Power BI-Inhalte und verwendet Datenströme, um Berichte und Dashboards mit Daten in einem gemeinsamen Datenspeicher zu verbinden. Die App kann dann für andere Mandanten über AppSource bereitgestellt werden. | ![Power BI-Symbol](./media/powerbi-icon.png) |

-   [Voraussetzungen](./cpp-prerequisites.md): Die technischen und geschäftlichen Anforderungen, die zum Erstellen oder Veröffentlichen eines Power BI App-Angebots erfüllt sein müssen.
-   [Power BI-App-Angebot erstellen](./cpp-create-offer.md): Die Schritte, die im [Cloud-Partnerportal](https://cloudpartner.azure.com) zum Erstellen eines Eintrags für ein neues Power BI-App-Angebot ausgeführt werden müssen.

## <a name="publishing-steps"></a>Veröffentlichungsschritte

Das folgende Diagramm veranschaulicht die allgemeinen Schritte zum Veröffentlichen eines Power BI-App-Angebots.

![Schritte zur Veröffentlichung einer Power BI-App](media/publishing-steps.png)

Prozess zur Veröffentlichung von Power BI-Apps

1. Erstellen Sie in Power BI eine Dienstanwendung (was zur Erstellung der Paketinstallations-URL führt).  Diese URL stellt die technischen Ressourcen für das Angebot dar.  Weitere Informationen finden Sie unter [Was ist Power BI?](https://go.microsoft.com/fwlink/?linkid=2028636).
2. Sammeln oder erstellen Sie die geschäftlichen Ressourcen wie beispielsweise:
    - Marketingmaterial
    - Rechtliche und Supportdokumente
3.  Erstellen des Angebots: Im Cloud-Partnerportal werden die Details und Informationen für das Angebot festgelegt. Hierzu zählen die Angebotsbeschreibung, Marketingmaterial, rechtliche und Supportinformationen sowie Ressourcenspezifikationen.
4.  Zertifizieren und Veröffentlichen des Angebots: Durch diese Übermittlung wird der Veröffentlichungsprozess gestartet.
    - Das AppSource-Onboardingteam testet, überprüft und zertifiziert Ihre Anwendung. 
    - Nach der Zertifizierung sollten Sie sie in ihrer Testumgebung prüfen und freigeben.

Nachdem alle diese Schritte erfolgt sind, wird Ihre App in AppSource live geschaltet.
