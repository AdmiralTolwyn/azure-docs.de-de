# <a name="getting-started-with-the-cloud-partner-portal"></a>Erste Schritte mit dem Cloudpartnerportal
Dieses Dokument ist eine exemplarische Vorgehensweise für die Nutzung des Cloudpartnerportals, das die Veröffentlichung von Produkten in [Azure Marketplace](https://azuremarketplace.microsoft.com/) und [AppSource](https://appsource.microsoft.com/) unterstützt. Falls noch nicht geschehen, sehen Sie sich unter [Was ist das Cloudpartnerportal?](./cloud-partner-portal-what-is-the-cloud-partner-portal.md) den kompletten Überblick an.

## <a name="portal-tour"></a>Tour durch das Portal
Das Cloudpartnerportal bietet alles, was Sie brauchen, um in der Cloud erfolgreich Geschäfte zu machen. Es folgt eine Kurzübersicht über die Abschnitte dieser Website, mit denen Sie wahrscheinlich am häufigsten zu tun haben werden.

### <a name="left-navigation-bar"></a>Linke Navigationsleiste
Wenn Sie sich erstmals im Portal befinden, sollten Sie die reduzierbare Navigationsleiste sehen. Hier können Sie zwischen den Menüelementen navigieren.

![Seite mit dem Cloudpartnerportal](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/cloud-partner-portal-page.png)

Die einzelnen Elemente werden nachstehend vorgestellt:
1.    **Neues Angebot**: Hier können Sie ein neues Angebot eingeben und alle [unterstützten Produkttypen](./Cloud-partner-portal-products-that-can-get-published-via-portal.md) überprüfen.
2.    **Alle Angebote**: Dient zum Überprüfen des Status aller Ihrer aktuellen Angebote.
3.    **Benutzer**: [Hierüber verwalten und steuern Sie den Zugriff für Ihr Unternehmen](./cloud-partner-portal-manage-users.md).
4.    **Weitere Informationen**: Hier können Sie sich mit dem Partnerportal und Neuerungen vertraut machen.
5.  **Insights**: Über einen Besuch im Application Insights-Portals erhalten Sie Einblicke und Nutzungsstatistiken zu Ihren Produkten und Kunden.

### <a name="offer-page"></a>Seite „Angebot“ 
Wir setzen unsere Tour auf der Seite „Angebot“ fort, die Sie häufig besuchen werden, um ein neues Angebot einzugeben oder Ihr Angebot zu bearbeiten. Hier konfigurieren Sie sämtliche Einstellungen für Ihr Angebot und überprüfen seinen Status. 

![Seite „Angebot“](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/offer-page.png )

1.    Im Modus **Editor** können Sie Inhalte hinzufügen. Im Modus **Status** können Sie den Status Ihres Angebots überprüfen.
2.    Über die **Aktionsleiste** können Sie Änderungen an Ihrem Angebot speichern oder Ihre Eingabe verwerfen.
3.    Navigieren Sie zwischen den erforderlichen **Formularen** für jedes Angebot, während Sie die Einstellungen Ihres Angebots ändern.

### <a name="menu-bar"></a>Menüleiste
Oben auf dem Bildschirm befindet sich die stets angezeigte Menüleiste. Diese Leiste ermöglicht interaktive Aktionen für Support und Feedback und enthält alle Informationen Ihres Herausgeberkontos.

![Menüleiste](./media/cloud-partner-portal-getting-started-with-the-cloud-partner-portal/menu-bar.png)

*    **Benötigen Sie Hilfe?** Wenn Sie Hilfe benötigen sollten, stellen Sie eine Supportanfrage, oder klicken Sie auf die Links zur Dokumentation.
*    **Feedback** Möchten Sie uns etwas mitteilen? Lassen Sie es uns hier wissen. Microsoft ist bestrebt, den Azure Marketplace, das AppSource-Veröffentlichungsverfahren und die gesamte Nutzung des Cloudpartnerportals möglichst einfach und intuitiv zu gestalten. Wir prüfen und lesen jedes Feedback, und wir schützen Ihre Privatsphäre. (Wir kontaktieren Sie nur, wenn wir weitere Informationen benötigen.) 
*    **Herausgeberkonto** Wenn Sie das Portal erstmals besuchen, wird eine Vereinbarung für Herausgeber angezeigt, die Sie akzeptieren müssen, um Herausgeber zu werden. Anschließend können Sie hier Ihr Herausgeberprofil bearbeiten oder die Vereinbarung erneut überprüfen. Hier stellen Sie auch eine Verbindung mit Ihrem Developer Center-Konto her, damit Sie Ihre Angebote verkaufen können.

## <a name="next-steps"></a>Nächste Schritte    
Nach dieser Tour durch das Cloudpartnerportal erfahren Sie mehr über die [unterstützten Produkttypen](./Cloud-partner-portal-products-that-can-get-published-via-portal.md), die Sie veröffentlichen können.
