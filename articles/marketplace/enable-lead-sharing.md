# <a name="enable-lead-sharing"></a>Gemeinsamen Zugriff auf Leads aktivieren  
Sie sollten sicherstellen, dass die Verwaltung von Leads für Ihr Angebot im Marketplace aktiviert ist, damit Sie Kontaktinformationen für die folgenden Zielgruppen erhalten.

| Storefront | audience |  
|:--- |:--- |  
| AppSource | Geschäftskunden |  
| Azure Marketplace | Kunden |  

Je mehr Leads Sie haben, desto erfolgreicher sind zukünftige Kampagnen und Verkaufsaktivitäten durch Ihre Außendienstmitarbeiter. Außerdem erheben Sie aussagekräftigere Informationen zur Effektivität Ihrer App oder Ihres Angebots. 

| Storefront | campaign |  
|:--- |:--- |  
| AppSource | Pflegen |  
| Azure Marketplace | Bedarfsgesteuerte Generierung |  

Gehen Sie wie folgt mit Leads um:  
*   Qualifizieren und hinsichtlich Verkaufschancen bewerten
*   Pflegen, sodass sie in eine Vertriebspipeline eintreten
*   Ihre Maßnahmen auf die allgemeine Marketingkampagnenstrategie abstimmen 

Die Leads zeigen Kundeninteresse am folgenden Angebot. 

| Storefront | Angebot |  
|:--- |:--- |  
| AppSource | <table> <tr><td>App</td></tr> <tr><td>Beratungsdienst</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Angebot</td></tr> <tr><td>Technologie</td></tr> </table> |  

Sie sollten einen Prozess einrichten, damit Leads weiterhin bearbeitet werden.  

| Storefront | Leads |  
|:--- |:--- |  
| AppSource | <table> <tr><td>Ihre CRM</td></tr> <tr><td>Microsoft-Außendienst-CRM</td></tr> </table> |  
| Azure Marketplace | <table> <tr><td>Azure Marketplace</td></tr> <tr><td>Microsoft-Außendienst-CRM</td></tr> </table> |  

Ihre Marketplace-Leads in der Storefront des Marketplace unterscheiden sich von Microsoft Seller-Qualified Leads. Diese sind eine Besonderheit des Co-Selling-Programms.  
*   Weitere Informationen zum Zugriff auf das Co-Selling-Programm finden Sie auf der Seite „Promote Your Business with Microsoft“ (Fördern Ihres Unternehmens mit Microsoft) unter [promote-your-business-with-microsoft](./promote-your-business-with-microsoft.md).
