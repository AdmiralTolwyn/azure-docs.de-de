## <a name="publishing-process"></a>Veröffentlichungsvorgang

Zum Erstellen und Veröffentlichen einer Dynamics 365 for Customer Engagement-Lösung führen Sie folgende Schritte aus, die in den weiteren Artikeln dieses Abschnitts erläutert werden:

1. [Registrieren](https://partners.microsoft.com/PartnerProgram/simplifiedenrollment.aspx) Sie sich im Microsoft Partner Network (MPN), falls Sie dort noch nicht registriert sind.
2. [Erstellen Sie eine Dynamics 365-Lösung](./dynamics-365-solution-preparation.md).
3. [Erstellen Sie ein AppSource-Paket](./appsource-package-preparation.md) und zugehörige Marketinginhalte.
4.  Laden Sie Ihr AppSource-Paket in Azure Storage hoch, und [generieren Sie einen SAS-Schlüssel](./store-your-appsource-package-to-azure-storage-and-generate-sas-key.md).
5. [Erstellen Sie das zugehörige Angebot](./setup-your-application-in-the-publishing-portal.md).
6. [Veröffentlichen Sie Ihr Angebot](./request-to-publish-the-app.md), um es live zu schalten.


## <a name="next-steps"></a>Nächste Schritte
