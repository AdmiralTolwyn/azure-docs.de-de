# <a name="lcs-subscription"></a>LCS-Abonnement

In diesem Tutorial erfahren Sie, wie Sie ein Abonnement erstellen. Mit diesem Abonnement erhalten Sie einen Microsoft Online-Testmandanten und ein Microsoft Dynamics Lifecycle Services-Projekt zum Bereitstellen einer Umgebung. Darüber hinaus richten Sie weitere Benutzer in Ihrem Microsoft Online-Mandanten ein und machen sich näher mit den folgenden Dienstverwaltungsfunktionen vertraut. In diesem Tutorial lernen Sie Folgendes:

- Abonnieren und Erstellen eines neuen Microsoft Online-Testmandanten
- Navigieren zu Lifecycle Services-Projekten
- Verwenden von Features in Lifecycle Services
- Hinzufügen weiterer Benutzer zu Microsoft Azure Active Directory und zum Client
- Anzeigen von Ressourcen in Ihrer Abonnement-E-Mail

## <a name="key-terms"></a>Schlüsselbegriffe

**Microsoft Online Services-Mandant:** Ein Mandant ist die Gruppe aller Abonnements und Benutzer für Ihre Organisation. Der Mandant wird zeitgleich mit Ihrem ersten Abonnement in Microsoft Online Services erstellt.

**Abonnement:** Ein Abonnement stellt Ihnen eine Onlineumgebung zur Verfügung. Mit dieser Umgebung können Sie Dynamics 365 for Operations in der Cloud nutzen und anzeigen, wie von Ihnen entwickelte Anpassungen in der Cloud bereitgestellt werden können.

**Microsoft Azure Active Directory:** Die Cloudumgebung beinhaltet Azure Active Directory (AD). Mit diesem Dienst können Sie Benutzer, Gruppen, Sicherheitsrollen und Lizenzen für Onlineanwendungen verwalten. Dabei gehen Sie ähnlich vor wie bei der Verwaltung für lokale Umgebungen.

**Benutzer:** Benutzer der Dienste, die Ihre Organisation abonniert hat, werden in Azure AD verwaltet. Alle Benutzer in Ihrem Mandanten können Sicherheitsrollen hinzugefügt und zugewiesen werden.

**Entwickler und Administratoren:** Entwickler und Administratoren sind Benutzer, die außerdem Zugriff auf Lifecycle Services besitzen und damit Projekte und Umgebungen verwalten können. Diese Benutzer sind auch Endbenutzer.

Weitere Informationen finden Sie unter [Sign up for preview subscriptions](https://docs.microsoft.com/dynamics365/operations/dev-itpro/dev-tools/sign-up-preview-subscription) (Registrieren für Vorschauabonnements).
