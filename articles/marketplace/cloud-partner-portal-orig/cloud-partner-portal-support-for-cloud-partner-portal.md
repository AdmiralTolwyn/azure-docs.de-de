# <a name="get-support-for-cloud-partner-portal"></a>Erhalten von Support für das Cloud-Partnerportal

Microsoft bietet Support für eine Vielzahl von Produkten und Diensten.
Um eine hilfreiche und zeitnahe Antwort zu erhalten, muss Ihre Anfrage natürlich das richtige Support-Team erreichen. Die folgenden Szenarien sollen Ihnen dabei helfen, Ihre Anfrage an das richtige Team zu richten:

-   Wenn Sie als Herausgeber eine Frage von einem Kunden erhalten haben, bitten Sie ihn darum, über die Supportlinks im [Azure-Portal](https://portal.azure.com/) Unterstützung anzufordern.

-   Wenn Sie ein Herausgeber sind und eine Frage hinsichtlich Ihrer App oder Ihres Diensts haben, gehen Sie entsprechend den folgenden Schritten vor, um ein Supportticket zu öffnen.

## <a name="to-open-a-support-ticket"></a>So öffnen Sie ein Supportticket

1. Melden Sie sich im [Cloud-Partnerportal](https://cloudpartner.azure.com/) mit Ihrem Geschäfts-, Schul- oder Unikonto an.

2. Wählen Sie im oberen Menü rechts auf Seite das **Hilfe**-Symbol aus und wählen Sie dann **Support** aus.

![Support1](./media/cloud-partner-portal-support-for-cloud-partner-portal/support1.png)


3. Geben Sie unter **Incident erstellen** den **Problemtyp** und die **Kategorie** an.


![Support2](./media/cloud-partner-portal-support-for-cloud-partner-portal/support2.png)


4. Wählen Sie **Anfrage starten** aus.

5. Stellen Sie Kontaktinformationen bereit, und beschreiben Sie das Problem. Beschreiben Sie das Problem so vollständig wie möglich. Stellen Sie alle Dateien bereit (z. B. Bildschirmfotos, Fehlerbeschreibungen oder Netzwerkablaufverfolgungen), die für das Beheben des Problems nützlich sein könnten.

![Support3](./media/cloud-partner-portal-support-for-cloud-partner-portal/support3.png)

6. Wählen Sie **Senden** aus, um das Ticket zu erstellen.

### <a name="support-followup"></a>Supportfortgang

Wir bestätigen den Erhalt Ihres Tickets innerhalb von 24 Stunden und weisen ihrem Problem eine Priorität und einen Schweregrad zu. Unser engagiertes Team von Supporttechnikern unterstützt Sie gemäß der in unserer Vereinbarung zum Servicelevel festgelegten Antwortzeiten bei der Lösung des Problems. 

Das nächste Bildschirmfoto zeigt die Incidentinformationen, die Sie vom Microsoft-Support als Bestätigung erhalten.

![Support4](./media/cloud-partner-portal-support-for-cloud-partner-portal/support4.png)


-   Wenn Sie Ihr Problem mit uns besprechen möchten, beantworten Sie die E-Mail, die Sie für die Supportzeit erhalten haben, und wir können Zeit für ein Gespräch mit Ihnen einplanen.

-   Sie können den Bearbeitungsstatus Ihres Problems jederzeit anzeigen, indem Sie auf den Link **Incident online anzeigen** klicken, der im vorherigen Bildschirmfoto gezeigt ist.
