# <a name="azure-application-offer"></a>Azure-Anwendungsangebot

<table> <tr> <td>In diesem Abschnitt wird erläutert, wie Sie ein neues verwaltetes Azure-Anwendungsangebot im <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a> veröffentlichen.  Jede verwaltete Anwendung enthält eine Azure Resource Manager-Vorlage, die alle technischen Ressourcen definiert, die von der Anwendung verwendet werden. Dazu gehören in der Regel mindestens ein virtueller Computer sowie andere unterstützende Azure- oder webbasierte Dienste. </td> <td><img src="./media/cloud-partner-portal-publish-managed-app/azureapp-icon1.png"  alt="Azure application icon" /></td> </tr> </table>

## <a name="publishing-overview"></a>Übersicht über die Veröffentlichung

Das folgende Video, [Building Solution Templates, and Managed Applications for the Azure Marketplace](https://channel9.msdn.com/Events/Build/2018/BRK3603) (Erstellen von Lösungsvorlagen und verwalteten Anwendungen für den Azure Marketplace), erläutert, wie Sie eine Azure Resource Manager-Vorlage erstellen, um eine Azure-Anwendungslösung zu definieren, und das Anwendungsangebot danach im Azure Marketplace veröffentlichen.

> [!VIDEO https://channel9.msdn.com/Events/Build/2018/BRK3603/player]


## <a name="publishing-process"></a>Veröffentlichungsvorgang

Die folgenden Artikel erläutern den Vorgang der Erstellung und Veröffentlichung einer verwalteten Azure-Anwendung:

1. [Registrieren eines Kontos beim Microsoft Developer Center](./cloud-partner-portal-dev-center-accounts-registration.md), um die Genehmigung für das Veröffentlichen im Azure Marketplace zu erhalten.
2. [Veröffentlichen einer in Azure verwalteten Anwendung im Azure Marketplace](./cloud-partner-portal-managed-app-publish.md).
3. [Live-Schaltung Ihres Angebots](./cloud-partner-portal-managed-app-offer-go-live.md). 


## <a name="next-steps"></a>Nächste Schritte

Als Nächstes müssen Sie die Anforderungen für diesen Angebotstyp überprüfen und den Veröffentlichungsprozess starten, wie unter [Erstellen einer verwalteten Anwendung](./cloud-partner-portal-managed-app-publish.md) beschrieben.
