# <a name="azure-saas-application-offer"></a>Azure-SaaS-Anwendungsangebot

<table> <tr> <td>In diesem Abschnitt wird erläutert, wie Sie eine SaaS-Anwendung (Software-as-a-Service) im <a href="https://azuremarketplace.microsoft.com/marketplace/">Microsoft Azure Marketplace</a> oder in <a href="https://appsource.microsoft.com">Microsoft AppSource</a> veröffentlichen. Verwenden Sie SaaS-Anwendungen, wenn Ihre Lösung in Ihrem eigenen Azure-Abonnement bereitgestellt wird und Kunden sich über eine Schnittstelle anmelden, die Sie entwerfen und verwalten, um die Anwendung zu testen. <a href="https://azure.microsoft.com/services/active-directory">Azure Active Directory</a> (AAD) wird verwendet, um Ihre vorhandene Testumgebung zu nutzen. </td> <td><img src="./media/cloud-partner-portal-publish-web-app/saasapp-icon1.png"  alt="Azure SaaS application icon" /></td> </tr> </table>


## <a name="publishing-benefits"></a>Vorteile der Veröffentlichung

Das Listen Ihrer Anwendungen in einem Microsoft Marketplace bietet u.a. folgende Vorteile:

- Sie erreichen 100 Millionen Azure Active Directory-Benutzer über Office 365 und Dynamics 365.
- Sie erweitern Ihr Vertriebsteam: Sie erreichen Geschäftsbenutzer auf der ganzen Welt und profitieren von einem Vertriebskanal, der Endbenutzer anspricht, beim Generieren von Leads hilft und die Kommunikation mit neuen Kunden in den unterschiedlichsten Branchen ermöglicht.

- Sie gewinnen umsetzbare Erkenntnisse: Wir informieren Sie darüber, wie Ihre App in AppSource abschneidet, was gut funktioniert und wie Sie Vertriebsverfahren weiter verbessern können.


## <a name="publishing-process"></a>Veröffentlichungsvorgang

Mit folgenden Schritten erstellen Sie Ihre Apps und veröffentlichen sie im Microsoft Azure Marketplace oder in Microsoft AppSource:

1. Vergewissern Sie sich, dass alle Voraussetzungen erfüllt sind, einschließlich der Registrierung bei Partnern: [Registrierung bei Dev Center](https://dev.windows.com/en-us/registration?accountProgram=Azure) *oder* [Registrierung beim Microsoft Partner Network](https://partner.microsoft.com/).
2.  Definieren Sie Ihr Angebot und Ihre Pläne für den Marketplace.
3.  Stellen Sie Marketinginhalte und Preisinformationen bereit.
4.  Testen und zertifizieren Sie Ihr Angebot im [Cloud-Partnerportal](https://cloudpartner.azure.com).
5.  Schalten Sie Ihr Angebot live.
