# <a name="azure-appsource-and-marketplace-review-policies"></a>Bewertungsrichtlinien für Azure AppSource und Marketplace

Die Bewertungs- und Rezensionsrichtlinien für Azure AppSource und Marketplace sind hier aufgeführt.

* Folgende Verhaltensweisen oder Inhalte in AppSource- und Azure Marketplace-Bewertungen werden von Microsoft nicht toleriert:
    * Bot-/Cyberangriffe
    * Verstöße gegen den Datenschutz
    * Spam-Inhalte
    * Betrügerische Inhalte
    * Anstößige Inhalte: obszöne oder anstößige Sprache oder Gesten
    * Illegale Inhalte
    * Beleidigende, hasserfüllte oder drohende Inhalte
    * Diffamierende Inhalte
    * Wiederholte Übermittlungen gleicher oder ähnlicher Inhalte
    * Werbung, einschließlich der Bewerbung anderer Apps und Dienste

* Microsoft behält sich das Recht vor, von einem Benutzer übermittelte Bewertungsinhalte aus welchem Grund auch immer zu entfernen.
* Microsoft behält sich das Recht vor, einen Benutzer aus welchem Grund auch immer für die Übermittlung von Bewertungsinhalten zu sperren.
* Herausgeber erhalten keine Benachrichtigung, wenn Bewertungsinhalte im Zusammenhang mit ihren Anwendungen entfernt werden.
* Benutzer erhalten keine Benachrichtigung, wenn von ihnen übermittelte Bewertungsinhalte entfernt werden.
* Microsoft ist nicht verpflichtet, eine Erklärung in Bezug auf das Entfernen von Bewertungsinhalten bereitzustellen.
* Microsoft berücksichtigt keine Anforderungen zum Wiederherstellen von entfernten Bewertungsinhalten.

## <a name="review-guidelines-for-customers"></a>Bewertungsrichtlinien für Kunden

* Übermitteln Sie keine Inhalte, die möglicherweise gegen die [Bewertungsrichtlinie für Azure AppSource und Marketplace](./rating-review-policies.md) verstoßen.
* Geben Sie keine personenbezogenen Informationen wie E-Mail-Adresse, physische Adresse, Telefonnummern usw. an.
* Achten Sie darauf, dass Ihre Bewertung eindeutig, lesbar und informativ ist.
* Achten Sie darauf, dass Sichtweisen in Ihrer Bewertung ausgewogen und neutral ausgedrückt sind.
* Denken Sie daran, dass Sie Beiträge in einem öffentlichen Forum veröffentlichen, und verhalten Sie sich entsprechend.
