# <a name="azure-marketplace-and-appsource-publisher-guide"></a>Herausgeberleitfaden für Azure Marketplace und AppSource  

## <a name="overview"></a>Übersicht  
Willkommen beim Herausgeberleitfaden für die Storefronts im Marketplace.
*  AppSource  
*  Azure Marketplace  

Neue und vorhandene Herausgeber erfahren in diesem Leitfaden mehr über die folgenden Themen: 
*   Verwenden von Azure Marketplace- und AppSource-Storefronts
*   Veröffentlichen von Apps und Angeboten  
*   Steigern des Wachstum Ihres Unternehmens in Zusammenarbeit mit Microsoft  

Sie erhalten ein besseres Bild von folgenden Themen:
*   Was sind die Vorteile der Auflistung in einer der Storefronts im Marketplace?  
*   Verwenden der Storefronts  
*   Welche Storefront ist für Ihre Angebote und Dienste geeignet?  
*   Welche Arten von Apps und Angeboten können Sie veröffentlichen?  
*   Wie lauten die technischen und geschäftlichen Anforderungen für die einzelnen Veröffentlichungsoptionen?  
*   Erstellen einer Vorveröffentlichungs-Objektprüfliste  
*   Herausgeber werden  
*   Wo werden Angebote erstellt und veröffentlicht?  
*   Optimieren einer Auflistung und Verwenden von Ressourcen für die Markteinführung, um die Wirkung zu verbessern  
*   Zugreifen auf Hilfe und Support  

Fragen und Kommentare zum Marketplace oder dem Herausgeberleitfaden können Sie per E-Mail an [cloudmarketplace@microsoft.com](mailto:cloudmarketplace@microsoft.com) senden.

## <a name="benefits"></a>Vorteile  

### <a name="benefits-of-participating-on-the-marketplace"></a>Vorteile der Teilnahme am Marketplace  
Die Storefronts im Marketplace sind Launchpads für gemeinsame Markteinführungsaktivitäten mit Microsoft.  
*   AppSource  
*   Azure Marketplace  

Die Storefronts sind Zugänge zu Partnerschaften, die Verkaufsgelegenheiten eröffnen. Sie können Ihre App oder Ihr Angebot in einer der folgenden Rollen verwenden.  
*   Als Kernstück Ihres Cloudgeschäfts  
*   Als Antriebskraft für das Wachstum Ihres Unternehmens 

Sie können das Wachstum Ihres Unternehmens mit den folgenden Möglichkeiten im Marketplace vorantreiben:  
*    Markteinführungskampagnen  
*    Generierung von Nachfrage  
*    Gemeinsamer Vertrieb und Marketing  

Es werden keine Gebühren für Ihre Teilnahme an den Storefronts im Marketplace berechnet. Microsoft ist bestrebt, Ihnen die besten Lösungen und Dienste von den Microsoft-Partnern bereitzustellen. 

Sie können die Funktionen des Marketplace nutzen, um Ihr Geschäftswachstum zu fördern.  

#### <a name="generate-leads-and-sales-opportunities"></a>Generieren Sie Leads und Verkaufschancen.  
Betreten Sie neue Märkte mit einem erweiterten Portfolio von Lösungen in den Storefronts im Marketplace. Setzen Sie Marketplace-Angebote für Up- und Cross-Selling ein.  

#### <a name="enhance-business-value-and-increase-deal-size-with-existing-and-new-customers"></a>Steigern Sie Geschäftswert und Auftragsvolumen für vorhandene und neue Kunden.  
Erhöhen Sie das Auftragsvolumen, und lösen Sie Kundenprobleme, nachdem Sie Workloads zu Azure verschoben haben. Verkürzen Sie Verkaufszyklen, und steigern Sie die Rentabilität von Geschäften, indem Sie vollständige Lösungen für spezifische Workloads und Branchenszenarien verkaufen.  

#### <a name="get-actionable-insights"></a>Verschaffen Sie sich verwertbare Einblicke.  
Ihr Erfolg ist beiderseitiger Erfolg. Erhalten Sie einen Einblick in die Leistung Ihrer Auflistungen über das Cloud-Partnerportal. Erhalten Sie weitere Informationen zu Leistung, Leads und maximierten Kampagnenaktivitäten Ihrer App.  

>[!NOTE]
>Apps zur Erweiterung von Office müssen Einblicke in den Veröffentlichungsprozess für Office-Apps erhalten.  

### <a name="go-to-market-benefits"></a>Vorteile bei der Markteinführung  
Mithilfe der Storefronts können Sie Ihre Lösungen Millionen von Kunden präsentieren. Für neue Listings im Marketplace wird automatisch eine Reihe kostenloser Ressourcen für die Markteinführung angeboten, um Ihr Angebot im Marketplace bekannt zu machen.  

 Ihr Launchpad für gemeinsame Markteinführungsaktivitäten mit Microsoft beginnt mit der Veröffentlichung Ihres Angebots in den Storefronts.  Die Storefronts sind Zugänge zu Partnerschaften, die Verkaufsgelegenheiten eröffnen. Alle neuen Apps und Angebote erhalten automatisch eine Reihe von kostenlosen Markteinführungsvorteilen, um die Bekanntheit Ihrer App oder Ihres Angebots unter den Microsoft-Kunden zu steigern. Die Markteinführungsvorteile umfassen verschiedene Aktivitäten des gemeinsamen Marketings und Vertriebs.  Diese gemeinsamen Marketing- und Vertriebsaktivitäten dienen zur Steigerung der Bekanntheit Ihrer Lösung und zum Erzielen von Verkäufen mithilfe der folgenden Elemente:
*   Microsoft-Marke  
*   Microsoft-Kanäle  
*   Microsoft-Kunden und -Partner  

Sie müssen keine zusätzlichen Maßnahmen ergreifen. Nachdem Sie Ihr Angebot veröffentlicht haben, kontaktiert Sie das Microsoft-Markteinführungsteam, um mit der Bereitstellung zu beginnen.  
*   Weitere Informationen zu den Microsoft-Markteinführungsvorteilen und zu Möglichkeiten, das Wachstum Ihres Geschäfts im Marketplace zu steigern, finden Sie auf der Seite „Microsoft Go-To-Market Services“ unter [partner.microsoft.com/reach-customers/gtm](https://partner.microsoft.com/reach-customers/gtm).

## <a name="storefronts"></a>Storefronts  
Microsoft bietet zwei unterschiedliche Storefronts im Marketplace. Die Storefronts ermöglichen Ihnen das Auflisten von Angeboten, das Aktivieren von Testversionen und direkte Transaktionen mit Microsoft-Kunden.  
*   Vergleich von AppSource und Azure Marketplace
    *   Weitere Informationen zu den Storefronts finden Sie auf der Seite [Vergleich von AppSource und Azure Marketplace](./comparing-appsource-azure-marketplace.md).
*   AppSource 
    *   Weitere Informationen zu AppSource finden Sie auf der Seite [Ausbauen Ihres Geschäfts mit AppSource](./grow-your-business-with-appsource.md).  
*   Azure Marketplace 
    *   Weitere Informationen zum Azure Marketplace finden Sie auf der Seite [Ausbauen Ihres Geschäfts mit Azure Marketplace](./grow-your-business-with-azure-marketplace.md).  

Mit Storefronts können Sie Apps und Dienste, mit denen die digitale Transformation beschleunigt werden kann, finden, ausprobieren und kaufen. Die Storefronts erleichtern Ihnen den Ausbau Ihres Geschäfts, indem sie einen erweiterten Zugang zu Kunden und Partnern von Microsoft bieten.
