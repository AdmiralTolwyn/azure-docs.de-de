# <a name="what-is-appsource"></a>Was ist AppSource?
Mit AppSource können Microsoft-Partner ihre Technologien und Dienste über eine von Microsoft unterstützte Webstorefront für Kunden auffindbar machen. Wecken Sie als AppSource-Herausgeber bei Geschäftsbenutzern das Interesse, in AppSource branchenspezifische SaaS-Anwendungen und Implementierungsdienste zu entdecken, zu testen und abzurufen und so gute Geschäftsergebnisse zu erzielen und die Amortisationszeit zu verkürzen: 

| Kundenanforderung       | AppSource          | 
|:---------------------------------------- |:----------------------------------------------------- |
| **Suche nach Geschäftslösungen, die mit bereits vom Kunden verwendeten Microsoft-Produkten verwendet werden können**    | Ermöglicht Partnern, Anwendungen und Dienste von Drittanbietern zu verwenden, um Cloudanwendungen und -technologien von Microsoft zu erweitern. |
| **Möglichkeit der einfachen Suche der geeigneten Lösung oder des passenden Implementierungsdiensts**     | Stellt eine zentrale Anlaufstelle zum Entdecken, Testen und Abrufen von Anwendungen, Diensten, Add-Ins und vielem mehr dar. |
| **Branchenspezifische Lösung für die speziellen Geschäftsanforderungen des Kunden.**   | Bietet fertige End-to-End-Branchenlösungen für spezifische Anforderungen in zahlreichen Branchen. |
| **Apps zur Optimierung der Produktivität, Effizienz und geschäftlichen Einblicke**       | Bietet Apps für verschiedene Geschäftsbereiche, darunter Kundenservice, Personalwesen, Betrieb und vieles mehr. |
| **Erfahrener Implementierungspartner zur Unterstützung bei der Anpassung von Apps auf die spezifische Situation des Kunden**     | Bietet einen Katalog mit Beratungsangeboten für auf Dynamics 365, Power BI und PowerApps basierenden Lösungen. Zudem stehen in AppSource Apps von Drittanbietern zur Verfügung, sodass Geschäftskunden Beratungsdienste für vorhersagbare Ergebnisse finden können. |

## <a name="appsource-publishing"></a>Veröffentlichung in AppSource
Über AppSource können Sie unter Verwendung von Microsoft-Produkten (z.B. Office 365, Dynamics 365, Power BI und PowerApps) eine Anwendung oder ein Beratungsangebot listen, das bei der Erfüllung von Kundenanforderungen unterstützt. Die folgenden Angebote sind im AppSource Marketplace erhältlich: 
*   **Dynamics 365 for Finance and Operations**: Eine ERP-Lösung (Enterprise Resource Planning). Die herausragendsten Lösungen sind die Enterprise Edition für mittelständische und Großunternehmen sowie die Business Edition für kleine und mittelständische Unternehmen.
*   **Dynamics 365 for Customer Engagement**: Eine CRM-Lösung (Customer Relationship Management), die die Anwendungen Sales, Customer Service, Field Service, und Project Service Automation umfasst.
*   **Verwalteter Dynamics NAV-Dienst**: Microsoft **Dynamics NAV** ist eine ERP-Softwaresuite (Enterprise Resource Planning) für mittelständische Organisationen. Der Dienst bietet spezialisierte Funktionen für das Produktionswesen, den Vertrieb, Behörden, den Einzelhandel und andere Branchen.
*   **Power BI**: Eine Business Analytics-Lösung mit Visualisierungsfunktionen.
*   **Beratungsangebote**: Von Microsoft-Partnern bereitgestellte Dienste, die Kunden über spezialisierte Technologielösungen aufklären und beim Testen und Implementieren unterstützen sollen.
*   **Cortana Intelligence**: Die Cortana Intelligence Suite ist eine Sammlung von unabhängigen, jedoch vollständig integrierten Daten- und Analyseplattformtools von Azure.
*   **Office 365**: Moderner Arbeitsbereich in der Cloud. Ermöglicht kostenlos die Zusammenarbeit bei der Verwendung von Microsoft Word, PowerPoint, Excel und OneNote.

### <a name="consulting-offers"></a>Beratungsangebote

Lesen Sie die [Informationen zu den Richtlinien, und erhalten Sie Informationen zum Übermitteln Ihres Angebots](https://smp-cdn-prod.azureedge.net/documents/Microsoft%20AppSource%20Partner%20Listing%20Guidelines.pdf "AppSource – Richtlinien zum Partnerlisting").

## <a name="azure-active-directory-integration"></a>Azure Active Directory-Integration
Für die Veröffentlichung einer Integration in Azure Active Directory (Azure AD) sind einige AppSource-Storefrontanwendungen erforderlich. Die Integration von Anwendungen in Azure AD ist umfassend dokumentiert. Microsoft bietet verschiedene SDKs und Ressourcen, um Ihre Anforderungen zu erfüllen. 

| Bezeichnung des Angebots | Erfordert eine Integration in Azure AD | Notizen |
|:--- |:---:|:--- |
| Dynamics 365 for Operations | Nein  | Eine lizenzbasiertes Angebot erfordert keine Integration in Azure AD. |
| Dynamics 365 for Finance | Nein  | Eine lizenzbasiertes Angebot erfordert keine Integration in Azure AD. |
| Dynamics 365 for Customer Engagement | Nein  | Eine lizenzbasiertes Angebot erfordert keine Integration in Azure AD. |
| Verwalteter Dynamics NAV-Dienst | Nein  | Eine lizenzbasiertes Angebot erfordert keine Integration in Azure AD. |
| Power BI | Nein  | Eine lizenzbasiertes Angebot erfordert keine Integration in Azure AD. |
| Beratungsangebote | Nein  | Dienste werden vom Partner koordiniert, nicht über eine Webumgebung. |
| Cortana Intelligence | JA | Azure AD ist eine bewährte Methode, um eine reibungslose Kundenerfahrung sicherzustellen, und die qualifiziertesten Leads zu generieren. |
| SaaS-Apps (ehemals Web-Apps) | Bevorzugt | Azure AD ist eine bewährte Methode, um eine reibungslose Kundenerfahrung sicherzustellen, und die qualifiziertesten Leads zu generieren. |
