# Herunterladbarer SQL-Abfragespickzettel für DocumentDB

Mit dem **DocumentDB-Abfragespickzettel** können Sie schnell Abfragen für DocumentDB-Daten schreiben, indem dort allgemeine Abfragen, Schlüsselwörter, integrierte Funktionen und Operatoren auf einem leicht zu druckenden Referenzblatt dargestellt werden.

DocumentDB unterstützt die relationale, hierarchische und räumliche Abfrage von JSON-Dokumenten mithilfe von [SQL](documentdb-sql-query.md) und ohne ein Schema oder sekundäre Indizes anzugeben. Neben den standardmäßigen ANSI-SQL-Schlüsselwörtern und -Operatoren unterstützt DocumentDB eine benutzerdefinierte JavaScript-Funktionen (UDFs), JavaScript-Operatoren und eine Vielzahl von integrierten Funktionen.

## Herunterladen des DocumentDB-Abfragespickzettels

Schreiben Sie Ihre Abfragen schneller, indem Sie den Abfragespickzettel herunterladen und ihn als Kurzübersicht verwenden. Der Spickzettel zeigt allgemeine SQL-Abfragen, die zum Abrufen von Daten aus zwei beispielhaften JSON-Dokumenten verwendet werden. Drucken Sie den einseitigen Spickzettel in Letter-Seitengröße (8.5 x 11 Zoll) aus, um schnell darauf zugreifen zu können.

**Laden Sie den Spickzettel hier herunter: [Microsoft DocumentDB-Abfragespickzettel](http://go.microsoft.com/fwlink/?LinkId=623215)**
