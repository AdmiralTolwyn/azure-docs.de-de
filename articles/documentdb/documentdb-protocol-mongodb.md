# DocumentDB-Protokollunterstützung für MongoDB

Microsoft Azure DocumentDB-Datenbanken können jetzt als Datenspeicher für Apps verwendet werden, die für MongoDB geschrieben wurden. Indem Sie vorhandene Treiber für MongoDB und diese Vorschaufunktion verwenden, können für MongoDB geschriebene Anwendungen einfach und transparent mit DocumentDB kommunizieren. Entwickler können ihre aktuellen Tools und Fähigkeiten für MongoDB verwenden, um Anwendungen zu erstellen, bei denen die vollständig verwalteten und skalierbaren NoSQL-Datenbanken von DocumentDB genutzt werden.

Wir führen derzeit das Onboarding für Kunden durch, die die Vorschauversion dieser neuen Funktion testen können. Falls Sie sich am Testen der Vorschaufunktion beteiligen möchten, können Sie die Anleitung unten befolgen und den Zugriff auf das Vorschauprogramm anfordern. Wir setzen uns dann mit Ihnen in Verbindung, nachdem die Genehmigung für Sie erteilt wurde.

## Voraussetzungen

Sie müssen über Folgendes verfügen, bevor Sie den Zugriff auf das Vorschauprogramm anfordern können:

- **Ein Azure-Abonnement**. Weitere Informationen finden Sie unter [Erstellen Sie noch heute Ihr kostenloses Azure-Konto](https://azure.microsoft.com/free/).

## Anfordern des Zugriffs auf DocumentDB-Protokollunterstützung für MongoDB (Vorschau)  

Führen Sie die folgenden Schritte aus, um den Zugriff auf die Vorschauversion anzufordern:

1. Melden Sie sich beim [Azure-Portal](https://portal.azure.com) an.
2. Klicken Sie nacheinander auf **NEU**, **Daten und Speicher**, **Alle anzeigen**, **Mehr** und **DocumentDB – Protokollunterstützung für MongoDB**.

	![Screenshot der Blätter „Marketplace“ und „Daten und Speicher“ mit hervorgehobener Option „DocumentDB – Protokollunterstützung für MongoDB“](./media/documentdb-protocol-mongodb/marketplacegallery1.png)

3. Alternativ hierzu können Sie unter der Kategorie **Daten und Speicher** nach „DocumentDB-Protokoll“ suchen.

	![Screenshot der Suchblätter „Marketplace“ und „Daten und Speicher“ mit hervorgehobener Option „DocumentDB – Protokollunterstützung für MongoDB“](./media/documentdb-protocol-mongodb/marketplacegallery2.png)

4. Klicken Sie auf **Erstellen**, um den Anmeldevorgang für die Vorschau zu starten.

	![Blatt „DocumentDB – Protokollunterstützung für MongoDB“ im Azure-Portal](./media/documentdb-protocol-mongodb/marketplacegallery3.png)

5. Klicken Sie im Blatt **Neues DocumentDB-Konto** auf **Für Vorschau registrieren**. Prüfen Sie die Informationen, und klicken Sie auf **OK**. Sie erhalten eine E-Mail, sobald Ihr Abonnement für die öffentliche Vorschau aktiviert wurde.

	![Blatt „Noch heute für die Vorschau anmelden“ für „DocumentDB – Protokollunterstützung für MongoDB“ im Azure-Portal](./media/documentdb-protocol-mongodb/registerforpreview.png)


## Nächste Schritte
- Nachdem Ihnen die Genehmigung für die Vorschauversion erteilt wurde, erhalten Sie eine Begrüßungs-E-Mail mit einer Anleitung, wie Sie das Konto erstellen und die ersten Schritte ausführen.
