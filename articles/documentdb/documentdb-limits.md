# <a name="default-quotas-for-azure-cosmos-db"></a>Standardkontingente für Azure Cosmos DB

Azure Cosmos DB ist eine Datenbank für globale Skalierung, in der Durchsatz und Speicher je nach den Anforderungen Ihrer Anwendung skaliert werden können. Wenn Sie Fragen zur Skalierung von Azure Cosmos DB haben, senden Sie eine E-Mail an askdocdb@microsoft.com.
