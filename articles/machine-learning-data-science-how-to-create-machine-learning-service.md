# Cloud Data Science-Ablauf für das maschinelle Lernen

Dieser Ablaufplan für Cloud Data Science veranschaulicht die Verwendung der Azure-Plattform für Data Science und das Erstellen operationalisierter Modelle mithilfe von Azure Machine Learning. Der Plan führt den Benutzer durch die typischen Data Science-Aufgaben wie das Erfassen von Daten aus verschiedenen Quellen, das Vorbereiten der Daten für die Verwendung in Azure Machine Learning und das Verwenden von Azure Machine Learning zum Erstellen operationalisierter Modelle, die von Endbenutzeranwendungen (programmgesteuert oder anderweitig) genutzt werden können. Der Plan zeigt die wichtigsten Schritte eines umfassenden Data Science-Ablaufs, aber nicht sämtliche Schritte, die erforderlich sind. Auch kann deren Reihenfolge je nach Speicherort, Größe und Komplexität der Daten variieren.

Der Schritt mit der Bezeichnung **Erweiterte Datenverarbeitung in Azure** enthält einen Link zu einem zweiten Plan. Wenn Sie von diesem Plan zurückkehren, fahren Sie mit den nachfolgenden Punkten fort, um die verarbeiteten Daten in Azure Machine Learning Studio zu erfassen und Modelle zu erstellen und bereitzustellen.
