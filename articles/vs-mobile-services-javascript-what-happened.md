###<span id="whathappened">Was ist mit meinem Projekt passiert?</id>

#####Verweise wurden hinzugefügt

Die Microsoft Azure Mobile Service-Bibliothek wurde Ihrem Projekt in Form einer Datei **MobileServices.js** hinzugefügt.
  
#####Werte für die Verbindungszeichenfolge für Mobile Services 

Im Ordner `services\mobileServices\settings` wurde eine neue JavaScript-Datei (JS-Datei) mit einem **MobileServiceClient** generiert. Sie enthält die Anwendungs-URL des ausgewählten mobilen Diensts sowie den zugehörigen Anwendungsschlüssel.
