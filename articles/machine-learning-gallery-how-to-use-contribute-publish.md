# **Machine Learning Gallery** #
Die Azure Machine Learning Gallery ist eine Community-basierte Website auf der Experimente zum Lösen interessanter Probleme vorgestellt werden. Sie ist für Benutzer gedacht, die aus Experimenten anderer lernen oder einfach Azure Machine Learning kennenlernen möchten. Diese Experimente werden in erster Linie durch die Benutzercommunity von Azure Machine Learning bereitgestellt. 

Die Galerie ermöglicht es Benutzern auch, Experimente des Microsoft Azure Machine Learning-Teams zu durchsuchen und kopieren.

**[ZUR GALERIE >>](http://gallery.azureml.net)**

[AZURE.INCLUDE [machine-learning-free-trial](../includes/machine-learning-free-trial.md)]

## Vorteile ##
Nutzen Sie die Galerie, und genießen Sie diese Vorteile:

- Entdecken Sie interessante Experimente. 
- Lernen Sie von anderen Benutzern.

Melden Sie sich mit Ihrem Microsoft-Konto an, um diese zusätzliche Vorteile zu erhalten:

- Kopieren Sie Experimente in Ihrem Arbeitsbereich, und beginnen Sie mit dem Lösen Ihrer eigenen Probleme mit Azure ML.
- Tragen Sie zur Galerie bei, und präsentieren Sie Ihre Arbeit.

## Ermitteln ##
Sie können interessante Experimente in der Galerie suchen und durchsuchen, mit deren Hilfe Sie Ihre Probleme mittels Machine Learning lösen können, oder beginnen Sie mit dem Erstellen von Experimenten mit Azure Machine Learning. 

Sie können die Galerie zunächste verwenden, um Experimente kennenzulernen und Weitere Informationen zu erhalten, ohne sich anzumelden. 
Sie können ganz einfach vor kurzem veröffentlichte Experimente und häufig angesehene Experimente in der Galerie finden, oder suchen Sie nach Experimentnamen, Tags, Algorithmen und anderen Attributen.

Zeigen Sie Experimentbeiträge von einem bestimmten Autor an, indem Sie in einer der Kacheln auf den Autorennamen klicken.

Wenn Sie an einem bestimmten Experiment interessiert sind, können Sie zusätzliche Informationen darüber auf der Detailseite des Experiments lesen. 

Sie können auch über den Kommentarbereich auf jeder Experimantseite kommentieren, Feedback geben oder Fragen stellen. Sie können außerdem interessante Experimente mit Freunden und Kollegen über LinkedIn oder Twitter mittels Freigabefunktionen teilen. 
Sie können auch Links zu diesen Experimente per E-Mail senden, um andere Benutzer einzuladen, die Experimentseiten anzusehen.

## Download ##
Mit der Azure Machine Learning Gallery können Sie it einem einzigen Mausklick problemlos Experimente in den Arbeitsbereich kopieren, und sie sofort selbst verwenden. 
Wenn Sie sich nicht bereits beim Browsen angemeldet haben, werden Sie aufgefordert, sich mit Ihrem Microsoft-Konto anzumelden, bevor das Experiment in Ihren Studio-Arbeitsbereich kopiert wird.

## Gestalten Sie die Galerie mit ##
Sind Sie bereit, Experimente zur Galerie hinzuzufügen?

Folgen Sie diesen Schritten:

- Melden Sie sich beim Studio mit Ihrem Microsoft-Konto an. (Vergewissern Sie sich, dass Ihr Profilbild und Ihr Name in den Microsoft-Konto-Profileinstellungen der gewünschten Anzeige entsprechen.)
- Erstellen Sie Ihr Experiment, und führen Sie es aus.
- Wenn Sie bereit sind, veröffentlichen Sie das Experiment in der Galerie. Klicken Sie im unteren schwarzen Balken des Studios auf "In Galerie veröffentlichen".
![Veröffentlichen](./media/machine-learning-gallery-how-to-use-contribute-publish/publish.png)
Viel Spaß!

## Häufig gestellte Fragen ##
**Ich möchte Änderungen am Workflow des Experiments vornehmen, das ich in die Galerie gestellt habe. Wie kann ich das tun?**

Aktuell werden keine Aktualisierungen für den Workflow von Experimenten unterstützt, die Sie bereits in der Galerie veröffentlicht haben. Sie können solche Änderungen als neues Experiment in der Galerie veröffentlichen und das alte löschen. Wir arbeiten aktiv daran, die Veröffentlichung von Updates der Workflows für bereits bereitgestellte Experimente in der Galerie zu ermöglichen. Bleiben Sie dran.

**Muss ich ein neues Experiment veröffentlichen, auch wenn ich nur Tags oder die Beschreibung bearbeiten möchte?**
Folgende Änderungen können an einem Experiment vorgenommen werden, das Sie bereits in die Galerie gestellt haben, ohne dass eine neue Veröffentlichung nötig ist:

- Experimentname
- Zusammenfassung
- Beschreibung
- Tags
- Bilder

Um diese Felder zu bearbeiten, klicken Sie auf das jeweilige Experiment, das Sie bearbeiten möchten. (Stellen Sie sicher, dass Sie mit Ihrem Microsoft-Konto angemeldet sind.) Dies bringt Sie zur Detailseite des Experiments. Hier werden die Optionen zum Bearbeiten oder Löschen angezeigt. Durch Klicken auf  'Edit' können Sie die oben genannten Felder bearbeiten.

**Ich habe gerade mein Experiment in der Azure Machine Learning Gallery veröffentlicht. Mein Profilbild mit meinen Namen wird nicht angezeigt.**	

Es ist möglich, dass Sie ein anderes Konto als @outlook, @msn, @live oder @hotmail verwenden. Wenn dies der Fall ist, sehen Sie ein Platzhalterbild anstelle des Profilbilds aus den Einstellungen für die Konfiguration des Microsoft-Kontos. Verwenden Sie ein Microsoft-Konto, und stellen Sie das Experiment erneut ein. Das sollte das Problem lösen.

**Ich habe mein Profilfoto in den Konfigurationseinstellungen für mein Microsoft-Konto aktualisiert. Warum werden meine vorhandenen Experimentbeiträge nicht mit dem neuen Profilbild angezeigt?**

Wenn Sie die letzten Profiländerungen (Profilbild oder Vor- oder Nachname) für alle Experimente anzeigen möchten, müssen Sie die älteren Experimente erneut in die Galerie einstellen. Dafür müssen Sie die alten Kopien mit den alten Profileinstellungen löschen.

**Was sind die Anforderungen beim Senden oder Bearbeiten eines Bildes für meine Experimente?**

Die Bilder, die Sie zusammen mit Ihrem Experiment übermitteln, werden verwendet, um eine Kachel für Ihren Beitrag zu erstellen. Es wird empfohlen, dass die Bilder kleiner als 500 KB sind, mit einem Seitenverhältnis von 3:2. Eine Auflösung von 960 x 640 wird empfohlen.

**Was geschieht mit dem Dataset, das ich im Experiment verwendet habe? Wird das Dataset auch in der Galerie veröffentlicht?**

Wenn Ihr Dataset Teil des Experiments ist und nicht über ein Reader-Modul bezogen wird, gehört es zum Experiment und wird mit ihm in der Galerie veröffentlicht. Stellen Sie aus diesem Grund sicher, dass das Dataset, das Sie mit dem Experiment veröffentlichen, über die entsprechenden Lizenzbedingungen verfügt, die Freigabe und Herunterladen durch alle Benutzern zulassen.

**Ich habe ein Experiment, das über ein Reader-Modul Daten aus HDInsight oder SQL bezieht. Es verwendet meine Anmeldeinformationen zum Abrufen der Daten. Wie kann ich ein solches Experiment veröffentlichen und gewährleisten, dass meine Anmeldeinformationen nicht anderen zugänglich werden?**

Zu diesem Zeitpunkt wird das Veröffentlichen von Experimenten, die Anmeldeinformationen verwenden, nicht zugelassen. 

**Wie kann ich Tags trennen?**

Sie können Tags mit Tabs trennen.

**Einige Experimente verfügen über detailreiche Beschreibungen mit umfassenden Renderingfunktionen. Wenn ich versuche, eine Beschreibung einzugeben, wird sie jedoch nur als Text angezeigt.** 

Beschreibungen mit umfassenden Renderingfunktionen ist nicht für alle Galeriebenutzer verfügbar. Wir arbeiten aktiv daran, diese Funktion durch die Unterstützung von Markdown-Rendering verfügbar zu machen.
