# <a name="how-to-code-and-test-azure-functions-locally"></a>Lokales Codieren und Testen von Azure-Funktionen
> [!NOTE]
> Derzeit können Sie in der Vorschau die lokale Entwicklung und den Umgang mit Azure Functions kennen lernen, und die Benutzerfreundlichkeit wird für die endgültige Version deutlich verbessert. Die einfachste Möglichkeit zum lokalen Ausführen des Azure Functions-Hosts besteht darin, die [Azure Functions-CLI](https://go.microsoft.com/fwlink/?linkid=832752) zu verwenden und über npm zu installieren. Derzeit wird nur Windows unterstützt. Die vollständige Dokumentation ist in Kürze verfügbar. Wenn Sie Feedback senden oder Fehler in der Azure Functions-CLI melden möchten, erstellen Sie einen Eintrag im GitHub-Repository [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script).
