# <a name="jenkins-plugins-for-azure"></a>Jenkins-Plug-Ins für Azure

Die folgenden Jenkins-Plug-Ins unterstützen verschiedene Features für die Verwendung mit Azure.
  
| Jenkins-Plug-In | BESCHREIBUNG                                   |
|------------------------------------------------------------------------------|
| [Azure App Service-Plug-In](https://plugins.jenkins.io/azure-app-service)     | Ein Jenkins-Plug-In zum Bereitstellen eines Azure App Service (unterstützt derzeit nur Web-Apps). | 
| [Azure AD-Plug-In](https://plugins.jenkins.io/azure-ad)                       | Ein Jenkins-Plug-In, das die Authentifizierung und Autorisierung über Azure Active Directory unterstützt. | 
| [Azure-Container-Agents-Plug-In](https://plugins.jenkins.io/azure-container-agents) | Das Plug-In für Azure-Container-Agents ermöglicht die Ausführung eines Containers als Agent in Jenkins. | 
| [Azure Container Service-Plug-In](https://plugins.jenkins.io/azure-acs)       | Ein Jenkins-Plug-In zum Bereitstellen von Konfigurationen für den Azure Container Service (AKS). | 
| [Azure Anmeldeinformationen-Plug-In](https://plugins.jenkins.io/azure-credentials)      | Ein Jenkins-Plug-In zum Verwalten von Azure-Anmeldeinformationen. | 
| [Azure Functions-Plug-In](https://plugins.jenkins.io/azure-function)           | Um dieses Plug-In für die Bereitstellung für Azure Function zu verwenden, müssen Sie zunächst über einen Azure-Dienstprinzipal in Ihrer Jenkins-Instanz verfügen. | 
| [Azure Service Fabric-Plug-In](https://plugins.jenkins.io/service-fabric)     | Ein Jenkins-Plug-In für Azure Service Fabric-Projekte für Linux. |
| [Azure Storage-Plug-In](https://plugins.jenkins.io/windows-azure-storage)     | Ein Plug-In zum Hochladen von Buildartefakten in oder Herunterladen von Buildabhängigkeiten aus Microsoft Azure Blob Storage. | 
| [Azure VM Agents-Plug-In](https://plugins.jenkins.io/azure-vm-agents)         | Ein Jenkins-Plug-In zum Erstellen von Jenkins-Agents auf virtuellen Azure-Computern (über die Azure Resource Manager-Vorlage). | 
| [Azure-VM-Skalierungsgruppen-Plug-In](https://plugins.jenkins.io/azure-vmss)           | Ein Jenkins-Plug-In zum Bereitstellen von VM-Images für Azure-VM-Skalierungsgruppen. |
