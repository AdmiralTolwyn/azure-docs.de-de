# <a name="xml-processing"></a>XML-Verarbeitung
Sie können XML-Dokumente, die Sie mit Partnern in B2B-Szenarien austauschen, mit dem Enterprise Integration Pack und Azure Logic Apps einfacher überprüfen und verarbeiten. Mit diesen Methoden können Sie XML-Nachrichten verarbeiten:

* [XML-Überprüfung](logic-apps-enterprise-integration-xml-validation.md "Informationen zur Überprüfung von XML-Nachrichten") – Überprüfen Sie eine Nachricht, die von einem Quellendpunkt stammt, anhand eines bestimmten Schemas.
* [XML-Transformation](../logic-apps/logic-apps-enterprise-integration-transform.md "Informationen zu XML-Nachrichtentransformationen und Zuordnungen") – Konvertieren Sie eine XML-Nachricht basierend auf den Anforderungen eines Zielendpunkts.
* [Flatfilecodierung und -decodierung](logic-apps-enterprise-integration-flatfile.md "Informationen zur Flatfilecodierung und -decodierung") – Codieren oder decodieren Sie eine Flatfile. 
* [XPath](https://msdn.microsoft.com/library/mt643789.aspx) – Reichern Sie eine Nachricht an, und extrahieren Sie bestimmte Eigenschaften aus der Nachricht. Mithilfe der extrahierten Eigenschaften können Sie die Nachricht an einen Ziel- oder Zwischenendpunkt weiterleiten.
