# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>Welche Produkte können über das Cloudpartnerportal veröffentlicht werden?

Das Cloudpartnerportal ist der zentrale Punkt für die Veröffentlichung Ihres Azure Certified-Images virtueller Computer auf dem Microsoft Azure Marketplace oder Ihrer Cortana Intelligence/Microsoft R Solutions-Anwendung bei Microsoft AppSource.  

Images virtueller Computer sind vorkonfigurierte Images mit einem komplett installierten Betriebssystem und einer oder mehreren Anwendungen. Wenn Sie Ihr Image virtueller Computer auf Azure Marketplace veröffentlichen möchten, müssen Sie die [technischen Voraussetzungen](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites) für die Zertifizierung Ihres Images erfüllen.
