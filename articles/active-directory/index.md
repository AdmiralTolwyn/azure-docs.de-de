# <a name="azure-active-directory-documentation"></a>Dokumentation zu Azure Active Directory

Azure Active Directory (Azure AD) ist der mehrinstanzenfähige cloudbasierte Verzeichnis- und Identitätsverwaltungsdienst von Microsoft.  Azure AD bietet Identitäts- und Zugriffsfunktionen für Anwendungen, die in Azure und lokal ausgeführt werden.
