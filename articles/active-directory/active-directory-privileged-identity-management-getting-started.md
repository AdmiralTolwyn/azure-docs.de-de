# Erste Schritte mit Azure Privileged Identity Management

## Starten der Erweiterung Privileged Identity Management

1.  Erstellen Sie ein Verzeichnis in Azure Active Directory.
2.  Fügen Sie Azure Active Directory Benutzer hinzu.
3.  Erstellen Sie im Verzeichnis einige globale Administratoren.
4.  Melden Sie sich mit einem globalen Administratorkonto beim [Azure-Portal](https://portal.azure.com/) an.
5.  Klicken Sie in der linken oberen Ecke des Azure-Portals auf den **Benutzernamen**.
6.  Wählen Sie das **Verzeichnis** aus, das Sie gerade erstellt haben.
7.  Klicken Sie im linken Navigationsbereich auf das Symbol **Neu**.
8.  Wählen Sie im Menü "Erstellen" **Sicherheit und Identität** aus.
9.  Wählen Sie im Menü **Sicherheit und Identität** die Option **Azure AD Privileged Identity Management** aus.
10. Lassen Sie das Kontrollkästchen **An Startmenü anheften** aktiviert, und klicken Sie dann auf die Schaltfläche **Erstellen**. Das Privileged Identity Management-Dashboard wird geöffnet.
