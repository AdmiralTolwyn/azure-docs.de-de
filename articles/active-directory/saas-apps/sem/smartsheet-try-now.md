# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet und Azure Active Directory

Beschleunigen Sie die Einführung von Smartsheet in Ihrem Unternehmen, und steigern Sie die Benutzerakzeptanz mit unserer vorgefertigten Integration. Ermöglichen Sie Benutzern das schnelle Zusammenarbeiten, indem Sie Smartsheet mit Azure Active Directory verbinden (Azure AD).

![Zeigt das Smartsheet-App-Logo an](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Smartsheet mit Azure AD testen >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Was können Sie mit Smartsheet und Azure AD erreichen?

- **Einfaches einmaliges Anmelden**: Beseitigen Sie mehrere Benutzernamen und Kennwörter und vereinfachen Sie den Zugriff, indem Sie es Benutzern ermöglichen, sich mit ihren Azure AD-Konten bei Smartsheet anzumelden.
- **Automatisieren der Benutzerbereitstellung**: Automatisieren Sie die Erstellung, Verwaltung und Entfernung von Benutzerkonten in Smartsheet, wenn Benutzer Ihrer Organisation beitreten, sie verschieben oder verlassen.
- **Sichern, Verwalten und Überwachen des Zugriffs**: Sichern Sie den Zugriff auf Smartsheet durch die Anwendung von Richtlinien für bedingten Zugriff, steuern Sie den Zugriff durch Zugriffsprüfungen und überwachen Sie den Zugriff mit Anmeldeprotokollen.

## <a name="get-started"></a>Erste Schritte

Richten Sie die [Benutzerbereitstellung](https://aka.ms/SmartsheetProvisioningDoc) und das [einmalige Anmelden](https://aka.ms/SmartsheetSSODoc) für Smartsheet mit unseren einfachen Leitfäden zur Konfiguration ein.
