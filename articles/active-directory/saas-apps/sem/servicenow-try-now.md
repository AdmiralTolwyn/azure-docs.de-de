# <a name="servicenow-and-azure-active-directory"></a>ServiceNow und Azure Active Directory

Beschleunigen Sie die Einführung von ServiceNow in Ihrem Unternehmen, und steigern Sie die Benutzerakzeptanz mit unserer vorgefertigten Integration. Optimieren Sie Ihr IT-Service-Management, indem Sie ServiceNow mit Azure Active Directory (Azure AD) verbinden.

![Zeigt das ServiceNow-App-Logo an](./media/servicenow.png)

> [!div class="nextstepaction"]
> [ServiceNow mit Azure AD testen >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Was können Sie mit ServiceNow und Azure AD erreichen?

- **Einfaches einmaliges Anmelden**: Beseitigen Sie mehrere Benutzernamen und Kennwörter und vereinfachen Sie den Zugriff, indem Sie es Benutzern ermöglichen, sich mit ihren Azure AD-Konten bei ServiceNow anzumelden.
- **Automatisieren der Benutzerbereitstellung**: Automatisieren Sie die Erstellung, Verwaltung und Entfernung von Benutzerkonten in ServiceNow, wenn Benutzer Ihrer Organisation beitreten, sie verschieben oder verlassen.
- **Sichern, Verwalten und Überwachen des Zugriffs**: Sichern Sie den Zugriff auf ServiceNow durch die Anwendung von Richtlinien für bedingten Zugriff, steuern Sie den Zugriff durch Zugriffsprüfungen und überwachen Sie den Zugriff mit Anmeldeprotokollen.

## <a name="get-started"></a>Erste Schritte

Richten Sie die [Benutzerbereitstellung](https://aka.ms/ServiceNowProvisioning) und das [einmalige Anmelden](https://aka.ms/ServiceNowSSO) für ServiceNow mit unseren einfachen Leitfäden zur Konfiguration ein.
