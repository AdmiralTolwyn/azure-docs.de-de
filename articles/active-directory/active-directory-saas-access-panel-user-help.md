# <a name="common-access-panel-tasks"></a>Allgemeine Aufgaben für den Zugriffsbereich

Der Zugriffsbereich ist ein Webportal, über das Sie folgende Aufgaben ausführen können:

- Verwalten und Starten cloudbasierter Anwendungen
- Verwalten von Gruppenmitgliedschaften
- Verwalten von Profileinstellungen 

![Anpassung des Zugriffsbereichs][1] 

Dieses Thema enthält Anleitungen zu allgemeinen Aufgaben sowie zur Behandlung von Problemen.



## <a name="how-to-troubleshoot-accessing-an-app"></a>Behandeln von App-Zugriffsproblemen

1.  Überprüfen Sie, ob die Zugriffsbereichserweiterung ordnungsgemäß in Ihrem Browser installiert ist.  
Eine Anleitung zum Behandeln von Problemen mit der IE-Erweiterung finden Sie [hier](active-directory-saas-ie-troubleshooting.md).
2.  Vergewissern Sie sich, dass sich Ihr Browser nicht im privaten Modus oder im Inkognitomodus befindet.
3.  Sollten Sie weiterhin nicht auf Ihre Apps zugreifen können, wenden Sie sich an den zuständigen Administrator oder Helpdesk.

## <a name="how-to-change-your-password"></a>Ändern des Kennworts

1.  Klicken Sie rechts oben im **Zugriffsbereich** auf das Benutzersteuerelement und anschließend auf **Profil**.
2.  Klicken Sie unter **Konto verwalten** auf **Kennwort ändern**.
3.  Geben Sie Ihr altes Kennwort ein, geben Sie Ihr neues Kennwort ein, und bestätigen Sie das neue Kennwort.
4.  Klicken Sie auf **Senden**.

Ausführlichere Informationen finden Sie unter [Aktualisieren Ihres eigenen Kennworts](active-directory-passwords-update-your-own-password.md).


## <a name="how-to-setup-password-reset"></a>Einrichten der Kennwortzurücksetzung

1.  Klicken Sie rechts oben im **Zugriffsbereich** auf das Benutzersteuerelement und anschließend auf **Profil**.
2.  Klicken Sie unter **Konto verwalten** auf **Self-Service-Kennwortzurücksetzung einrichten**.
3.  Führen Sie die im Dialogfeld angegebenen Schritte aus, und übermitteln Sie die Daten.

Ausführlichere Informationen finden Sie unter [Zurücksetzen des Kennworts](active-directory-passwords-update-your-own-password.md#how-to-reset-your-password).


## <a name="how-to-add-new-apps-or-request-permission-to-app-access"></a>Hinzufügen neuer Apps oder Anfordern von App-Zugriffsberechtigungen

1.  Klicken Sie im **Zugriffsbereich** auf **App hinzufügen**.
2.  Suchen Sie nach der App, die Sie hinzufügen möchten, und klicken Sie auf **Hinzufügen**.  
Falls die App eine Berechtigung erfordert, ist unter Umständen eine Administratorgenehmigung erforderlich.

Sollten Sie die gewünschte App nicht finden, wenden Sie sich an den zuständigen Administrator.


## <a name="how-to-manage-your-group-memberships"></a>Verwalten von Gruppenmitgliedschaften

1. Klicken Sie im **Zugriffsbereich** auf die App-Kachel **Gruppen**.
2. Klicken Sie zum Erstellen einer Gruppe unter **Gruppen in meinem Besitz** auf **Gruppe erstellen**, und folgen Sie den Anweisungen.
3. Klicken Sie zum Beitreten zu einer Gruppe unter **Groups I'm in** (Gruppen, denen ich angehöre) auf **Gruppe beitreten**, und folgen Sie den Anweisungen.

**Anmerkungen:**
  
- Wenn Sie auf eine Gruppe klicken, der Sie bereits angehören, können Sie Details anzeigen und die Gruppe verlassen.
- Wenn Sie auf eine Gruppe klicken, die sich in Ihrem Besitz befindet, können Sie Details anzeigen, Mitglieder hinzufügen/entfernen und die Gruppe verlassen.
