# Azure Privileged Identity Management: Aktivieren oder Deaktivieren einer Rolle

## Aktivieren oder Deaktivieren einer Rolle

1. Melden Sie sich bei [http://portal.azure.com](http://portal.azure.com) an.
2. Führen Sie die Schritte in [Erste Schritte mit Azure Privileged Identity Management](active-directory-privileged-identity-management-getting-started.md) aus, um Azure PIM im Portal-Dashboard zu platzieren.
3. Nachdem Sie die Schritte im Sicherheits-Assistenten abgeschlossen haben, wird das Hauptmenü von Azure PIM angezeigt.
4. Klicken Sie auf **Meine Rolle aktivieren**.
5. Eine Liste der Ihnen zugewiesenen Rollen wird angezeigt.
6. Klicken Sie auf die Rolle, die Sie aktivieren möchten.
7. Klicken Sie auf **Aktivieren**. Das Blatt **Rollenaktivierung anfordern** wird angezeigt.
8. Geben Sie im Textfeld den Grund für die Aktivierungsanforderung ein.
9. Klicken Sie auf **OK**. Die Rolle wird jetzt aktiviert.
10. Nachdem eine Rolle aktiviert wurde, können Sie sie deaktivieren, indem Sie auf **Deaktivieren** klicken. Außerdem kann die Rolle des Benutzers mithilfe der Schritte in [Hinzufügen oder Entfernen einer Rolle](active-directory-privileged-identity-management-how-to-add-role-to-user.md) entfernt werden.

- Weitere Informationen zu Rollenaktivierungseinstellungen finden Sie unter [Konfigurieren der Rollenaktivierung](active-directory-privileged-identity-management-how-to-configure-role-activation.md).
- Weitere Informationen zu Sicherheitswarnungen für Rollenaktivierungseinstellungen finden Sie unter [Konfigurieren von Sicherheitswarnungen](active-directory-privileged-identity-management-how-to-configure-security-alerts).
