# <a name="updates-to-the-sign-in-experience-for-online-services"></a>Updates für den Anmeldevorgang für Onlinedienste

Wir aktualisieren den Anmeldevorgang für die Microsoft-Onlinedienste wie Office 365 und Azure. Es handelt sich zwar nur um eine optische Änderung, aber Sie werden feststellen, dass die Bildschirme klarer strukturiert und weniger überladen sind, sodass Sie sich ganz auf die Anmeldung konzentrieren können.

## <a name="current-ui-versus-new-ui"></a>Aktuelle Benutzeroberfläche im Vergleich zur neuen Benutzeroberfläche

Sehen Sie sich die vorhandene Benutzeroberfläche im Vergleich zur neuen Benutzeroberfläche an. Die neue Benutzeroberfläche ist übersichtlicher und zeigt ein moderneres Design.

![Bisherige Microsoft-Benutzeroberfläche für die Anmeldung](./media/sign-in-experience-updates/microsoft-sign-in-experience-old.jpg)

![Neue Microsoft-Benutzeroberfläche für die Anmeldung](./media/sign-in-experience-updates/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>Wann wird die neue Benutzeroberfläche angezeigt?

Die Änderung wird nach folgendem Zeitplan umgesetzt:

| Zeitraum | Change |
|-----------|--------|
| Anfang Mai | Auf der vorhandenen Anmeldeseite wird ein Benachrichtigungsbanner angezeigt. Hier können Sie sich über die Änderungen informieren und sich darauf vorbereiten. |
| Mitte Mai   | Das Benachrichtigungsbanner enthält einen Link, über den Sie sich eine Vorschau der aktualisierten Benutzeroberfläche ansehen können. Sie können während der Vorschau einen Link verwenden, der Sie zurück zur vorhandenen Oberfläche führt. |
| Mitte Juni  | Die aktualisierte Benutzeroberfläche für die Anmeldung ist verfügbar und allen Benutzer wird standardmäßig die neue Benutzeroberfläche angezeigt. |

## <a name="related-info"></a>Verwandte Informationen

Wenn Sie Administrator sind, finden Sie hier weitere produktbezogene Informationen zu den Updates:

- **Office 365**. Informationen zu den Updates erhalten Sie im [Office 365 Message Center](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail). Sie müssen sich als Administrator anmelden, um auf diese Website zuzugreifen.

- **Andere Onlinedienste**. Informationen zu den Updates erhalten Sie in diesem[Blogbeitrag](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/).
