# Windows 10 für Unternehmen: Möglichkeiten der geschäftlichen Nutzung von Geräten

Windows 10 bietet Ihnen die Möglichkeit, Azure Active Directory (Azure AD) zu nutzen. Sie können Windows 10-Geräte mit Azure AD verbinden, damit sich Benutzer mit Azure AD-Konten oder durch Hinzufügen ihrer Azure-IDs bei Windows anmelden können, um Zugriff auf Unternehmens-Apps und -Ressourcen zu erhalten.
