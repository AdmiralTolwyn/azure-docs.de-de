# Was ist Azure Active Directory?


###Weitere Artikel zu diesem Thema
[Was ist Azure AD?](active-directory-whatis.md)<br> [Wie funktioniert Azure AD?](active-directory-works.md)<br> [Erste Schritte](active-directory-get-started.md)<br> [Nächste Schritte](active-directory-next-steps.md)<br> [Weitere Informationen](active-directory-learn-map.md)

Azure Active Directory (Azure AD) bietet Ihrem Unternehmen eine einfache Möglichkeit für die Identitäts- und Zugriffsverwaltung – in der Cloud und lokal. Ihre Benutzer können ein Geschäfts- oder Schulkonto für das einmalige Anmelden bei allen Cloud- und lokalen Webanwendungen über ihr bevorzugtes Gerät (z. B. iOS-, Mac OS X-, Android- und Windows-Geräte) verwenden. Ihr Unternehmen kann vertrauliche Daten und Anwendungen mit der integrierten Multi-Factor Authentication lokal und in der Cloud schützen und so einen sicheren lokalen und Remotezugriff gewährleisten. Zudem können Sie die lokalen Verzeichnisse erweitern, sodass Wissensarbeiter ein einziges Unternehmenskonto verwenden können, um auf sichere und konsistente Weise auf ihre Unternehmensressourcen zuzugreifen. Azure AD bietet außerdem umfassende Berichte, Analysen und Self-Service-Funktionen zur Kostensenkung und Erhöhung der Sicherheit. Und mit der Unternehmensstaffelung und der SLA von Azure AD wird Ihr Unternehmen jederzeit am Laufen gehalten.

![][1]


> [AZURE.NOTE]Um Azure Active Directory verwenden zu können, benötigen Sie ein Azure-Konto. Wenn Sie nicht über ein Azure-Konto verfügen, können Sie sich [für ein kostenloses Azure-Konto registrieren](http://azure.microsoft.com/pricing/free-trial/).
