# <a name="my-apps-portal-overview"></a>Übersicht über das Portal „Meine Apps“

Wenn Sie über ein Geschäfts-, Schul- oder Unikonto verfügen und Ihr Administrator Ihnen Zugriff erteilt hat, können Sie das Portal „Meine Apps“ verwenden und cloudbasierte Apps starten.

>[!Important]
>Dieser Inhalt richtet sich an Benutzer. Administratoren finden in der [Azure Active Directory-Dokumentation](https://docs.microsoft.com/azure/active-directory) weitere Informationen zum Einrichten und Verwalten der Azure Active Directory-Umgebung (Azure AD).
