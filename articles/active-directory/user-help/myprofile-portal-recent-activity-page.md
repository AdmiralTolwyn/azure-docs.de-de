# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>Anzeigen der letzten Anmeldeaktivität im Portal „Mein Profil (Vorschauversion)
Im Portal **Mein Profil** können Sie auf der Seite **Letzte Aktivität** alle kürzlich mit Ihrem Geschäfts-, Schul- oder Unikonto erfolgten Anmeldeaktivitäten anzeigen.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>Anzeigen Ihrer letzten Anmeldeaktivität
1. Melden Sie sich bei Ihrem Geschäfts-, Schul- oder Unikonto an, und rufen Sie die Seite https://myprofile.microsoft.com/ auf.

2. Wählen Sie im linken Navigationsbereich **Letzte Aktivität** aus, oder wählen Sie im Block **Letzte Aktivität** den Link **Aktivität überprüfen** aus.

    ![Seite „Mein Profil“ mit hervorgehobenen Links für die letzte Aktivität](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. Erweitern Sie alle Anmeldeelemente, und überprüfen Sie diese. Stellen Sie sicher, dass Sie alle erkennen. Wenn Sie ein Anmeldeelement finden, das Ihnen unbekannt erscheint, empfehlen wir dringend, Ihr Kennwort zu ändern, um Ihr Konto vor einer Sicherheitsgefährdung zu schützen.

    ![Seite „Letzte Aktivität“ mit erweiterten Anmeldedetails](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >Wenn Sie annehmen, dass Ihr Konto kompromittiert wurde, finden Sie unter [Zugreifen auf ein kompromittiertes Microsoft-Konto](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account) weitere Hilfe beim Zurücksetzen Ihres Kennworts oder beim Entsperren Ihres Kontos.

## <a name="next-steps"></a>Nächste Schritte

- Nachdem Sie Ihre letzte Anmeldeaktivität angezeigt haben, können Sie wahlweise Ihre [Sicherheitsinformationen](user-help-security-info-overview.md), verbundene [Geräte](myprofile-portal-devices-page.md) oder [Organisationen](myprofile-portal-organizations-page.md) anzeigen oder verwalten bzw. anzeigen, wie Ihre Organisation [Ihre Daten verwendet](myprofile-portal-privacy-page.md).
