# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Aktivieren der Vorschauversion zum Verwalten Ihrer verwalteten Domäne über das Azure-Portal

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Ihre vorhandene verwaltete Domäne wird im Azure-Portal (Vorschauversion) nicht angezeigt
Sie verfügen über eine vorhandene verwaltete [Azure AD Domain Services](active-directory-ds-overview.md)-Domäne, die mit dem klassischen Azure-Portal erstellt wurde. Wenn Sie zum Azure-Portal navigieren, führt das Blatt für die Azure AD Domain Services (Vorschauversion) diese vorhandene verwaltete Domäne nicht auf.

![Vorhandene verwaltete Domäne wird im Azure-Portal nicht angezeigt](./media/getting-started/missing-domain-services-instance.png)

Dies liegt daran, dass Ihre vorhandene verwaltete Domäne noch nicht zum neuen Azure-Portal migriert wurde. Sie können uns zum Aktivieren der neuen Azure-Portal-Umgebung (Vorschauversion) für Ihre vorhandene verwaltete Domäne kontaktieren.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Verwenden Sie die Vorschauversion der Azure-Portal-Umgebung für Azure AD Domain Services
Befolgen Sie die Anweisungen auf der Seite [So erreichen Sie uns](active-directory-ds-contact-us.md), und schreiben Sie uns eine E-Mail. Wir freuen uns, die Vorschauversion des Azure-Portals für Ihre vorhandene verwaltete Domäne zu aktivieren.

> [!NOTE]
> Wenn die neue Azure-Portal-Umgebung für Azure AD Domain Services allgemein verfügbar (GA) ist, werden alle vorhandenen verwalteten Domänen automatisch migriert. Dann werden Sie in der Lage sein, Ihre vorhandene verwaltete Domäne im Azure-Portal zu verwalten.
