## Unterstützte Azure-Regionen
Azure Active Directory-Domänendienste sind in den folgenden Azure-Regionen verfügbar:

- USA
  - USA (Mitte)
  - USA (Ost)
  - USA (Ost 2)
  - USA (Mitte/Süden)
  - USA (West)

- Asien
  - Ostasien
  - Südostasien
