# Azure AD-Domänendienste (Vorschau) – so erreichen Sie uns

## So erreichen Sie uns
Falls Probleme mit Ihrer verwalteten Domäne auftreten, überprüfen Sie, ob diese sich mit den Schritten in diesem [Leitfaden zur Problembehandlung](active-directory-ds-troubleshooting.md) beheben lassen. Falls sich die Probleme nicht beseitigen lassen, können Sie sich gern an uns wenden.

Sie können uns eine E-Mail an [Azure AD-Domänendienste – Feedback](mailto:aaddsfb@microsoft.com) senden.

Stellen Sie folgende Informationen bereit, damit wir das Problem untersuchen können:

- Die **Mandanten-ID** für Ihr Azure AD-Verzeichnis. Bei der Mandanten-ID handeld es sich um die GUID, die Sie in der Adressleiste des Browsers sehen, wenn Sie auf die Seite navigieren, auf der Sie die Azure AD-Domänendienste für Ihr Verzeichnis aktivieren.

- Den **DNS-Domänennamen**, den Sie für Ihre über die Azure AD-Domänendienste verwaltete Domäne konfiguriert haben.


## Feedback geben
Sie können uns Ihr Feedback zu den Azure AD-Domänendiensten auch gerne über den **[Azure Active Directory User Voice-Kanal](https://feedback.azure.com/forums/169401-azure-active-directory/)** mitteilen.

Stellen Sie Ihrer Frage oder Ihrem Feedback das Kürzel **„AADDS“** voran, um uns zu erreichen.
