# <a name="virtual-network-documentation"></a>Dokumentation zu Virtual Network

Hier erfahren Sie, wie Sie Virtual Network (VNet) – eine logische Darstellung Ihres Netzwerks in der Cloud – verwenden. Die Dokumentation zeigt, wie Sie ein VNet einrichten und verwalten, wie Sie ein VNet zum Starten von virtuellen Computern oder Clouddiensten verwenden, wie Sie das VNet mit Ihrem lokalen Netzwerk verbinden und wie Sie Ihr Netzwerk auf die Cloud ausdehnen, ohne die Kontrolle über Ressourcen aufzugeben.
