#<a name="virtual-network-documentation"></a>Dokumentation zu Virtual Network

Bei einer Azure Virtual Network-Instanz (VNet) handelt es sich um eine logische Darstellung Ihres eigenen Netzwerks in der Cloud.  Bei einem VNet haben Sie die vollständige Kontrolle über Elemente des Netzwerks. Außerdem können Sie das VNet in Subnetze segmentieren und damit Azure Virtual Machines oder Cloud Services starten.  Das VNet kann zudem unter Verwendung einer der in Azure verfügbaren Verbindungsoptionen mit Ihrem lokalen Netzwerk verbunden werden, um Ihr Netzwerk in die Cloud zu erweitern, ohne die Kontrolle über Ressourcen abzugeben.
