# <a name="transform-your-iot-data-preview-features"></a>Transformieren Ihrer IoT-Daten (Vorschaufeatures)

[!INCLUDE [iot-central-pnp-original](../../../includes/iot-central-pnp-original-note.md)]

Als Anwendungsplattform bietet IoT Central mehrere wichtige Facetten, mit denen Sie Ihre IoT-Daten in die Geschäftseinblicke transformieren können, die zu verwertbaren Erkenntnissen führen. IoT Central bietet Möglichkeiten zum Ausweiten Ihrer IoT-Daten auf externe Systeme, um Integrationen in Branchenanwendungen und benutzerdefinierte Anwendungen zu erstellen. Sie können die Integrität und den Betrieb von Geräten überwachen, indem Sie Regeln erstellen, die Techniker über einen mobilen Dienst benachrichtigen. Sie können durch benutzerdefinierte Analysen und maschinelles Lernen spezifische Geschäftseinblicke generieren, indem Sie unformatierte Telemetriedaten in Azure-Dienste exportieren. Sie können Dienste und Tools erstellen, mit denen Sie Geräte überwachen und steuern sowie Ihre IoT Central-App mithilfe der öffentlichen APIs verwalten. 

![Transformation in IoT Central – Übersicht](media/overview-iot-central-transform/transform.png)

## <a name="monitor-device-health-and-operations-using-rules"></a>Überwachen von Geräteintegrität und -vorgängen mithilfe von Regeln
Sobald Ihre Computer verbunden sind und Daten senden, können Sie ermitteln, auf welchen Computern Probleme auftreten oder Fehlermeldungen gesendet werden, damit sie – nach der Behebung – mit minimalen Ausfallzeiten ordnungsgemäß ausgeführt werden können. Sie können in Ihrer IoT Central-App Regeln zum Überwachen der Telemetriedaten erstellen, die von ihren Geräten gesendet werden und Sie warnen, wenn ein Schwellenwert überschritten oder eine bestimmte Ereignismeldung gesendet wurde. Sie können Aktionen wie z. B. E-Mail-Aktionen und Webhooks für ihre Regeln konfigurieren, damit die richtigen Personen und die richtigen Downstreamsysteme benachrichtigt werden. Hier finden Sie weitere Informationen zu Regeln.

## <a name="run-custom-analytics-and-processing-on-your-exported-data"></a>Ausführen benutzerdefinierter Analysen und Verarbeiten für Ihre exportierten Daten
Sie können sehr benutzerdefinierte Geschäftseinblicke generieren, z. B. Computereffizienztrends ermitteln oder den zukünftigen Energieverbrauch auf Werksebene vorhersagen. Dazu erstellen Sie benutzerdefinierte Analysepipelines, um Ihre unformatierten IoT-Telemetriedaten zu verarbeiten und das Endergebnis zu speichern. Sie können Datenexporte in Ihrer IoT Central-App erstellen, um Telemetriedaten, Geräteeigenschaften und Lebenszyklus sowie Informationen zu Änderungen an Gerätevorlagen in andere Azure-Dienste zu exportieren, damit Sie die für Sie besonders wertvollen Daten in den Tools analysieren, speichern und visualisieren können. Hier erfahren Sie mehr zum Datenexport.

## <a name="build-custom-iot-solutions-and-integrations-with-apis"></a>Erstellen von benutzerdefinierten IoT-Lösungen und -Integrationen in APIs
Sie können IoT-Lösungen wie mobile Begleit-Apps erstellen, die Geräte remote steuern und neue Geräte einrichten können, oder benutzerdefinierte Integrationen in vorhandene Branchenanwendungen für die Interaktion mit ihren IoT-Geräten und -Daten erstellen, die auf Ihr Unternehmen zugeschnitten sind. Sie können IoT Central als Backbone für Gerätemodellierung, Onboarding, Verwaltung und Datenzugriff verwenden. In diesem Lernmodul erfahren Sie mehr zur öffentlichen API.
