# <a name="azure-netapp-files-documentation"></a>Azure NetApp Files-Dokumentation

Erfahren Sie, wie Sie mithilfe von Azure NetApp Files NetApp-Konten, Kapazitätspools und Volumes erstellen und verwalten, die Exportrichtlinie konfigurieren, ein Volume für einen virtuellen Computer bereitstellen und Momentaufnahmen verwalten.

> [!IMPORTANT] 
> Alle in dieser Dokumentation beschriebenen Aufgaben setzen voraus, dass Sie am Public Preview-Programm teilnehmen und auf der Whitelist für den Zugriff auf den Azure-Ressourcenanbieter „Microsoft.NetApp“ stehen. Unter [Register for Azure NetApp Files](azure-netapp-files-register.md) (Registrieren für Azure NetApp Files) finden Sie Informationen zur Teilnahme am Public Preview-Programm.
