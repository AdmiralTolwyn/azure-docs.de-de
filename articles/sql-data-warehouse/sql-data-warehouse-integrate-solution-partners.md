# SQL Data Warehouse-Lösungspartner #

Für die Erstellung Ihrer eigenen End-to-End-Data Warehouse-Lösung können Sie auf eine Vielzahl branchenführender Tools zurückgreifen. In diesem Artikel sind Partnerunternehmen aufgeführt, die Lösungen für SQL Data Warehouse anbieten.

Geschäftskunden im gesamten Unternehmen können dann Informationen mit erstklassigen BI-Tools protokollieren und analysieren.| [Weitere Informationen][birst_more] |
|![][bryte_systems_logo]|**Bryte Systems**<br>Bryte Systems automatisiert die Datenübertragung in SQL Data Warehouse aus einer Vielzahl von Datenbanken und Quellen. Über schnelle funktionale Verbesserungen und Updates können Sie fortlaufend von neuen und verbesserten MicroERD-Funktionen profitieren. | [Weitere Informationen][inbrein_more] |

Deploy Anywhere."-Funktionen der Informatica-Plattform. Weltweit setzen über 5.500 Unternehmen auf Informatica, um ihre Informationsbestände optimal zu nutzen.| [Weitere Informationen][informatica_more] |

Durch die Visualisierung der Daten können Benutzer im Hinblick auf schnelle Entscheidungsprozesse ihre eigene Berichts- und Datenermittlung durchführen.| [Weitere Informationen][jinfonet_more] |
|![][logianalytics_logo]|**LogiAnalytics**<br>Zusammen können Logi Analytics und Microsoft Azure SQL Data Warehouse Ihrem Unternehmen nun ermöglichen, die weltweit größten und vielfältigsten DataSets zu sammeln, zu analysieren und unmittelbar auf diese zu reagieren, um die Leistungsfähigkeit des Unternehmens zu steigern. | [Weitere Informationen][logianalytics_more] |
|![][looker_logo]|**Looker**<br>Looker ermöglicht allen Mitarbeitern Ihres Unternehmens, die Daten zu untersuchen und zu analysieren, die die Grundlage für die Geschäftsentwicklung bilden, während Datenanalysten diese Daten gleichzeitig in einer flexiblen und wiederverwendbaren Modellierungsebene kontrollieren und überprüfen können. Durch die Nutzung von Looker als Katalysator konnten Unternehmen ihre Unternehmenskultur grundlegend ändern. | [Weitere Informationen][looker_more] |

Qlik bietet Einblicke und Klarheit, wo sie am meisten benötigt werden: bei den Entscheidern. | [Weitere Informationen][qlik_more] |
|![][redpoint_global_logo]|**Redpoint Global**<br>Mit RedPoint Interaction und RedPoint Data Management können Marketer alle Daten für die Entwicklung des kanalübergreifenden Customer Engagement wirksam einsetzen sowie eine strukturierte und unstrukturierte Datenverwaltung durchführen. Durch die Nutzung von Microsoft Azure SQL Data Warehouse können Benutzer von RedPoint den Wert all ihrer strukturierten oder unstrukturierten Daten maximieren, um die individualisierten und kontextbezogenen Interaktionen bereitstellen zu können, die für die Bindung der heute auf allen Kanälen aktiven Kunden erforderlich sind. Die Drag & Drop-Benutzeroberfläche macht das Entwerfen und Ausführen der Datenverwaltungsprozesse sehr leicht. | [Weitere Informationen][redpoint_global_more] |
|![][sisense_logo]|**SiSense**<br>SiSense ist ein vollständiger Business Intelligence-Software-Stack mit Tools, die ein Unternehmen für die Analyse und Visualisierung von Daten benötigt: eine leistungsstarke Analysedatenbank, die Möglichkeit, mehrere Datenquellen zusammenzuführen, einfache Datenextraktion (ETL) und webbasierte Datenvisualisierung. Analysieren und visualisieren Sie große DataSets noch heute mit SiSense BI und Analytics. | [Weitere Informationen][sisense_more] |
|![][snaplogic_logo]|**SnapLogic**<br>Mit der SnapLogic iPaaS-Lösung (Integration Platform-as-a-Service) können Kunden Daten auf schnelle Weise in und aus Microsoft Azure SQL Data Warehouse übertragen. SnapLogic bietet die Möglichkeit der Integration von Hunderten von Anwendungen, Diensten und IoT in der Cloud, lokal oder im Hybridmodus für schnelle und wirksame Einblicke. | [Weitere Informationen][snaplogic_more] |
|![][sql_sentry_logo]|**SQL Sentry**<br>SQL Sentry Performance Advisor bietet Echtzeit- und Verlaufssichten der wichtigsten Leistungsmetriken für Azure SQL Data Warehouse, sodass Kunden Leistungstrends erkennen und für detaillierte Analysen und Problembehandlungen einfach auf ausführlichere Informationen zugreifen können. Mit Performance Advisor können Leistungsdaten für jeden beliebigen Zeitpunkt oder Datumsbereich angezeigt werden. | [Weitere Informationen][sql_sentry_more] |
|![][tableau_logo]|**Tableau**<br>Mit den Self-Service-Analysefunktionen von Tableau können alle Benutzer ihre Daten, angefangen von Flatfiles bis hin zu Datenbanken, einsehen und analysieren. Tableau verfügt über einen systemeigenen, optimierten Connector für Microsoft Azure SQL Data Warehouse, der Livedaten- und In-Memory-Analysen unterstützt. | [Weitere Informationen][tableau_more] |
|![][targit_logo]|**Targit**<br>Targit Decision Suite ist eine BI- und Analyseplattform, die Echtzeit-Dashboards, Self-Service-Analysen, benutzerfreundliche Berichte, beeindruckende mobile Funktionen und eine einfache Datenermittlungstechnologie in einer einzigen einheitlichen Lösung bietet. Targit liefert Unternehmen Impulse zum Handeln. | [Weitere Informationen][targit_more] |

Stellen Sie eine Verbindung mit Azure SQL Data Warehouse her, und erstellen und teilen Sie dann Berichte und Dashboards mithilfe preisgekrönter Business Intelligence- und Location Intelligence-Funktionen. | [Weitere Informationen][yellowfin_more] |
