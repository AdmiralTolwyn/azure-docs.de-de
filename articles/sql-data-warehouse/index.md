# <a name="sql-data-warehouse-documentation"></a>Dokumentation zu SQL Data Warehouse

In Azure SQL Data Warehouse ist die relationale SQL Server-Datenbank mit Azure-Cloudfunktionen für das horizontale Hochskalieren kombiniert.  SQL Data Warehouse basiert auf unserer MPP-Architektur (Massively Parallel Processing) und ist für die Verarbeitung Ihrer Enterprise-Workloads geeignet.
