# Nutzungsverfolgung mit benutzerdefinierten Ereignissen und Metriken

### In Kürze verfügbar

Mithilfe des Application Insights SDK können Sie Codezeilen in Ihre Anwendung einfügen, um die Benutzeraktivitäten zu überwachen, damit Sie Ihre Anwendung an deren Bedürfnisse anpassen können.

Wir sind dabei, dieses Feature auf das neue Application Insights-Portal in Microsoft Azure zu übertragen. (Sie können die [Funktionsweise im alten Portal][Funktionsweise im alten Portal] anzeigen.)

Bereits jetzt können Sie [Diagnose-Ereignisprotokolle, die mit Trace, NLog oder Log4Net generiert wurden, erfassen und durchsuchen][Diagnose-Ereignisprotokolle, die mit Trace, NLog oder Log4Net generiert wurden, erfassen und durchsuchen].
