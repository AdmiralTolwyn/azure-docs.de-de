# Nutzungsverfolgung mit benutzerdefinierten Ereignissen und Metriken

### In Kürze verfügbar

Mithilfe des Application Insights SDK können Sie Codezeilen in Ihre Anwendung einfügen, um die Benutzeraktivitäten zu überwachen, damit Sie Ihre Anwendung an deren Bedürfnisse anpassen können.

Wir sind dabei, dieses Feature auf das neue Application Insights-Portal in Microsoft Azure zu übertragen. (Sie können die [Funktionsweise im alten Portal](http://msdn.microsoft.com/library/dn481100.aspx) anzeigen.)

Bereits jetzt können Sie [Diagnose-Ereignisprotokolle, die mit der Trace-, NLog- oder Log4Net-Diagnose generiert wurden, erfassen und durchsuchen][].
