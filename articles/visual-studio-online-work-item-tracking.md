Verfolgung von Arbeitsaufgaben für Ihre Website
===============================================

In diesem Thema erfahren Sie mehr über die Verwendung der Tools zur Verfolgung von Arbeitsaufgaben, um die Arbeit für Ihre Website zu verwalten.

Gewusst wie: Hinzufügen von Arbeitsaufgaben zum Backlog
-------------------------------------------------------

1.  Navigieren Sie zum Teamprojekt-Blatt.

2.  Klicken Sie in der "Arbeitslinse" unten auf den Backlog-Bereich.

    Das Backlog-Fenster wird angezeigt. Der Backlog enthält eine Arbeitsliste nach Prioritäten für Ihre Website.

    ![Geöffneter Backlog](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-backlog.png)

3.  Fügen Sie ein Element hinzu, indem Sie den Titel in das Textfeld oben eingeben und die Eingabetaste drücken.

    Wenn Sie eine Scrum-Prozessvorlage verwenden, können Sie einen Bug oder ein Produkt-Backlogelement hinzufügen, indem Sie es in der Dropdown-Liste auf der rechten Seite auswählen.

4.  Klicken Sie auf die Arbeitsaufgabe, um sie zu öffnen und weitere Details hinzuzufügen.

    ![Arbeitsaufgabe öffnen](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-work-item.png)

5.  Im Arbeitsaufgabenformular können Sie den Titel bearbeiten, eine Beschreibung hinzufügen, die Arbeitsaufgabe einem Benutzer zuweisen und sie als abgeschlossen markieren.

6.  Unten im Arbeitsaufgabenformular können Sie einen Kommentar zur Arbeitsaufgabe hinzufügen und den Verlauf anzeigen.

    Um den Verlauf der Arbeitsaufgabe genauer zu betrachten, klicken Sie auf eine Änderung. Daraufhin wird ein Fenster mit weiteren Details geöffnet.

Gewusst wie: Verwalten Ihres Backlogs
-------------------------------------

Das Verwalten eines Backlogs mit den richtigen Prioritäten ist der Schlüssel zum erfolgreichen Verwalten Ihrer Website. Dies besteht aus zwei Teilen:

### Entfernen von Arbeiten nach dem Abschließen

1.  Öffnen Sie die Arbeitsaufgabe, die Sie als abgeschlossen markieren möchten.
2.  Markieren Sie das Kontrollkästchen mit der Bezeichnung "completed".
3.  Speichern Sie die Arbeitsaufgabe.
4.  Ein Stub für diese Aufgabe verbleibt im Backlog, bis Sie den Backlog aktualisieren oder ihn schließen und erneut öffnen.

![Abgeschlossene Arbeitsaufgabe](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-work-item.png)

### Reihenfolge der Prioritäten im Backlog

1.  Zeigen Sie auf die Arbeitsaufgabe, deren Priorität Sie ändern möchten.
2.  Klicken Sie auf die Ziehleiste links von der Arbeitsaufgabe.
3.  Ziehen Sie die Arbeitsaufgabe an die neue Position im Backlog, und legen Sie sie dort ab.

![Arbeitsaufgabe ziehen](./media/visual-studio-online-work-item-tracking/visual-studio-online-dragging-work-item.png)

Gewusst wie: Anzeigen kürzlich abgeschlossener Arbeiten
-------------------------------------------------------

1.  Klicken Sie auf dem Teamprojekt-Blatt auf den Bereich der abgeschlossenen Aufgaben.
2.  Zeigen Sie die Liste der kürzlich abgeschlossenen Arbeiten an.

![Arbeitsaufgabe ziehen](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-items.png)
