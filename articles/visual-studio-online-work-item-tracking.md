 # Verfolgung von Arbeitsaufgaben für Ihre WebsiteIn diesem Thema erfahren Sie mehr über die Verwendung der Tools zur Verfolgung von Arbeitsaufgaben, um die Arbeit für Ihre Website zu verwalten. ## Inhaltsverzeichnis * <a href="#how-to-add-work-items-to-your-backlog">Gewusst wie: Hinzufügen von Arbeitsaufgaben zum Backlog</a> * <a href="#how-to-manage-your-backlog">Gewusst wie: Verwalten Ihres Backlogs</a> * <a href="#how-to-view-recently-completed-work">Gewusst wie: Anzeigen kürzlich abgeschlossener Arbeiten</a> ## Gewusst wie: Hinzufügen von Arbeitsaufgaben zu Ihrem Backlog 1. Navigieren Sie zu Ihrem Teamprojekt-Fenster.2. Klicken Sie unten im Bereich "Arbeit" auf den Backlog-Bereich.  	Das Backlog-Fenster wird angezeigt. Der Backlog enthält eine Arbeitsliste nach Prioritäten für Ihre Website. 	![Open backlog](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-backlog.png) 3. Fügen Sie ein Element hinzu, indem Sie den Titel in das Textfeld oben eingeben und die Eingabetaste drücken.		Wenn Sie eine Scrum-Prozessvorlage verwenden, können Sie aus der Dropdown-Liste auf der rechten Seite einen Bug oder ein Product Backlog Item hinzufügen. 4. Klicken Sie auf die Arbeitsaufgabe, um sie zu öffnen und weitere Details hinzuzufügen.	![Open work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-open-work-item.png) 5. Im Arbeitsaufgabenformular können Sie den Titel bearbeiten, eine Beschreibung hinzufügen, die Arbeitsaufgabe einem Benutzer zuweisen und sie als abgeschlossen kennzeichnen. 6. Unten im Arbeitsaufgabenformular können Sie der Arbeitsaufgabe einen Kommentar hinzufügen und ihren Verlauf anzeigen.		Um den Verlauf der Arbeitsaufgabe genauer zu betrachten, klicken Sie auf eine Änderung. Daraufhin wird ein Fenster mit weiteren Details geöffnet. ## Gewusst wie: Verwalten Ihres BacklogsDas Verwalten eines Backlogs mit den richtigen Prioritäten ist der Schlüssel zum erfolgreichen Verwalten Ihrer Website. Dies umfasst zwei Teile: ###Entfernen von abgeschlossenen Arbeitsaufgaben 1. Öffnen Sie die Arbeitsaufgabe, die Sie als abgeschlossen kennzeichnen möchten. 2. Aktiiveren Sie das Kontrollkästchen "Abgeschlossen." 3. Speichern Sie die Arbeitsaufgabe. 4. Ein Stub für diese Aufgabe verbleibt im Backlog, bis Sie es aktualisieren oder es schließen und erneut öffnen.![Completed work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-work-item.png) ###Bewahren der Prioritätsreihenfolge Ihres Backlogs 1. Bewegen Sie den Mauszeiger über die Arbeitsaufgabe, deren Priorität Sie ändern möchten. 2. Klicken Sie auf das Ziehelement links von der Arbeitsaufgabe. 3. Ziehen Sie die Arbeitsaufgabe an die gewünschten Stelle im Backlog, und legen Sie sie dort ab.![Dragging work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-dragging-work-item.png) ## Gewusst wie: Anzeigen kürzlich abgeschlossener Arbeiten 1. Klicken Sie im Teamprojekt-Fenster auf das Fenster mit den abgeschlossenen Aufgaben. 2. Zeigen Sie die Liste kürzlich abgeschlossener Arbeiten an. ![Dragging work item](./media/visual-studio-online-work-item-tracking/visual-studio-online-completed-items.png)
