# Nachverfolgen von Arbeitsaufgaben für Ihre WebsiteIn diesem Thema erfahren Sie mehr über die Verwendung der Tools zur Nachverfolgung von Arbeitsaufgaben, um die Arbeit für Ihre Website zu verwalten.\#\# Inhaltsverzeichnis\* [Gewusst wie: Hinzufügen von Arbeitsaufgaben zum Backlog][Gewusst wie: Hinzufügen von Arbeitsaufgaben zum Backlog]\* [Gewusst wie: Verwalten Ihres Backlogs][Gewusst wie: Verwalten Ihres Backlogs]\* [Gewusst wie: Anzeigen kürzlich abgeschlossener Arbeiten][Gewusst wie: Anzeigen kürzlich abgeschlossener Arbeiten]\#\# Gewusst wie: Hinzufügen von Arbeitsaufgaben zum Backlog1. Navigieren Sie zum Teamprojektfenster.2. Klicken Sie in der "Arbeitslinse" unten auf den Backlog-Bereich. Das Backlog-Fenster wird angezeigt. Der Backlog enthält eine Arbeitsliste nach Prioritäten für Ihre Website. ![Backlog öffnen][Backlog öffnen]3. Fügen Sie eine Aufgabe hinzu, indem Sie den Titel in das Textfeld oben eingeben und die Eingabetaste drücken. Wenn Sie eine Scrum-Prozessvorlage verwenden, können Sie einen Bug oder eine Produkt-Backlogaufgabe hinzufügen, indem Sie sie/ihn in der Dropdownliste auf der rechten Seite auswählen.4. Klicken Sie auf die Arbeitsaufgabe, um sie zu öffnen und weitere Details hinzuzufügen. ![Arbeitsaufgabe öffnen][Arbeitsaufgabe öffnen]5. Im Arbeitsaufgabenformular können Sie den Titel bearbeiten, eine Beschreibung hinzufügen, die Arbeitsaufgabe einem Benutzer zuweisen und sie als abgeschlossen markieren.6. Unten im Arbeitsaufgabenformular können Sie einen Kommentar zur Arbeitsaufgabe hinzufügen und den Verlauf anzeigen. Um den Verlauf der Arbeitsaufgabe genauer zu betrachten, klicken Sie auf eine Änderung. Daraufhin wird ein Fenster mit weiteren Details geöffnet.\#\# Gewusst wie: Verwalten Ihres BacklogsDas Verwalten eines Backlogs mit den richtigen Prioritäten ist der Schlüssel zum erfolgreichen Verwalten Ihrer Website. Dies besteht aus zwei Teilen:\#\#\#Entfernen von Arbeiten nach dem Abschließen1. Öffnen Sie die Arbeitsaufgabe, die Sie als abgeschlossen markieren möchten.2. Aktivieren Sie das Kontrollkästchen mit der Bezeichnung "Abgeschlossen".3. Speichern Sie die Arbeitsaufgabe.4. Ein Stub für diese Aufgabe verbleibt im Backlog, bis Sie den Backlog aktualisieren oder ihn schließen und erneut öffnen.![Abgeschlossene Arbeitsaufgabe][Abgeschlossene Arbeitsaufgabe]\#\#\#Verwalten der Prioritäten im Backlog1. Zeigen Sie auf die Arbeitsaufgabe, deren Priorität Sie ändern möchten.2. Klicken Sie auf das Ziehelement links von der Arbeitsaufgabe.3. Ziehen Sie die Arbeitsaufgabe an die neue Position im Backlog, und legen Sie sie dort ab.![Arbeitsaufgabe ziehen][Arbeitsaufgabe ziehen]\#\# Gewusst wie: Anzeigen kürzlich abgeschlossener Arbeiten1. Klicken Sie im Teamprojektfenster auf das Fenster der abgeschlossenen Aufgaben.2. Zeigen Sie die Liste der kürzlich abgeschlossenen Arbeiten an. ![Arbeitsaufgabe ziehen][1]
