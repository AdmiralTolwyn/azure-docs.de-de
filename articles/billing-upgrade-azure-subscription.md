# <a name="upgrade-your-free-azure-subscription-to-pay-as-you-go"></a>Aktualisieren Sie Ihr kostenlose Azure-Abonnement auf nutzungsbasierte Bezahlung
Sie können Ihre [kostenlose Testversion](https://azure.microsoft.com/free/) oder Ihr [Microsoft Imagine](https://azure.microsoft.com/offers/ms-azr-0144p/) Abonnement im Azure-Kontoportal auf [nutzungsbasierte Bezahlung](https://azure.microsoft.com/offers/ms-azr-0003p/) aktualisieren. 

1. Melden Sie sich beim [Azure-Kontoportal](https://account.windowsazure.com/subscriptions) an, und navigieren Sie zur Registerkarte **Abonnements**.
2. Wählen Sie im Bereich „Abonnementstatus“ das Banner **Jetzt aktualisieren**.
   
    ![Hinweis zu Upgrade von einer kostenlosen Testversion auf nutzungsbasierte Bezahlung](./media/billing-upgrade-azure-subscription/billpage.png)
3. Bestätigen Sie das Upgrade.
   
    ![Bestätigung Ihres Abonnement-Upgrades](./media/billing-upgrade-azure-subscription/Upgrade.png)

Wenn Sie von der kostenlosen Testversion auf nutzungsbasierte Bezahlung aktualisieren, bleiben Vorteile Ihrer kostenlosen Testversion wie die 30-Tage-Grenze oder die Gutschrift in Höhe von 200 USD erhalten, bis einer der Vorteile aufgebraucht ist. Die Aktualisierung des Abonnements erfolgt danach.
