# Azure-Blog zu Sicherheit und Compliance

Im Azure-Teamblog zu Sicherheit und Compliance ([Azure Security and Compliance Team Blog](https://blogs.msdn.microsoft.com/azuresecurity/)) finden Sie aktuelle technische Informationen zu Sicherheit und Compliance in Azure.

Darüber hinaus finden Sie in diesem Blog wichtige Aktualisierungen der technischen Themen zu [Azure Security Center](https://azure.microsoft.com/documentation/services/security-center/).

Neben technischen Informationen finden Sie im Azure-Teamblog zu Sicherheit und Compliance Verweise auf aktuelle Ankündigungen zu Azure-Sicherheit und -Compliance, wenn sie im Azure-Blog veröffentlicht werden.

Im Azure-Teamblog zu Sicherheit und Compliance finden Sie Neuigkeiten rund um das Thema Azure-Sicherheit.
