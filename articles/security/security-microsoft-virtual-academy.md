# Azure-Sicherheit in der Microsoft Virtual Academy

In der Microsoft Virtual Academy werden kostenlose Onlineschulungen angeboten, damit Entwickler, IT- und Datenspezialisten und Studenten immer auf dem neuesten Stand der Technologie bleiben, ihre Kompetenzen und Kenntnisse vertiefen und sich weiterentwickeln können.

Auf dieser Seite finden Sie eine Aufstellung ausgewählter Kurse zur Azure-Sicherheit. Besuchen Sie die [Microsoft Virtual Academy](https://mva.microsoft.com/), um sich einen Überblick über alle verfügbaren Kurse zu verschaffen.

[Dev/Test in the Cloud](https://mva.microsoft.com/training-courses/devtest-in-the-cloud-16274?l=9aAijd7LC_2005190311) (Entwicklung und Testen in der Cloud)

Möchten Sie als Entwickler schneller bessere Anwendungen bereitstellen? Dies können Sie durch Verschieben Ihrer Entwicklungs- und Testumgebungen in die Cloud erreichen. Erfahren Sie, wie Sie dabei vorgehen müssen, und lernen die Vorteile dieses Wechsels kennen. In Demos und Präsentationen wird außerdem gezeigt, wie Ihre Anforderungen an die Entwicklungs- und Testumgebungen mit Microsoft Azure unterstützt werden können. Die Schulung umfasst Lektionen zu Methoden der Sicherheitsentwicklung und Bereitstellung.

[Common Tasks for Linux on Azure](https://mva.microsoft.com/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937) (Häufige Aufgaben für Linux bei Azure)

In diesem ausführlichen Kurs erhalten Sie Antworten auf Ihre Fragen zur Verwendung von Linux auf der Microsoft Azure Platform. Lernen Sie mit den Experten der [Linux Academy](https://linuxacademy.com/) einige allgemeine Aufgaben kennen. Sie erhalten Informationen zum Erstellen eines virtuellen Linux-Computers in Azure, zum Zugreifen auf virtuelle Linux-Computer über Remotedesktop-Software und zum Ausführen von virtuellen Hosts. In diesem Kurs werden viele Sicherheitstechnologien und -konfigurationen behandelt.

[Secure the Cloud](https://mva.microsoft.com/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881) (Sichern der Cloud)

In dieser Sitzung erfahren Sie, wie Sie mit Microsoft globale Compliancevorgaben (z.B. ISO 27001 oder ISO 27018, FedRAMP, PCI oder HIPAA) einhalten können, mit neuen Sicherheitskontrollen, angefangen bei der Verschlüsselung für ruhende Daten, über die Schlüsselverwaltung, den Schutz für virtuelle Computer, die Protokollierung und Überwachung bis hin zu Antischadsoftwarediensten, zur Identitätsverwaltung, Access Control und vielem mehr.

[Design and Implement Cloud Data Platform Solutions](https://mva.microsoft.com/training-courses/design-and-implement-cloud-data-platform-solutions-15711?l=jbCdW0j1B_3005244527) (Entwickeln und Implementieren von Lösungen für Datenplattformen in der Cloud)

Lernen Sie die Features und Funktionen der Lösungen für Datenplattformen in der Microsoft-Cloud kennen. Verschaffen Sie sich einen allgemeinen Überblick über Plattformen, und informieren Sie sich über Sicherheitsfeatures, Optionen für hohe Verfügbarkeit, Verfahren für die Überwachung und Verwaltung von Clouddaten und mehr. Darüber hinaus erhalten Sie Anleitungen zum Identifizieren von Kompromissen und zur Entscheidungsfindung beim Entwerfen von öffentlichen und Hybrid Cloud-Lösungen mithilfe der Features für Datenplattformen in der Microsoft-Cloud.

[Manage and Secure Identities in a Cloud and Mobile World](https://mva.microsoft.com/training-courses/manage-and-secure-identities-in-a-cloud-and-mobile-world-14013?l=GIJ2GcvrB_405192797) (Verwalten und Sichern von Identitäten in der Cloud und der mobilen Welt)

In dieser Sitzung erfahren Sie, wie Sie mithilfe von Azure Active Directory und Microsoft Advanced Threat Analytics die Benutzeridentität in Ihrem Unternehmen sichern und verwalten, wie Sie Sicherheitsverletzungen erkennen können, bevor sie Schaden anrichten, und wie Sie für Ihre Benutzer eine einzelne Identität für den Zugriff auf alle Unternehmensressourcen bereitstellen. Lernen Sie die Technologien kennen, mit denen Sie Schatten-IT ermitteln, den Anwendungszugriff verwalten und verdächtige Aktivitäten über erweiterte Sicherheitsberichte und Analysen, Überwachung und Warnungen hinsichtlich des Benutzerverhaltens überwachen können.

[Security in a Cloud-Enabled World](https://mva.microsoft.com/training-courses/security-in-a-cloudenabled-world-12725?l=CfLHobAcB_3904300474) (Sicherheit in einer Cloud-fähigen Welt)

Experten erläutern Ihnen anhand des Posters [Microsoft Cloud Security for Enterprise Architects](http://www.microsoft.com/download/48121) (Microsoft-Cloudsicherheit für Enterprise-Architekten) die Verantwortlichkeiten des Kunden und geben Tipps zur Erneuerung aller Bereiche Ihres Sicherheitsstatus, einschließlich Governance, Einschlussstrategien, Sicherheitsvorgängen, höchstem Assetschutz, Informationssicherheit sowie Benutzer- und Gerätesicherheit mit Schwerpunkt auf dem Schutz von Administratorberechtigungen. Sie machen sich mit dem Framework vertraut, das auch von den Microsoft-Mitarbeitern für die Internetsicherheit verwendet wird, um die Cloudsicherheit von Kunden zu bewerten und für sie eine Sicherheitsroadmap zu erstellen.

[Microsoft Azure IaaS Deep Dive](https://mva.microsoft.com/training-courses/microsoft-azure-iaas-deep-dive-14339?l=PtppYVQgB_8300115888) (Tiefgehende Informationen zu Microsoft Azure IaaS)

Erfahren Sie, wie Sie die Infrastrukturfunktionen von Microsoft Azure nutzen können. Als IT-Experte müssen Sie über keine Vorkenntnisse in Azure verfügen. In diesem Kurs wird schrittweise die Erstellung und Konfiguration von virtuellen Azure-Computern, virtuellen Azure-Netzwerken und der standortübergreifenden Konnektivität zum Aufnehmen des Betriebs in der Cloud erläutert. Auch auf Sicherheitsfeatures und-überlegungen wird während des gesamten Kurses eingegangen.

[Getting Started with Azure Security for the IT Professional](https://mva.microsoft.com/training-courses/getting-started-with-azure-security-for-the-it-professional-11165?l=HfHzCXSAB_7404300474) (Erste Schritte mit Azure Security für IT-Experten)

In dieser Schulung werden diese und weitere wichtige Themen anhand zahlreicher Demos erörtert. Ein Team aus Sicherheitsexperten und Azure-Technikern erläutert, welche Möglichkeiten es in Azure gibt, die über die grundlegenden Zertifizierungen hinausgehen. Hier erhalten Sie Informationen zum Entwurf und zur Verwendung verschiedener Technologien, um sicherzustellen, dass Sie über die Sicherheit und Architektur verfügen, die für den erfolgreichen Start Ihrer Projekte in der Cloud erforderlich sind. Verschaffen Sie sich einen gründlichen Einblick in die Abläufe im Rechenzentrum, die Konfiguration von virtuellen Computern, die Netzwerkarchitektur und die Speicherinfrastruktur.

[Deep Dive into Azure Resource Manager Scenarios and Patterns](https://mva.microsoft.com/training-courses/deep-dive-into-azure-resource-manager-scenarios-and-patterns-13793?l=i1m06ZJYB_7001937557) (Tiefgehende Informationen zu Azure Resource Manager-Szenarios und -Mustern)

Erfahren Sie, wie Sie Azure Resource Manager (ARM) als Tool zum Verwalten von Ressourcengruppen verwenden können. Lernen Sie ARM mit einem Team aus internationalen Experten kennen, die Ihnen Skripts und Tools vorstellen, mit denen Sie mühelos Elemente Ihrer Anwendungsinfrastruktur zwischen der Entwicklungs-, Test- oder Produktionsumgebung verschieben können. Machen Sie sich mit der Verwendung der rollenbasierten Access Control (RBAC) zum Implementieren der Sicherheit mit ARM vertraut.

[Azure Rights Management Services Core Skills](https://mva.microsoft.com/training-courses/azure-rights-management-services-core-skills-10500?l=QLoxMwuCB_1805094681) (Wichtige Fähigkeiten für Azure Rights Management-Dienste)

In diesem Kurs wird erläutert, weshalb die Informationssicherheit ein 100-prozentiges Muss in Ihrer Organisation ist und wie das geistige Eigentum Ihrer Organisation auf allen Geräten und in der Cloud durch die Rechteverwaltung geschützt wird. Microsoft-Experten vermitteln Ihnen entsprechende praktische Kenntnisse und technisches Know-how.
