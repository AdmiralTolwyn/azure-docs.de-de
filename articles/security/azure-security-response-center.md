# Microsoft Security Response Center

Das Microsoft Security Response Center (MSRC) wird von einigen der weltweit erfahrensten Sicherheitsexperten geleitet. Durch Überwachung rund um die Uhr an jedem Tag des Jahres identifizieren diese Experten Sicherheitsvorfälle und Sicherheitsrisiken lokal und in der Cloud, reagieren darauf und lösen sie.

Neben der fortlaufenden Arbeit von MSRC im Hintergrund stellt Ihnen das MSRC-Team eine Reihe von Ressourcen zur Verfügung, damit Sie verstehen können, wie Sie Ihre Assets und Bereitstellungen in Azure effektiver schützen können.

## Der MSRC-Blog

Im [MSRC-Blog](https://blogs.technet.microsoft.com/msrc/) erfahren Sie die neuesten Nachrichten darüber, was das MSRC unternimmt, um Sie vor Bedrohungen in der Cloud zu schützen.
 
## Whitepapers

Das MSRC hat eine Anzahl von [Whitepapers](https://technet.microsoft.com/library/bb969102.aspx) veröffentlicht, die Ihnen helfen, seine Aktivitäten und die Art ihrer Ausführung zu verstehen. Einige bieten Einblicke in die Methoden, mit denen wir die Microsoft Cloud schützen, und enthalten nützliche Informationen dazu, wie Sie die gleichen Sicherheitskonfigurationen einsetzen können.
 
## Engagement als Sicherheitsexperte und Prämienprogramme

Das MSRC unterstützt die Zusammenarbeit und Beziehungen mit Sicherheitsexperten weltweit, um die Sicherheit von Microsoft-Produkten ständig zu verbessern.

Microsoft-Prämienprogramme honorieren Experten für neuartige Ausnutzungstechniken, Ideen zur Verteidigung, die neuartige Ausnutzungstechniken entschärfen, und Identifizierung kritischer Sicherheitslücken in lokaler und Cloudsoftware von Microsoft.
 
Mehr über diese Programme erfahren Sie auf der Seite [Microsoft Bounty Programs](https://technet.microsoft.com/security/dn425036) (Prämienprogramme von Microsoft) und im [MSRC-Blog](https://blogs.technet.microsoft.com/msrc/).

Weitere Informationen zum MSRC finden Sie auf der [MSRC-Homepage](https://technet.microsoft.com/library/dn440717.aspx).
