# <a name="microsoft-security-response-center"></a>Microsoft Security Response Center

Das Microsoft Security Response Center (MSRC) wird von einigen der weltweit erfahrensten Sicherheitsexperten geleitet. Durch Überwachung rund um die Uhr an jedem Tag des Jahres identifizieren diese Experten Sicherheitsincidents und Sicherheitsrisiken lokal und in der Cloud, reagieren darauf und lösen sie.

Neben der fortlaufenden Arbeit stellt Ihnen das MSRC-Team eine Reihe von Ressourcen zur Verfügung, die Ihnen helfen zu verstehen, wie Sie Ihre Assets und Bereitstellungen in Azure effektiver schützen können.

## <a name="the-msrc-blog"></a>Der MSRC-Blog

Im [MSRC-Blog](https://blogs.technet.microsoft.com/msrc/) erfahren Sie die neuesten Nachrichten darüber, was das MSRC unternimmt, um Sie vor Bedrohungen in der Cloud zu schützen.

## <a name="white-papers"></a>Whitepapers

Das MSRC hat eine Anzahl von [Whitepapers](https://technet.microsoft.com/library/bb969102.aspx) veröffentlicht, die Sie über die Aktivitäten und die Art und Weise ihrer Ausführung informieren. Einige bieten Einblicke in die Methoden, mit denen wir die Microsoft Cloud schützen, und enthalten nützliche Informationen dazu, wie Sie die gleichen Sicherheitskonfigurationen einsetzen können.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Engagement als Sicherheitsexperte und Prämienprogramme

Das MSRC unterstützt die Zusammenarbeit und Beziehungen mit Sicherheitsexperten weltweit, um die Sicherheit von Microsoft-Produkten ständig zu verbessern.

Microsoft-Prämienprogramme entlohnen Experten für Folgendes:

- Aufdecken neuartiger Exploittechniken
- Vorschläge zum Minimieren neuartiger Exploits
- Ermitteln kritischer Sicherheitslücken in lokaler und cloudbasierter Microsoft-Software.

Weitere Informationen zum MSRC finden Sie auf der [MSRC-Homepage](https://technet.microsoft.com/library/dn440717.aspx).
