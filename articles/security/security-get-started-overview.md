# Azure-Sicherheitsübersicht

Auf der Microsoft-Website für Informationen zur Azure-Sicherheit auf Azure.com finden Sie technische Antworten auf Ihre Fragen zur Azure-Sicherheit. Informationen zur Compliance von Azure und zur eigenständigen Sicherung der Azure-Infrastruktur finden Sie im [Microsoft Trust Center](https://www.microsoft.com/TrustCenter/default.aspx).

Wir wissen, dass Sicherheit in der Cloud an erster Stelle steht und wie wichtig es ist, dass Sie präzise und zeitnahe Informationen zur Azure-Sicherheit finden. Einer der besten Gründe für die Verwendung von Azure für Ihre Anwendungen und Dienste liegt darin, dass Sie die Vielzahl an Sicherheitstools und -funktionen nutzen können, die wir Ihnen zur Verfügung stellen, damit Sie sichere Lösungen auf der sicheren Azure-Plattform erstellen können.

Diese Seite und diese Website werden sich fortwährend weiterentwickeln – ebenso wie neue Azure-Dienste konstant online geschaltet und vorhandene Dienste rasch aktualisiert werden, werden wir auch hier die Informationen beständig anpassen. Unsere Sicherheitsangebote verbessern sich kontinuierlich, und wir werden Ihnen hier die technischen Details dieser Verbesserungen mitteilen. Es empfiehlt sich also, diese Website regelmäßig zu besuchen und sich über die Neuigkeiten zu informieren.

## Verwenden der Website für Informationen zur Azure-Sicherheit
Im linken Bereich der Seite sehen Sie mehrere Kategorien von Sicherheitsthemen. Klicken oder tippen Sie auf ein Thema, um die vorhandenen Artikel der entsprechenden Kategorie anzuzeigen.

Jede Kategorie beinhaltet einen Artikel mit einer allgemeinen Übersicht. Diese allgemeine Übersicht soll Ihnen eine Zusammenfassung des Angebots von Azure hinsichtlich der Sicherheit für die jeweilige Kategorie geben. In den Übersichtsartikeln erhalten Sie einen allgemeinen Einblick in die in Azure verfügbaren Produkte, Dienste und Features, mit denen Sie die Sicherheit Ihrer Bereitstellungen optimieren können. Die Übersichtsartikel enthalten eine Fülle von Links zu technischen Inhalten, sodass Sie sich eingehend über die Themen und deren Umsetzung informieren können.

Die Website für Informationen zur Azure-Sicherheit umfasst derzeit die folgenden Kategorien:

- Ressourcen
- Netzwerksicherheit
- Speichersicherheit
- Sicherheit virtueller Computer
- Endpoint Protection
- Sicherheitsverwaltung und -überwachung
- Identitätsverwaltung
- PaaS-Dienste
- Branchenspezifisches
- Architektur

> [AZURE.NOTE] Alle auf der Website für Informationen zur Azure-Sicherheit vorhandenen Artikel werden im Navigationsbereich links auf der Seite angezeigt. Wenn Sie alle Artikel anzeigen möchten, klicken oder tippen Sie einfach auf **Mehr Informationen** unten in der Liste der Kategorien.

Dazu können Sie den Bereich „Kommentare“ unten verwenden oder uns unter [azsecinfosite@microsoft.com](mailto:azsecinfosite@microsoft.com) eine E-Mail schreiben. Wir werden Ihre Vorschläge dann in unsere Planung aufnehmen.
