# <a name="azure-operational-security-checklist"></a>Azure operational Sicherheitscheckliste
Bereitstellen einer Anwendung in Azure ist schnell, einfach und kostengünstig. Vor der Bereitstellung von Cloud-Anwendung in der Produktion sinnvoll, eine Prüfliste zu verwenden, bei der Bewertung Ihrer Anwendung mit einer Liste von wichtige und empfohlene operational Sicherheitsaktionen Sie berücksichtigen müssen.

## <a name="introduction"></a>Einführung

Azure bietet eine Suite von Infrastrukturdienste, die Sie zum Bereitstellen Ihrer Anwendungen verwenden können. Azure Operational Security bezieht sich auf die Dienste, Steuerelemente und Features, die für Benutzer zum Schützen ihrer Daten, Anwendungen und anderen Ressourcen in Microsoft Azure zur Verfügung stehen.

-   Um den maximalen Vorteil aus der Cloudplattform heraus zu erhalten, wird empfohlen, dass Sie Azure-Dienste nutzen, und führen Sie die Checkliste.
-   Organisationen, die investieren Sie Zeit und Ressourcen, die die Einsatzbereitschaft ihrer Anwendungen vor dem Start der Bewertung haben einen viel höheren Anteil an Zufriedenheit als denjenigen, die nicht an. Beim Ausführen dieser Arbeit kann Prüflisten ein unschätzbarem Wert Mechanismus, um sicherzustellen, dass Anwendungen konsistent und einheitlich ausgewertet werden.
-   Die Ebene der operativen Analyse wird variiert, je nach Cloud-Entwicklungsstands des Unternehmens und der Anwendungsverzeichnis Entwicklungsphase verfügbarkeitsanforderungen und Empfindlichkeit datenanforderungen.
