# Operational Insights-Dashboards

[AZURE.INCLUDE [operational-insights-note-moms](../includes/operational-insights-note-moms.md)]

Diese Anleitung vermittelt, wie Operational Insights-Dashboards zum Visualisieren aller Ihrer gespeicherten Suchen dienen und Ihnen einen zentralen Überblick über Ihre Umgebung bieten.

![Beispieldashboard](./media/operational-insights-use-dashboards/example-dash.png)

## Wie erstelle ich mein Dashboard?

Wechseln Sie zunächst zur Azure Operational Insights-Übersicht, indem Sie im linken Navigationsbereich auf die Schaltfläche "Übersicht" klicken. Auf der linken Seite sehen Sie die Kachel "Mein Dashboard". Klicken Sie darauf, um das Dashboard detailliert zu durchsuchen.

![Übersicht](./media/operational-insights-use-dashboards/overview.png)



## Hinzufügen einer Kachel

In Dashboards werden Kacheln von Ihren gespeicherten Suchen bestimmt. Operative Insights bietet viele vordefinierte gespeicherte Suchen, damit Sie sofort loslegen können. Die folgenden Abbildungen veranschaulichen, wie Sie beginnen können.

![Abbildungen](./media/operational-insights-use-dashboards/pictorial.png)

Klicken Sie in der Ansicht "Mein Dashboard" unten auf der Seite auf das Zahnrad  'customize', um den Anpassungsmodus zu öffnen. Im rechts auf der Seite geöffneten Bereich werden alle für den Arbeitsbereich gespeicherten Suchen gezeigt.

![Kacheln hinzufügen 1](./media/operational-insights-use-dashboards/add-tile1.png)

Um eine gespeicherte Suche als Kachel zu visualisieren, ziehen Sie sie auf den Leerraum auf der linken Seite. Beim Ziehen wird die Suche in eine Kachel umgewandelt.

![Kacheln hinzufügen 2](./media/operational-insights-use-dashboards/add-tile2.png)

![Kacheln hinzufügen 3](./media/operational-insights-use-dashboards/add-tile3.png)


## Bearbeiten einer Kachel

Klicken Sie in der Ansicht "Mein Dashboard" unten auf der Seite auf das Zahnrad  'customize', um den Anpassungsmodus zu öffnen. Klicken Sie auf die Kachel, die Sie bearbeiten möchten. Der rechte Bereich wird zum Bearbeiten geöffnet und bietet eine Auswahl von Optionen:
![Kachel bearbeiten](./media/operational-insights-use-dashboards/edit-tile.png)

### Kachelvisualisierungen
Es stehen zwei Arten von Kachelvisualisierungen zur Auswahl:

**Balkendiagramm**
<p>
![Balkendiagramm](./media/operational-insights-use-dashboards/bar-chart.png)

Dieses Diagramm zeigt eine Zeitachse mit den Ergebnissen Ihren gespeicherten Suche oder eine Liste von Ergebnissen gemäß einem Feld. Dies hängt davon ab, ob Ihre Suche Ergebnisse gemäß einem Feld aggregiert oder nicht.

**Metrik**
<p>
![Metrik](./media/operational-insights-use-dashboards/metric.png)

Dieses Diagramm zeigt Ihre gesamten Suchergebnistreffer als Zahl in einer Kachel an. Metrikkacheln ermöglichen Ihnen das Festlegen eines Schwellenwerts, der auf der Kachel hervorgehoben wird, sobald der Schwellenwert erreicht wird.

### Schwellenwert
Mithilfe der Metrikvisualisierung können Sie einen Schwellenwert auf einer Kachel erstellen. Klicken Sie darauf, um einen Schwellenwert auf der Kachel zu erstellen. Wählen Sie, ob die Kachel hervorgehoben werden soll, wenn der Wert über oder unter dem gewählten Schwellenwert liegt, und legen Sie anschließend den Schwellenwert darunter fest.

## Organisieren des Dashboards
Um das Dashboard zu organisieren, navigieren Sie zur Ansicht "Mein Dashboard" und klicken unten auf der Seite auf das Zahnrad  'customize', um den Anpassungsmodus zu öffnen. Klicken Sie auf die Kachel, die Sie verschieben möchten, und ziehen Sie sie an die gewünschte Position.

![Organisieren des Dashboards](./media/operational-insights-use-dashboards/organize.png)

## Entfernen einer Kachel
Um eine Kachel zu entfernen, navigieren Sie zur Ansicht "Mein Dashboard" und klicken unten auf der Seite auf das Zahnrad **Anpassen**, um den Anpassungsmodus zu öffnen. Wählen Sie die Kachel aus, die Sie entfernen möchten, und klicken Sie dann im rechten Bereich auf **Kachel entfernen**.
![Entfernen einer Kachel](./media/operational-insights-use-dashboards/remove-tile.png)
