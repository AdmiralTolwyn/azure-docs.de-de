# <a name="azure-government-marketplace"></a>Azure Government Marketplace
Partner, die ihre Angebote auf dem Azure Marketplace für Government veröffentlichen möchten, finden die Informationen im Folgenden.

## <a name="publishing"></a>Veröffentlichung
> [!NOTE]
> Wenn Sie noch kein für Azure Marketplace zertifizierter Partner sind, führen Sie zuerst die [hier](../marketplace-publishing/marketplace-publishing-getting-started.md) beschriebenen Schritte aus, bevor Sie fortfahren.
> 
> 

### <a name="step-1"></a>Schritt 1
Melden Sie sich auf [https://publish.windowsazure.com](https://publish.windowsazure.com) an.

### <a name="step-2"></a>Schritt 2
Klicken Sie auf das Angebot, das Sie veröffentlichen möchten.

### <a name="step-3"></a>Schritt 3
Klicken Sie auf **SKUS** und anschließend auf das Feld „Azure Government-Cloud“.

> [!NOTE]
> Es werden nur „Bring your own license (BYOL)“-SKUs unterstützt.  Diese Option ist nicht für SKUs mit nutzungsbasierter Bezahlung (PayG) verfügbar.
> 
> 

![alt text](./media/government-manage-marketplace-partner-1.png)

### <a name="step-4"></a>Schritt 4
Klicken Sie auf den Link „+ Add Certification (+ Zertifizierung hinzufügen)“, um Zertifizierungen zu Ihrem Angebot hinzuzufügen.

![alt text](./media/government-manage-marketplace-partner-2.png)

### <a name="step-5"></a>Schritt 5
Fordern Sie ein Testkonto in der Microsoft Azure Government-Cloud an, damit Sie Ihr Image im Veröffentlichungsportal testen können: [https://azuregov.microsoft.com/trial/azuregovtrial](https://azuregov.microsoft.com/trial/azuregovtrial)

Ihre Berechtigung als Partner, der Instanzen auf Bundes-, Staats-, Orts- oder Stammesebene der USA dient, wird überprüft. Sie erhalten die Bestätigung per E-Mail.  Ihr Testkonto steht Ihnen innerhalb von 3-5 Werktagen zur Verfügung.

### <a name="step-6"></a>Schritt 6
Klicken Sie auf „Veröffentlichen“ und dann auf „Für Stagingumgebung freigeben“. 

![alt text](./media/government-manage-marketplace-partner-3.png)

Sie werden aufgefordert, ein Abonnement in der Zulassungsliste einzugeben, das Zugriff auf das bereitgestellte Angebot hat. Geben Sie die Abonnement-ID Ihres neuen Testkontos für Azure Government ein.

![alt text](./media/government-manage-marketplace-partner-4.png)

### <a name="step-7"></a>Schritt 7
Sobald das Angebot erfolgreich bereitgestellt wurde, können Sie Ihr Image testen, indem Sie sich mit Ihrem Testkonto für Azure Government bei [https://portal.azure.us](https://portal.azure.us) anmelden.

### <a name="step-8"></a>Schritt 8
Sobald Sie Ihr Image mithilfe des Test-Abonnements überprüft haben, können Sie das Angebot online verfügbar machen, indem Sie auf „Veröffentlichen“ klicken, und die Genehmigung anfordern, in den Produktivbetrieb zu gehen. 

![alt text](./media/government-manage-marketplace-partner-5.png)
