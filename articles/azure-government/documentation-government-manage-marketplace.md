# <a name="azure-marketplace-for-government"></a>Azure Marketplace für Government
Der Azure Marketplace ist für Azure Government verfügbar, und zwar mit einer aktualisierten Liste von Images von unseren Marketplace-Herausgebern. 

## <a name="variations"></a>Variationen
Hier finden Sie einige Überlegungen zur Verwendung des Azure Marketplace für Government:

* Es sind nur „Bring your own license (BYOL)“-Images verfügbar. Sie können nicht auf Images zugreifen, die Kauftransaktionen in Azure Marketplace voraussetzen.
* Im Vergleich zum öffentlichen Marketplace ist derzeit nur eine Teilmenge der Images verfügbar. Sie finden die Liste der verfügbaren Images [hier](../azure-government-image-gallery.md) 
* Bevor ein Image bereitgestellt werden kann, muss Ihr Unternehmensadministrator Einkäufe im Marketplace für Ihr Azure-Abonnement zulassen.
  * Melden Sie sich beim Portal als Unternehmensadministrator an.
  * Navigieren Sie zu *Verwalten*.
  * Klicken Sie unter *Registrierungsdetails* auf das Stiftsymbol neben der Zeile *Azure Marketplace*.
  * Wechseln Sie nach Bedarf zwischen*Aktiviert/Deaktiviert*.
  * Klicken Sie unten auf der Seite auf *Speichern*

> [!NOTE]
> Wenn Sie Ihre Images in Azure Government verfügbar machen wollen, finden Sie weitere Informationen unter [partner onboarding guidelines (Richtlinien zum Partner-Onboarding)](documentation-government-manage-marketplace-partners.md).
> 
> 

### <a name="step-1"></a>Schritt 1
Starten Sie den Marketplace.

![alt text](./media/government-manage-marketplace-launch.png)  

### <a name="step-2"></a>Schritt 2
Durchsuchen Sie verschiedene Produkte, um das passende zu finden.

Der Marketplace-Herausgeber stellt als Teil der Produktbeschreibung eine Liste von Zertifizierungen bereit, um Ihnen die richtige Entscheidung zu erleichtern. 

![alt text](./media/government-manage-marketplace-service.png)

### <a name="step-3"></a>Schritt 3
Wählen Sie ein Produkt oder Image aus.

![alt text](./media/government-manage-marketplace-image.png)

### <a name="step-4"></a>Schritt 4
Starten Sie den Erstellen-Flow und geben Sie die erforderlichen Berechtigungen für die Bereitstellung an.

![alt text](./media/government-manage-marketplace-deployment.png)

> [!NOTE]
> In der Dropdownliste „Speicherort“ sind nur Azure Government-Speicherorte verfügbar.
> 
> 

### <a name="step-5"></a>Schritt 5
Beachten Sie die Preise.

![alt text](./media/government-manage-marketplace-pricing.png)

### <a name="step-6"></a>Schritt 6
Führen Sie alle Schritte aus, und klicken Sie anschließend auf „Ok“, um den Bereitstellungsprozess zu starten.
