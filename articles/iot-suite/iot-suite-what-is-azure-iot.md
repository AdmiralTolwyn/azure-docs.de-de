## Azure IoT Suite
Azure IoT Suite ist eine Lösung für Unternehmen, mit der Sie schnell mithilfe eines Satzes von erweiterbaren, vorkonfigurierte Lösungen loslegen können, die für gängige IoT-Szenarien ausgelegt sind, z. B. *Remoteüberwachung* und *verbundene Geräte*.

Die vorkonfigurierten Lösungen sind:

- vollständige, funktionsfähige End-to-End-Lösungen, die für die ersten Schritte simulierte Geräte enthalten, z. B. vorkonfigurierte Azure-Dienste wie Azure IoT Hub, Azure Event Hubs, Azure Stream Analytics, Azure Machine Learning, Azure-Speicher und lösungsspezifische Verwaltungskonsolen.
- enthalten bewährten, einsatzbereiten Code, den Sie anpassen und erweitern können, um Ihre eigenen spezifischen IoT-Szenarios zu implementieren.

Möglicherweise ist auch der Azure IoT Hub-Dienst für Sie interessant, der von vielen der vorkonfigurierten Lösungen verwendet wird. Der Azure IoT Hub-Dienst bietet viele der Funktionen im Bereich "Datenverarbeitung und Analysen", die in der beispielhaften Lösungsarchitektur definiert sind.
