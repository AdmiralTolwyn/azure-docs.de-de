# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Anzeigen von Analysedaten für Metriken über alle Azure-Web-App-Ressourcen

![Symbol für Web-Apps](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Die Azure Web-App-Analyselösung wurde in den Communitysupport verlagert.  
>- Die Lösung ist nicht mehr im Azure Marketplace verfügbar, kann aber über [Azure-Schnellstartvorlagen](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) installiert werden und wird dort von der Community unterstützt.
>- Kunden, bei denen die Lösung bereits installiert ist, können diese mit einigen Änderungen weiter verwenden.  
>- Zur Überwachung Ihrer Webanwendung empfiehlt Microsoft [Application Insights](../../azure-monitor/app/app-insights-overview.md).
