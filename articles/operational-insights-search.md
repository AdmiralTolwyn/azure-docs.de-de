# Operational Insights Search
Im Kern von Microsoft Azure Operational Insights befindet sich die Suchfunktion Search, die Ihnen das Kombinieren und Korrelieren beliebiger Computerdaten aus mehreren Quellen in Ihrer Umgebung ermöglicht. Intelligence Packs werden auch von Search beim Bereitstellen von Metriken unterstützt, die eine Pivotierung in Bezug auf einen bestimmten Problembereich bieten.


## Suchen nach Daten
Mithilfe der Funktion **Search** in Azure Operational Insights finden Sie die Daten, die Sie suchen 

![Abbildung der Search-Kachel](./media/operational-insights-search/overview-search.png)

Auf der Seite **Search** können Sie eine Abfrage erstellen und die Ergebnisse dann bei der Suche anhand von Facetsteuerelementen filtern. Sie können auch erweiterte Abfragen zum Transformieren, Filtern und Berichten für Ihre Ergebnisse erstellen.

![Abbildung der Search-Seite](./media/operational-insights-search/search-01.png)

Allgemeine Suchabfragen werden auf den meisten Search-Seiten angezeigt. In der gesamten Operational Insights-Konsole können Sie anhand von **Search** auf Kacheln klicken oder einen Drilldown in andere Elemente ausführen, um Details über das Element anzuzeigen.
Sie können Facets verwenden und Steuerelemente filtern, um die Ergebnisse einzugrenzen. Die Eingabehilfe mit der AutoVervollständigen-Funktion hilft Ihnen beim Erlernen der Syntax und führt zu schnellen Ergebnissen.
