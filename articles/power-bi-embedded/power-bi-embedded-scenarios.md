# <a name="common-power-bi-embedded-scenarios"></a>Häufige Power BI Embedded-Szenarios
**Microsoft Power BI Embedded** zielt auf das Verfügbarmachen der meisten vorhandenen Power BI-API-Funktionen als Teil eines Azure-Diensts ab, mit dem Sie Anwendungen entwickeln können.  Darüber hinaus können Sie die erforderlichen Ressourcen und den Power BI-Inhalt programmgesteuert bereitstellen, entwickeln und bereitstellen. Weitere Informationen finden Sie unter [Erste Schritte mit Microsoft Power BI Embedded](power-bi-embedded-get-started.md).

Nachstehend finden Sie einige Szenarien für die Verwendung von **Power BI Embedded** in Ihren Anwendungen.

* Vollständig interaktive Berichte, die auf dem Power BI Desktop erstellt wurden, können mithilfe eines neuen Anwendungstoken-Authentifizierungsmodells in Ihre eigenen Anwendung eingebettet werden.
* Unabhängige Softwarehersteller und Kunden, die kundenorientierte Anwendungen erstellen, können den **Power BI Embedded** -Dienst und das Power BI-SDK verwenden, um interaktive Berichte einzubetten. Weitere Informationen finden Sie unter [Erste Schritte mit Microsoft Power BI Embedded](power-bi-embedded-get-started.md).
* Als Entwickler können Sie das Power BI-Visualisierungsframework zum Erstellen von benutzerdefinierten Visualisierungen verwenden, die in Ihrer eigenen App verwendet werden können. Weitere Informationen finden Sie unter [Erweitern Sie Power BI um Ihre eigenen Visualisierungen](https://powerbi.microsoft.com/custom-visuals/).
