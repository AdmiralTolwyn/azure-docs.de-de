# Power BI Embedded REST-API

Mit **Microsoft Power BI Embedded** können Sie REST-API verwenden, um Anwendungen, die **Power BI-Berichte** in Ihre Anwendungen integrieren, programmgesteuert zu entwickeln.

Informationen zu **Power BI Embedded**-REST-API finden Sie unter [Power BI-REST-Vorgänge](https://msdn.microsoft.com/library/mt711493).
