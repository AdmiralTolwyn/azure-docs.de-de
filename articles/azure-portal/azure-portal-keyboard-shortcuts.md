# <a name="azure-portal-keyboard-shortcuts"></a>Tastenkombinationen für das Azure-Portal
## <a name="open-hubs"></a>Hubs öffnen
| Tastenkombination | Aktion |
| --- | --- |
| N |Benachrichtigungen |
| / |Durchsuchen von Diensten |
| A |Aktive Sitzungen |
| B |Abrechnung |
| C |Erstellen |

## <a name="navigation"></a>Navigation
| Tastenkombination | Aktion |
| --- | --- |
| H |Azure-Startdashboard |
| F |Erstes geöffnetes Blatt |
| J |Vorheriges geöffnetes Blatt |
| K |Nächstes geöffnetes Blatt |
| L |Letztes geöffnetes Blatt |
| - |Blatt minimieren oder aus maximierter Ansicht wiederherstellen |
| + |Blatt maximieren oder aus minimierter Ansicht wiederherstellen |
| # |Aktuelles Blatt schließen |
| ¦ |Alle Blätter schließen |
| dann |Fokus zur Befehlszeile verschieben |
|  verfügbar. |Den Fokus auf die Navigationsleiste verschieben |
| ? |Tastenkombinationen anzeigen |
