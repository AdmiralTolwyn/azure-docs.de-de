# <a name="how-to-manage-your-azure-location-based-services-account-and-keys"></a>Verwalten Ihres Azure Location Based Services-Kontos und der dazugehörigen Schlüssel

Sie können Ihr Azure Location Based Services-Konto und die dazugehörigen Schlüssel über das Azure-Portal verwalten. Wenn Sie über ein Konto und einen Abonnementschlüssel verfügen, können Sie die APIs auf Ihrer Website oder in Ihrer mobilen Anwendung implementieren.

Wenn Sie kein Azure-Abonnement besitzen, können Sie ein [kostenloses Konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) erstellen, bevor Sie beginnen.

## <a name="log-in-to-azure"></a>Anmelden an Azure 

Melden Sie sich unter „http://portal.azure.com“ beim Azure-Portal an.

## <a name="create-a-new-account"></a>Erstellen eines neuen Kontos

1. Klicken Sie in der linken oberen Ecke des Azure-Portals auf die Schaltfläche **Neu**.

2. Suchen Sie nach **Location Based Services**, und wählen Sie die Option aus. Klicken Sie anschließend auf **Erstellen**.

3. Geben Sie die Informationen zum Location Based Services-Konto ein. 

![Eingeben von Kontoinformationen im Portal](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="manage-keys-on-the-account-page"></a>Verwalten von Schlüsseln auf der Kontoseite

Klicken Sie im Menü auf der linken Seite unter „Einstellungen“ auf **Schlüssel**.

![Verwalten von Kontoschlüsseln im Portal](./media/how-to-manage-account-keys/account-keys-portal.png)

Verwenden Sie Schlüssel, um sich für Azure Location Based Services zu authentifizieren, und beginnen Sie damit, Ihre standortbasierte Anwendung zu entwickeln.

## <a name="delete-an-account"></a>Löschen eines Kontos

Sie können ein Konto löschen, indem Sie oben im Navigationsbereich auf die Schaltfläche **Löschen** klicken.

![Löschen des Kontos im Portal](./media/how-to-manage-account-keys/account-delete-portal.png)
