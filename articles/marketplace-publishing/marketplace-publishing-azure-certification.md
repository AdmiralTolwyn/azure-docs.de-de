# Erhalt einer Vorabgenehmigung für das Microsoft Azure-Zertifizierungsprogramm

Dieser Artikel enthält Informationen, wie Sie die Microsoft Azure-Zertifizierung erhalten. Eine ausführliche Übersicht über das Microsoft Azure-Zertifizierungsprogramm und dessen Vorteile finden Sie unter [http://azure.com/certified](http://azure.com/certified)

> [AZURE.NOTE]Die Azure-Zertifizierung ist nur anwendbar, wenn Sie beabsichtigen, eine virtuelle Maschine, Projektmappenvorlage und Entwicklerdienste für den Azure Marketplace zu erstellen. Wenn Sie keines dieser Artefakte erstellen, überspringen Sie diesen Schritt und wechseln zu **Schritt 1**, [Microsoft-Verkäuferkonto: Erstellung und Registrierung](marketplace-publishing-accounts-creation-registration.md).

## Erste Schritte
1. Der erste Schritt auf dem Weg zur Zertifizierung Ihrer Anwendung oder des Diensts und der Integration in Azure Marketplace ist das Ausfüllen des Anwendungsformulars auf der Website [Microsoft Azure Certified](https://azure.microsoft.com/marketplace/partner-program/).

2. Wenn Sie die grundlegenden Kriterien erfüllen, bitten wir Sie um einige weitere Details zu Ihrem Unternehmen und die Anwendung oder den Dienst, die oder der zertifiziert werden soll. Wenn wir alle Details von Ihnen erhalten haben, überprüfen wir Ihre Anwendung, um zu gewährleisten, dass Sie gut zu Azure Marketplace passt. Wenn die Anwendung genehmigt ist, arbeiten wir mit Ihnen zusammen, um die Lösung in Azure Marketplace zu integrieren.

3. Wenn Sie mit einem Partner Account Manager oder einem DX Partner Manager zusammenarbeiten, fordern Sie diesen bitte dazu auf, Sie für das Azure-Zertifizierungsprogramm zu nominieren. Oder Sie besuchen die Website [Microsoft Azure Certified](http://azure.com/certified), klicken auf **“Informationen anfordern“** und füllen das Anwendungsformular aus.

**Verkäuferländer**

> [AZURE.WARNING]Um Ihre Dienste im Microsoft Azure Marketplace verkaufen zu können, muss die registrierte Organisation in einem der genehmigten Verkäuferländer ansässig sein. Diese Einschränkung hat auszahlungstechnische und steuerliche Gründe. Die Länderliste wird in Kürze erweitert, schauen Sie also regelmäßig vorbei. Eine vollständige Liste finden Sie in Abschnitt 1b des Dokuments unter diesem Link: [http://go.microsoft.com/fwlink/?LinkID=526833](http://go.microsoft.com/fwlink/?LinkID=526833).

## Nächster Schritt
Sobald Sie die Vorabgenehmigung vom Microsoft Azure-Zertifizierungsprogramm erhalten haben, können Sie mit **Schritt 1** fortfahren, [Microsoft-Verkäuferkonto: Erstellung und Registrierung](marketplace-publishing-accounts-creation-registration.md).
