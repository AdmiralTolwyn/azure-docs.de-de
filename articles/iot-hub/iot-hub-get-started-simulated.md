# <a name="azure-iot-hub-get-started-with-simulated-devices-tutorials"></a>Tutorials zu ersten Schritten für Azure IoT Hub mit simulierten Geräten

Diese Tutorials dienen als Einführung in Azure IoT Hub und die Geräte-SDKs. In den Tutorials werden allgemeine IoT-Szenarien behandelt, um die Fähigkeiten von IoT Hub zu veranschaulichen. Die Tutorials erläutern auch das Kombinieren von IoT Hub mit anderen Azure-Diensten und -Tools zum Erstellen leistungsfähiger IoT-Lösungen. In den in der folgenden Tabelle aufgeführten Tutorials wird veranschaulicht, wie simulierte IoT-Geräte erstellt werden.

| Programmiersprache |
|----------------------|
| [.NET][Sim_NET]      |
| [Java][Sim_Jav]      |
| [Node.js][Sim_Nd]    |
| [Python][Sim_Pyth]   |

Darüber hinaus können Sie ein IoT Edge-Gateway verwenden, um simulierten Geräten eine Verbindung mit Ihrem IoT Hub zu ermöglichen.

|----------------------|------------------- |
