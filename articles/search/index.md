# <a name="azure-search-documentation"></a>Azure Search-Dokumentation

Hier erfahren Sie, wie Sie benutzerdefinierten Web- oder Mobilanwendungen mithilfe von Azure Search (einem verwalteten Cloudsuchdienst) Suchfunktionen hinzufügen. Anhand von Tutorials, API-Referenzen und anderem Dokumentationsmaterial wird gezeigt, wie Sie eine skalierbare Volltextsuche in mehreren Sprachen sowie standortbezogene Suche, Filter und Facettennavigation, Vervollständigung von Abfragen, Treffermarkierung und benutzerdefinierte Analysen hinzufügen.
