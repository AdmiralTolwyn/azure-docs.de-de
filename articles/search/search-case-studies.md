# Verwendung von Azure Search: Fallstudien und Kundenberichte

Erfahren Sie, wie die folgenden Organisationen sich positiv entwickeln konnten, indem die vielfältigen Suchfunktionen von Azure Search in ihre Onlineanwendungen integriert wurden.

[![][1]](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18596)

autoTRADER.ca erstellt eine IT-Infrastruktur basierend auf den Platform-as-a-Service (PaaS)-Funktionen von Microsoft Azure. Das Unternehmen hat mit der Verwendung von Azure begonnen, um Millionen von Images zu hosten und bereitzustellen. Es verlagert die Anwendungssuchfunktionen in Azure und möchte die gesamten Webeigenschaften zu Azure migrieren. So kann das Unternehmen seine Dienstleistungen verbessern und neue Angebote erstellen – und muss seine eigene Infrastruktur nicht mehr selbst verwalten.


[![][2]](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18633)

Wie kann es eine kleine Suchmaschine für albanische Muttersprachler heutzutage mit den großen Suchanbietern aufnehmen? Indem die lokale Sprache verstanden wird, eine zielgerichtete Benutzeroberfläche erstellt wird und die Microsoft Cloud-Suchtechnologie eingesetzt wird.


[![][3]](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18662)

JLL (früher Jones Lang LaSalle) möchte das Entwerfen und Hosten seiner Anwendungen ändern, indem die Vorteile der Cloud genutzt werden. Durch den Einsatz von Microsoft Azure kann JLL seine weltweiten Mitarbeiter und Kunden auf flexiblere und besser skalierbare Weise bedienen und sich auf Lösungen konzentrieren, anstatt auf die Infrastruktur.

[![][6]](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18667)

Es ist eine typische Tech Startup-Geschichte: Entwickler mit einer Idee erkennen einen Markt und möchten sofort loslegen. Der Markt? Einzelhandel. Die Idee? Back-End-Dienste für die Entwicklung von Webanwendungen und mobilen Anwendungen zur Wiederbelebung eines schwächelnden Einkaufserlebnisses im Geschäft. XOMNI nutzt Microsoft Azure zum schnellen Implementieren einer NoSQL-Datenbank und einer Volltextsuche. Kunden verfügen somit über eine skalierbare Plattform zum Entwickeln von Clientsoftware-Apps, Integrieren in soziale Netzwerke und Analysieren von Daten zur Schaffung von umfassenden, interaktiven Erlebnissen für Konsumenten.


[![][4]](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=19540)

Als Startup mit großen Plänen für seine Suchwebsite im Bereich des Gesundheitswesens benötigte Medihoo schnelle und leistungsstarke Suchfunktionen, für die keine hohen Investitionen in neue Infrastruktur erforderlich waren. Das Unternehmen hat sich für die Erstellung seiner Website auf der Microsoft Azure-Cloudplattform mit der Azure Search-Lösung entschieden, die schnelle Leistung, viele Funktionen und Geodaten sowie Unterstützung in mehreren Sprachen bietet. So können alle Benutzer – vor allem Auswanderer und Reisende – an jedem Ort der Welt auf qualitativ hochwertige Informationen zum Gesundheitswesen zugreifen.


[![][5]](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=21252)

Ein unabhängiger Immobilienhändler hat erkannt, dass die Cloud die beste Option darstellt. Weichert Realtors stellt auf die Cloud um, was zu einer starken Veränderung bei den Geschäftsabläufen des Unternehmens führt. Durch die Migration zu Microsoft Azure konnte das Unternehmen seine Betriebskosten um 60 Prozent reduzieren, nachdem teure lokale Hardware und Software ausgemustert wurde, die CRM-Lösung mit robusten Suchfunktionen komplett erneuert wurde und viele neue Datendienste ermöglicht wurden, die für die Zukunft des Unternehmens von entscheidender Bedeutung sind.
