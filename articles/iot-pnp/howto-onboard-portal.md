# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>Onboarding von Geräten im Azure Certified for IoT-Portal

Sie können das [Azure Certified for IoT](https://aka.ms/ACFI)-Portal für folgende Aufgaben verwenden:

- Schließen Sie den Zertifizierungsprozess für Ihr IoT-Gerät ab.
- Suchen Sie nach Funktionsmodellen für Geräte in IoT Plug & Play (Vorschau). Mithilfe dieser Modelle können Sie im Handumdrehen IoT-fähige Geräte erstellen und in Lösungen integrieren.
- Veröffentlichen Sie ein Gerätefunktionsmodell im öffentlichen Repository.

## <a name="sign-in"></a>Anmelden

Verwenden Sie Ihr Microsoft-_Geschäftskonto_, um sich beim [Azure Certified for IoT](https://aka.ms/ACFI)-Portal anzumelden. Bei der Anmeldung überprüft das Portal Ihre Mitgliedschaft beim Microsoft Partner Center:

- Wenn Ihre Organisation bereits Mitglied von [Microsoft Partner Center](https://docs.microsoft.com/partner-center/) ist, wird Ihnen Zugriff auf alle Funktionen im Portal gewährt.
- Wenn Ihre Organisation kein Mitglied von [Microsoft Partner Center](https://docs.microsoft.com/partner-center/) ist, können Sie Zertifizierungstests durchführen und die Repositorys mit Gerätefunktionsmodellen durchsuchen. Sie können Ihr Gerät nicht im Katalog mit zertifizierten IoT-Geräten veröffentlichen.
- Wenn Sie in einer Benachrichtigung darüber informiert werden, dass Ihre Microsoft Partner Network-ID fehlt, müssen Sie ein [Partner Center-Konto erstellen](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account). Wenn Ihr Konto genehmigt wurde, können Sie zum Portal zurückkehren und Ihr Gerät veröffentlichen. Weitere Informationen finden Sie in den [häufig gestellten Fragen zu Partner Center](https://support.microsoft.com/help/4340639/partner-center-account-faqs).

Bevor Sie Ihr Gerät im Katalog mit zertifizierten IoT-Geräten veröffentlichen, lesen und akzeptieren Sie die Programmvereinbarung. Sie finden die Vereinbarung auf der Azure Certified for IoT-Portalseite mit den Einstellungen.
