#  Microsoft Azure Government - Imagegalerie

<p> Die Imagegalerie von Microsoft Azure Government ähnelt dem öffentlichen Azure-Portal.  Kunden können auch vorgefertigte Images von Microsoft und unseren Partnern bereitstellen oder ihre eigenen VHDs hochladen. Dies bietet Flexibilität, um bei Bedarf Ihre eigenen standardisierten Images bereitzustellen.

Unten ist eine Liste angeführt, welche die in der Bildergalerie von Azure Government verfügbaren Images enthält.  Einige der vorgefertigten Images enthalten eine Zahlung für bestimmte Software-Lizenzierungen.  Informieren Sie sich auf der Seite <a href="http://azure.microsoft.com/pricing/details/virtual-machines/">Preisgestaltung für virtuelle Computer</a>, wenn Sie nähere Anleitungen benötigen und kontaktieren Sie Ihr Microsoft-Konto-Team oder den Händler von Azure Government bezüglich bestimmter Preise.
