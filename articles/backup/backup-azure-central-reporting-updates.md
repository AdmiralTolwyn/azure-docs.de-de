# <a name="updating-azure-backup-central-reporting-content-pack"></a>Aktualisieren des Inhaltspakets für die zentrale Azure Backup-Berichterstellung 

Mit dem [Azure Backup-Inhaltspaket](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) können Berichte zur Sicherung zentral angezeigt werden. Das Inhaltspaket wird regelmäßig aktualisiert, um weitere Funktionen und Fehlerbehebungen hinzuzufügen. In diesem Artikel werden Sie durch die Schritte zum Aktualisieren des Inhaltspakets und zum Zurückstellen des Updates sowie durch die im Laufe der Zeit vorgenommenen Updates geführt.

## <a name="how-to-get-updates-to-the-content-pack"></a>Abrufen von Updates für das Inhaltspaket

### <a name="to-get-the-updated-content-pack"></a>So rufen Sie das aktualisierte Inhaltspaket ab
Ihre Kopie des Inhaltspakets wird automatisch aktualisiert, wenn Sie keine Änderungen daran vorgenommen haben. Falls das Inhaltspaket geändert wurde, erhalten Sie eine Benachrichtigung in PowerBI sowie eine E-Mail-Benachrichtigung. Sie können auswählen, ob Sie das aktualisierte Inhaltspaket abrufen möchten. 

### <a name="to-delay-the-update"></a>So stellen Sie das Update zurück
Die bewährte Methode besteht darin, das Inhaltspaket in einen [benutzerdefinierten Arbeitsbereich](https://youtu.be/26zyOtyHPJM?t=1m57s) zu importieren. Sie haben jetzt die Möglichkeit, die Berichte zu bearbeiten.
Wie bereits erwähnt wird bei einer Änderung des Inhaltspakets eine Benachrichtigung in Power BI angezeigt. Sie können festlegen, dass Sie das Inhaltspaket später erhalten möchten. 

## <a name="coming-soon"></a>In Kürze verfügbar
   
Das Azure Backup-Inhaltspaket wird aktualisiert, um zusätzlich zur aktuellen Unterstützung von MAB und Azure-VM-Sicherung mehr Workloads (etwa SQL in der IaaS-VM-Sicherung) und SC DPM zu unterstützen. Das bedeutet, dass Sie in Kürze alle Sicherungsdaten an einem zentralen Ort anzeigen und analysieren können. Die [Berichte können außerdem angepasst werden](https://youtu.be/26zyOtyHPJM), um den Anforderungen Ihrer Organisation gerecht zu werden.

Die für das Azure Backup-Inhaltspaket vorkonfigurierten Berichte werden geändert, um sie für verschiedene Workloads aussagekräftiger zu gestalten. Hier finden Sie einen kurzen Überblick über die neuen Berichte:
