# Erste Schritte mit Azure AD

[Was ist Azure AD?](active-directory-whatis.md)<br> [Wie funktioniert Azure AD?](active-directory-works.md)<br> [Nächste Schritte](active-directory-next-steps.md)<br> [Erfahren Sie mehr](active-directory-learn-map.md)


Wollen Sie loslegen? Azure AD ist für alle Benutzer kostenlos verfügbar. Registrieren Sie sich jetzt für ein Azure-Testabonnement. Ihr Verzeichnis wartet schon auf Sie. Das Verzeichnis gehört Ihnen, unabhängig davon, ob Sie etwas kaufen. [Azure AD jetzt testen](http://azure.microsoft.com/trial/get-started-active-directory/)

## Woher weiß ich, welches Verzeichnis ich für das Testen, das Staging und die Produktion verwenden soll?

Wenn Sie über ein Abonnement für einen Microsoft-Clouddienst wie Office 365 verfügen, haben Sie bereits ein Verzeichnis in Azure AD, das Sie verwenden können. Dieses Verzeichnis authentifiziert bereits Benutzer, die sich bei Office 365 anmelden. Sie brauchen sich bei Azure lediglich mit dem Geschäfts- oder Schulkonto zu registrieren, mit dem Sie sich auch bei Office 365 anmelden. Sie können das Verzeichnis dann im Azure-Verwaltungsportal verwalten und nach Bedarf Verzeichnisse für Tests und Arbeit hinzufügen. [Erfahren Sie hier mehr über das Verwalten des Verzeichnisses für Ihr Office 365-Abonnement in Azure](active-directory-how-subscriptions-associated-directory.md#manage-the-directory-for-your-office-365-subscription-in-azure).

Wenn Sie über noch kein Abonnement für einen Microsoft-Clouddienst verfügen, wird Ihnen bei der Registrierung für Azure ein neues Verzeichnis für Schule oder Arbeit bereitgestellt, das Sie als Ihr primäres Verzeichnis verwenden können.


## Wie füge ich meinen eigenen registrierten Domänennamen hinzu?
Sie möchten einen benutzerdefinierten Domänennamen hinzufügen oder Ihren eigenen registrierten Domänennamen verwenden? [Erfahren Sie hier mehr über das Hinzufügen einer benutzerdefinierten Domäne](active-directory-add-domain.md).

## Wie füge ich mein Unternehmensbranding hinzu?
Mit Azure AD Premium können Sie der Anmeldeseite und der Zugriffsbereichsseite von Azure Ihr Firmenlogo und andere Corporate Identity hinzufügen. [Erfahren Sie hier mehr über das Hinzufügen eines Unternehmensbranding zur Anmelde- und Zugriffsbereichsseite](active-directory-add-company-branding.md).

## Wie integriere ich Azure AD mit einem anderen Verzeichnis?
Sie können eine Verzeichnissynchronisierung einschließlich einer Kennwortsynchronisierung zwischen einem lokalen Verzeichnis und Azure AD einrichten und für den Zugriff sowohl auf lokale Anwendungen als auch auf Anwendungen in der Cloud einmaliges Anmelden (SSO) aktivieren. [Erfahren Sie hier mehr über die Verzeichnisintegration](active-directory-aadconnect.md).

## Wie erstelle und bearbeite ich Benutzer?
Sie können für jeden Benutzer, der auf einen Microsoft-Clouddienst zugreifen muss, ein eigenes Konto erstellen. Benutzer haben standardmäßig keine Administratorberechtigung, Sie können ihnen diese jedoch erteilen. [Erfahren Sie hier mehr über das Erstellen von Azure AD-Benutzern](active-directory-create-users.md).

## Anzeigen von Zugriffs- und Nutzungsberichten
Azure AD bietet Administratoren eine Reihe von Berichten. Die kostenpflichtigen Editions von Azure AD bieten dabei noch wesentlich mehr Berichte, wobei Sie in der Berichts-API auch benutzerdefinierte Berichte erstellen können. [Erfahren Sie hier mehr über Azure AD-Berichte](active-directory-view-access-usage-reports.md).
