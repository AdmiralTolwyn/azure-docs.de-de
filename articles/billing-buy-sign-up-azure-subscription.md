# Erwerben oder Registrieren eines Azure-Abonnements

Dieses Thema unterstützt Sie bei den folgenden Aufgaben:

* Registrieren für ein kostenloses Azure-Testabonnement
* Durchführen eines Upgrades für Ihr kostenloses Testabonnement auf ein Abonnement mit nutzungsbasierter Bezahlung
* Aktivieren Ihres Azure-Abonnements mit einem Aktivierungsschlüssel für Onlinedienste (5 x 5-Schlüssel)

## Registrieren für ein kostenloses Azure-Testabonnement
Wenn Sie sich für ein kostenloses Azure-Testabonnement registrieren möchten, müssen Sie eine Telefonnummer, eine Kreditkarte und ein [Microsoft-Konto](https://www.microsoft.com/de-DE/account/faq.aspx) (früher Windows Live ID) bzw. ein Schul-/Geschäftskonto (auch als Organisations-ID bezeichnet) angeben. Ihre Telefonnummer und die Kreditkarte dienen nur zur Überprüfung Ihrer Identität, und Ihre Kreditkarte wird nicht belastet.

>[AZURE.NOTE]Wir platzieren eine Verifizierungsbuchung von 1 $ in den Vereinigten Staaten bzw. einen ähnlichen Betrag in der Währung Ihrer Region zur Autorisierung Ihrer Kreditkarte, um deren Gültigkeit sicherzustellen. Dieser Betrag wird innerhalb von 3 bis 5 Tagen, je nach den Richtlinien Ihres Finanzinstituts, wieder freigegeben. Ihre Kreditkarte wird nicht belastet.

So registrieren Sie sich für ein kostenloses Azure-Testabonnement

1. Öffnen Sie die [Seite mit der kostenlosen Azure-Testversion](https://azure.microsoft.com/de-DE/pricing/free-trial/), und wählen Sie dann **Jetzt testen**.

2. Melden Sie sich bei Ihrem Microsoft-Konto oder bei Ihrem Schul-/Geschäftskonto an.

3. Geben Sie im Abschnitt **Persönliche Informationen** Ihre persönlichen Informationen ein. Beachten Sie, dass das Land nicht nachträglich geändert werden kann. ![Bildschirm für persönliche Informationen](./media/billing-buy-sign-up-azure-subscription/AboutYou.png)

4. Bei der Überprüfung per Telefon wird über die von Ihnen bevorzugte Methode (Telefonanruf oder SMS) ein Code an Ihr Telefon gesendet. Wenn Sie den erhaltenen Code eingeben, können wir Ihre Identität überprüfen. </br></br> Wählen Sie im Abschnitt **Überprüfung per Telefon** die internationale Vorwahl für Ihr Land aus, geben Sie Ihre Telefonnummer (ohne interne Vorwahl) ein, und wählen Sie dann **SMS senden** oder **Anrufen**. Nachdem Sie einen Überprüfungscode erhalten haben, geben Sie ihn im Feld **Code überprüfen** ein, und wählen Sie dann die Schaltfläche **Code überprüfen**, um Ihre Telefonnummer zu verifizieren. Eine VOIP-Rufnummer kann nicht für die Überprüfung des Anschlusses verwendet werden.</br> ![Bildschirm zur Überprüfung des Anschlusses](./media/billing-buy-sign-up-azure-subscription/PhoneVerify.png)</br>Wenn Sie keine SMS-Nachricht und keinen Anruf erhalten, verwenden Sie eine andere Telefonnummer, oder wenden Sie sich an den [Microsoft Support](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade). </br>
5. Nach der Überprüfung des Anschlusses fahren Sie mit dem Abschnitt zur Kartenüberprüfung fort.

 ![Kartenverifizierung](./media/billing-buy-sign-up-azure-subscription/VardVerify.png)</br>

6. Wählen Sie nach der Überprüfung der Kreditkarte die Option unter **Vereinbarung** aus, und wählen Sie dann **Registrieren**. ![Anmeldung](./media/billing-buy-sign-up-azure-subscription/Signup.png)</br> Es dauert einige Minuten, bis Ihr Abonnement eingerichtet ist. Wenn die Abonnementeinrichtung nicht in 15 Minuten abgeschlossen ist, wenden Sie sich an den [Microsoft Support](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade). </br> Wenn Sie eine Fehlermeldung erhalten, führen Sie die in der Nachricht angegebenen Schritte durch, um das Problem zu beheben. Eine typische Fehlermeldung lautet:

**Wir konnten den Vorgang leider nicht abschließen. Versuchen Sie es später noch einmal, oder laden Sie die Webseite neu.** </br>

Versuchen Sie in diesem Fall, den Registrierungsvorgang in einem anderen Browser abzuschließen. Falls das Problem weiterhin auftritt, wenden Sie sich an den [Microsoft Support](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade).

![Anmeldefehler1](./media/billing-buy-sign-up-azure-subscription/signuperror1.png)</br>7. Wenn das Abonnement bereit steht, wählen Sie **Mit der Verwaltung meines Diensts beginnen**. ![Dienst starten](./media/billing-buy-sign-up-azure-subscription/startservice.png)

## Durchführen eines Upgrades für Ihr kostenloses Testabonnement auf ein Abonnement mit nutzungsbasierter Bezahlung

1. Melden Sie sich mithilfe Ihres Microsoft-Kontos bzw. Ihres Schul-/Geschäftskontos, das Sie für die Registrierung bei der kostenlosen Testversion verwendet haben, beim [Azure-Kontoportal](https://account.windowsazure.com/subscriptions) an.
2. Wählen Sie das Abonnement aus, und wählen Sie dann **Upgrade**.</br>![Abrechnungsseite](./media/billing-buy-sign-up-azure-subscription/billpage.png)
3. Wählen Sie **Ja**, und wählen Sie dann **Jetzt aktualisieren**. </br>![Upgrade](./media/billing-buy-sign-up-azure-subscription/Upgrade.png)

Nach dem Upgrade auf ein Abonnement mit nutzungsbasierter Bezahlung können Sie weiterhin die Vorteile Ihrer Testversion nutzen (30 Tage/200 $), bis eine von beiden Optionen ausgeschöpft ist.

## Aktivieren Ihres Azure-Abonnements mit einem Aktivierungsschlüssel für Onlinedienste (5 x 5-Schlüssel)

Wenn Sie Azure in Open-Gutschriften über einen Händler erworben haben, befolgen Sie diese Schritte zum Aktivieren Ihres Abonnements:

1. Öffnen Sie die Seite mit dem [Azure in Open-Lizenzschlüssel](http://azure.microsoft.com/offers/ms-azr-0111p/), und wählen Sie die Option zum Aktivieren eines neuen oder eines vorhandenen Abonnements.
2. Geben Sie den Product Key ein, und aktivieren Sie die erste Vereinbarungsoption (und auch die zweite Option, wenn Sie spezielle Angebote erhalten möchten), um das Abonnement zu aktivieren.</br>![OSA-SCHLÜSSEL](./media/billing-buy-sign-up-azure-subscription/OSAkey.png)
3. Wählen Sie **Aktivieren**.
