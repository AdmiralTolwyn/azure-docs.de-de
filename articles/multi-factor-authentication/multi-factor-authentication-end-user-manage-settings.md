# Ändern der Einstellungen mit Azure Multi-Factor Authentication
Die folgenden Informationen unterstützen Sie beim Verwalten der Multi-Factor Authentication-Einstellungen.

## Die Vorgehensweise zum Ändern der Einstellungen hängt davon ab, wie Sie Multi-Factor Authentication verwenden.
Sie können die Multi-Factor Authentication-Einstellungen an verschiedenen Stellen ändern. Die genaue Stelle hängt davon ab, wie Sie Multi-Factor Authentication verwenden. Wählen Sie die Option, die am besten auf Sie zutrifft.

Verwendung von Multi-Factor Authentication|Beschreibung
:------------- | :------------- | 
[Ich verwende sie mit Office 365.](multi-factor-authentication-end-user-manage-o365.md)| Dies bedeutet, dass Sie die Einstellungen über das Office 365-Portal ändern sollten.
[Ich verwende sie mit Microsoft Azure.](multi-factor-authentication-end-user-manage-azure.md)| Dies bedeutet, dass Sie die Einstellungen über das Azure-Portal ändern sollten.
[Nicht bekannt](multi-factor-authentication-end-user-manage-myapps.md)|Dies bedeutet, dass Sie sich bei [http://myapps.microsoft.com](http://myapps.microsoft.com) anmelden und dort die Einstellungen ändern sollten.
