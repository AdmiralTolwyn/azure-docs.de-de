# Verwalten der Einstellungen für Azure Multi-Factor Authentication im Office 365-Portal


Wenn Sie Multi-Factor Authentication mit Office 365 verwenden, sollten Sie die Einstellungen für die zusätzliche Sicherheitsüberprüfung über das Office 365-Portal verwalten.

## So greifen Sie auf die Einstellungen für die zusätzliche Sicherheitsüberprüfung im Office 365-Portal zu

<ol>

<li>Melden Sie sich beim [Office 365-Portal] (https://login.microsoftonline.com/) an.</li>
<li>Klicken Sie oben auf das Zahnradsymbol. Eine Dropdownliste mit der Bezeichnung "Office&#160;365-Einstellungen" wird angezeigt.</li>


<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365a.png)</center>

<li>Klicken Sie auf "Office&#160;365-Einstellungen". Die Seite "Einstellungen" wird geöffnet.</li>
<li>Klicken Sie links auf "Zusätzliche Sicherheitsüberprüfung".</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-o365/o365.png)</center>

<li>Wählen Sie auf der rechten Seite "Aktualisieren meiner Telefonnummern für die Kontosicherheit" aus. Sie werden auf eine neue Seite weitergeleitet. Sie müssen Ihre Identität erneut über Ihr Mobiltelefon oder die mobile App bestätigen.</li>
<li>Sobald dieser Vorgang abgeschlossen ist, sollten Sie sich auf der Nachweisseite befinden, auf der Sie die Einstellungen ändern können.</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-o365/addsec.png)</center>
