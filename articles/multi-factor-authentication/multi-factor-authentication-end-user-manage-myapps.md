# Verwalten der Einstellungen für Azure Multi-Factor Authentication im Portal "Meine Apps"


Wenn Sie Multi-Factor Authentication mit Office 365 verwenden, sollten Sie die Einstellungen für die zusätzliche Sicherheitsüberprüfung über das Office 365-Portal verwalten.

## So greifen Sie auf die Einstellungen für die zusätzliche Sicherheitsüberprüfung im Portal "Meine Apps" zu
<ol>
<li>Melden Sie sich bei [https://myapps.microsoft.com](https://myapps.microsoft.com) an.</li>
<li>Wählen Sie oben "Profil" aus.</li>
<li>Klicken Sie auf Ihren Benutzernamen, und wählen Sie "Zusätzliche Sicherheitsüberprüfung" aus.</li>

<center>![Cloud](./media/multi-factor-authentication-end-user-manage-myapps/myapp.png)</center>

<li>Jetzt sollte die Nachweisseite angezeigt werden, auf der Sie Ihre Einstellungen ändern können.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-myapps/proofup.png)</center>
