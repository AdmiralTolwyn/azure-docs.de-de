# Verwalten der Einstellungen für Azure Multi-Factor Authentication im Azure-Portal

Wenn Sie Multi-Factor Authentication mit Azure verwenden, sollten Sie die Einstellungen für die zusätzliche Sicherheitsüberprüfung über das Azure-Portal verwalten.

## So greifen Sie auf die Einstellungen für die zusätzliche Sicherheitsüberprüfung im Azure-Portal zu

<ol>
<li>Melden Sie sich beim Azure-Portal an. <li>Klicken Sie oben im Azure-Portal auf Ihren Benutzernamen. Eine Dropdownliste wird angezeigt. <li>Wählen Sie in der Dropdownliste "Zusätzliche Sicherheitsüberprüfung" aus. Der Abschnitt "Zusätzliche Sicherheitsüberprüfung" wird geöffnet.

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/changesettings.png)</center>

<li>Jetzt sollte die Nachweisseite angezeigt werden, auf der Sie Ihre Einstellungen ändern können.</li>

<center>![Setup](./media/multi-factor-authentication-end-user-manage-azure/proofup.png)</center>
