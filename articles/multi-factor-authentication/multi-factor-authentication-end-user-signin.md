# Der Anmeldevorgang mit Azure Multi-Factor Authentication
Die folgenden Informationen sollen Sie beim Anmeldevorgang unterstützen.


## Was sieht Ihr Anmeldevorgang aus?
Der Anmeldevorgang hängt davon ab, wie Sie sich anmelden und Multi-Factor Authentication verwenden. In diesem Abschnitt finden Sie Informationen dazu, was Sie bei der Anmeldung erwartet. Wählen Sie die Option aus, die am besten auf Sie zutrifft.


Vorgehensweise|Beschreibung
:------------- | :------------- | 
[Anmeldung mit Mobil- oder Bürotelefon](multi-factor-authentication-end-user-signin-phone.md) | Beschreibt die Anmeldung mit einem Mobil- oder Bürotelefon.
[Anmelden mit der mobilen App über eine Benachrichtigung](multi-factor-authentication-end-user-signin-app-notify.md) | Beschreibt die Anmeldung bei der mobilen App mit einer Benachrichtigung.
[Anmelden mit der mobilen App über einen Überprüfungscode](multi-factor-authentication-end-user-signin-app-verify.md)|Beschreibt die Anmeldung bei der mobilen App mit einem Überprüfungscode.
[Anmelden mit einer alternativen Methode](multi-factor-authentication-end-user-signin-alt.md)|Hier erfahren Sie, was Sie erwartet, wenn Sie eine alternative Methode verwenden möchten.
