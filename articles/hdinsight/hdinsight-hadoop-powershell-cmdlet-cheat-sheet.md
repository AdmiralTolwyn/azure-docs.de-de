# Microsoft Azure HDInsight – Cheat Sheet zu PowerShell-Cmdlets für gängige Hadoop-Aufgaben

Microsoft Azure HDInsight bietet PowerShell-Cmdlets zum Ausführen von Hadoop-Aufgaben in einem Cluster. Das **Microsoft Azure HDInsight-Cheat Sheet zu PowerShell-Cmdlets** zeigt PowerShell-Cmdlets für die gängigsten Hadoop-Aufgaben, wie z. B. Cluster erstellen, Aufträge ausführen, Cluster löschen usw. Das Cheat Sheet enthält auch eine Liste aller PowerShell-Cmdlets, die für Microsoft Azure HDInsight zur Verfügung stehen.

## HDInsight Cheat Sheet: Dienstverwaltungsmodell

**Laden Sie das Cheat Sheet hier herunter: [Microsoft Azure HDInsight – Cheat Sheet zu PowerShell-Cmdlets für gängige Hadoop-Aufgaben](http://download.microsoft.com/download/B/7/D/B7DBB509-164D-4343-9894-12D1FB053776/HDI_PowerShell_Cmdlet_CheatSheet.pdf)**

**Die Referenzdokument zu Microsoft Azure HDInsight PowerShell finden Sie hier: [Azure HDInsight PowerShell-Referenz](https://msdn.microsoft.com/library/azure/dn858087.aspx)**
