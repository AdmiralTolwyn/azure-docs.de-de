# <a name="manage-and-connect-to-your-subscription-in-azure-germany"></a>Verwalten Ihres Abonnements und Herstellen einer Verbindung mit Ihrem Abonnement in Azure Deutschland
Azure Deutschland verfügt über eindeutige URLs und Endpunkte für die Verwaltung Ihrer Umgebung. Es ist wichtig, dass die richtigen Verbindungen zum Verwalten der Umgebung über das Azure-Portal oder PowerShell verwendet werden. Nach dem Herstellen der Verbindung mit der Azure Deutschland-Umgebung können die normalen Vorgänge zum Verwalten der Dienste durchgeführt werden, sofern die Komponente bereitgestellt wurde.

> [!IMPORTANT]
>
>Dies gilt für Azure-Kunden der Microsoft Cloud Deutschland, die Azure-Dienste mit nutzungsbasierter Bezahlung und Zahlung per Kreditkarte direkt von Microsoft erwerben. 
>
>Ab dem 14. September 2019 tritt ein neues Mandat der Europäischen Union in Kraft, das erfordert, dass Kreditkartenzahlungen von Kunden eine strenge Kundenauthentifizierungsaufforderung (Strong Customer Authentication, SCA) durchlaufen. Folglich können betroffene Kunden erst nach der [Migration zu einer neuen Azure-Region](./germany-migration-main.md) ein neues Abonnement erstellen oder Zahlungsinformationen hinzufügen oder aktualisieren. 
>
>**Aktion erforderlich**:
>
>Wenn Sie vom Mandat der Europäischen Union betroffen sind, müssen Sie bis zum 14. September 2019 Ihr Zahlungsmittel aktualisieren und neue Abonnements erstellen. Im [Kontocenter](https://account.windowsazure.de) können Sie Ihre Kreditkarteninformationen aktualisieren oder auf eine Zahlung per Rechnung umstellen. Wählen Sie im Portal das Abonnement aus, für das Sie die Zahlungsmethode ändern möchten, und klicken Sie dann auf „Zahlungsmethoden verwalten“. Wenn Sie auf eine Zahlung per Rechnung umstellen möchten, dies jedoch nicht als Option angezeigt wird, wenden Sie sich an den Support. Nach dem 14. September können Sie Ihr Zahlungsmittel nicht mehr aktualisieren. 
>

## <a name="connect-by-using-the-portal"></a>Herstellen einer Verbindung mithilfe des Portals
Sie können Ihre Azure-Ressourcen im [Azure-Portal](https://portal.microsoftazure.de) verwalten. Wenn Sie Ihre Kontoinformationen verwalten möchten, besuchen Sie das [Kontocenter](https://account.windowsazure.de).

## <a name="create-subscriptions"></a>Erstellen von Abonnements
Im [Kontocenter](https://account.windowsazure.de) können Sie Abonnements für Ihr Konto erstellen.
