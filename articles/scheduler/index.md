# <a name="scheduler-documentation"></a>Dokumentation zu Scheduler

Mit Azure Scheduler können Sie Aktionen, die in der Cloud ausgeführt werden sollen, deklarativ beschreiben.  Anschließend werden diese Aktionen automatisch geplant und ausgeführt.  Scheduler speichert dann den Verlauf der Auftragsausführungsergebnisse zur späteren Überprüfung.
