# Was sind Websites?

[WACOM.INCLUDE [Haftungsausschluss](../includes/disclaimer.md)]

Websites sind Webanwendungshosts, die beliebte Webanwendungstechnologien wie .NET, Node.js und PHP unterstützen, ohne Codeänderungen an vorhandenen Anwendungen erforderlich zu machen. Alle gängigen Programmiermodelle und -ressourcen, die .NET-, Node.js- und PHP-Entwickler verwenden, um auf Ressourcen wie Dateien und Datenbanken zuzugreifen, funktionieren weiterhin wie erwartet auf den in Azure bereitgestellten Websites. Websites bieten Zugriff auf Datei- und Datenbankressourcen sowie auf alle Standardkonfigurationsdateien (z. B. web.config, php.ini, config.js), die heutzutage von Webanwendungen verwendet werden. Eine Website ist der Host für eine Webanwendung, die in der Cloud ausgeführt wird und die Sie steuern und verwalten können.
