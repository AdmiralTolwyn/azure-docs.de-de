# Verwenden von Visual Studio zum Erstellen eines Clouddienstcontainers

Wählen Sie zum Anzeigen des **Server-Explorer** in Visual Studio in der Menüleiste nacheinander die Optionen **Ansicht** und **Server-Explorer**.

1.  Erweitern Sie den Knoten **Azure**.

2.  Wenn der Benutzer nicht angemeldet ist, wird automatisch ein Anmeldedialogfeld geöffnet. Befolgen Sie die Anweisungen, um sich Ihrem Azure-Abonnement anzumelden.

3.  Klicken Sie mit der rechten Maustaste auf den Knoten **Clouddienst**, und wählen Sie **Clouddienst erstellen…** aus.

4.  Das Dialogfeld **Clouddienst erstellen** wird geöffnet. Füllen Sie die Felder **Abonnement**, **Name** und **Region oder Affinitätsgruppe** aus, und klicken Sie auf die Schaltfläche **Erstellen**, um einen leeren Clouddienstcontainer zu erstellen.

## Nächste Schritte

Nachdem Sie einen Clouddienstcontainer erstellt haben, können Sie nun einen Clouddienst [erstellen und bereitstellen](cloud-services-how-to-create-deploy.md).
