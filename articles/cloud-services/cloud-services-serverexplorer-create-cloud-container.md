# Verwenden von Visual Studio zum Erstellen eines Cloud-Dienst-Containers

Um den **Server-Explorer** in Visual Studio anzuzeigen, wählen Sie in der Menüleiste nacheinander die Optionen **Ansicht** und **Server-Explorer**. 

1.  Erweitern Sie den Knoten **Azure**.

2.  Wenn der Benutzer nicht angemeldet ist, wird automatisch ein Anmeldedialogfeld geöffnet. Befolgen Sie die Anweisungen, um sich Ihrem Azure-Abonnement anzumelden.

3.  Klicken Sie mit der rechten Maustaste auf den Knoten **Cloud-Dienst**, und wählen Sie **Cloud-Dienst erstellen...** aus.

4.  Das Dialogfeld **Cloud-Dienst erstellen** wird geöffnet. Füllen Sie die Felder **Abonnement**, **Name** und **Region oder Affinitätsgruppe** aus, und klicken Sie auf die Schaltfläche **Erstellen**, um einen leeren Cloud-Dienst-Container zu erstellen.

## Nächste Schritte

Nachdem Sie einen Cloud-Dienst-Container erstellt haben, können Sie nun einen Cloud-Dienst [erstellen und bereitstellen](cloud-services-how-to-create-deploy.md) .
