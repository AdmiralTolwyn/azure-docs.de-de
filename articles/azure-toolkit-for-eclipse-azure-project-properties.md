# <a name="azure-project-properties"></a>Azure-Projekteigenschaften
Im Azure-Toolkit für Eclipse können verschiedene Einstellungen für Ihr Azure-Projekt festgelegt werden.

## <a name="configuring-azure-project-properties"></a>Konfigurieren der Azure-Projekteigenschaften
Die Eigenschaften Ihres Azure-Projekts werden über die Eigenschaftsdialogfelder für Ihr Azure-Projekt konfiguriert. Klicken Sie im Projekt-Explorer-Bereich von Eclipse mit der rechten Maustaste auf das Projekt, klicken Sie auf **Eigenschaften**, und wählen Sie dann das Untermenü **Azure** aus.

![][ic719480]

Auf dieser Eigenschaftenseite können Sie folgende Änderungen vornehmen: 

* **Dienstname:** Der Name Ihres Azure-Projekts.
* **Build per:** Gibt an, ob das Projekt zum Erstellen des Serveremulators oder zur Bereitstellung in Azure festgelegt ist.
* **Ziel-Betriebssystem:** Das bei der Bereitstellung Ihres Projekts in Anzure verwendende Betriebssystem.
