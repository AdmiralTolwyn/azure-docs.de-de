# Azure-Projekteigenschaften #

Im Azure-Toolkit für Eclipse können verschiedene Einstellungen für Ihr Azure-Projekt festgelegt werden.

## Konfigurieren der Azure-Projekteigenschaften ##

Die Eigenschaften Ihres Azure-Projekts werden über die Eigenschaftsdialogfelder für Ihr Azure-Projekt konfiguriert. Klicken Sie mit der rechten Maustaste des Projekts im Projekt-Explorer-Bereich von Eclipse, klicken Sie dann auf **Eigenschaften**, und wählen Sie dann das Untermenü **Azure** aus.

![][ic719480]

Auf dieser Eigenschaftenseite können Sie folgende Änderungen vornehmen:

* **Dienstname:** Der Name Ihres Azure-Projekts.
* **Build per:** Gibt an, ob das Projekt zum Erstellen des Serveremulators oder zur Bereitstellung in Azure festgelegt ist.
* **Ziel-Betriebssystem:** Das bei der Bereitstellung Ihres Projekts in Anzure verwendende Betriebssystem.
