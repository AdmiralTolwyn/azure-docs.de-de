# <a name="azure-network-watcher-documentation"></a>Azure Network Watcher-Dokumentation

Erfahren Sie, wie Sie Network Watcher verwenden. Dabei handelt es sich um einen Dienst, mit dem Sie Bedingungen auf der Ebene des Netzwerkszenarios überwachen und diagnostizieren können. Anhand von Tutorials, API-Referenzen und weiterer Dokumentation erfahren Sie, wie Sie mithilfe von Tools wie Paketerfassung, Topologie usw. Einblicke in Ihr Azure-Netzwerk erhalten.
