# Azure Compute-, Netzwerk- und Speicheranbieter für Linux-Anwendungen unter dem Azure Resource Manager-Bereitstellungsmodell
Wenn das Azure Resource Manager-Bereitstellungsmodell um Compute-, Netzwerk- und Speicherfunktionen erweitert wird, stellt dies eine erhebliche Vereinfachung der Bereitstellung und Verwaltung komplexer Anwendungen dar, die in IaaS ausgeführt werden. Für viele Anwendungen ist eine Kombination von Ressourcen wie virtuellen Netzwerken, Speicherkonten, virtuellen Computern und Netzwerkschnittstellen erforderlich. Mit dem Azure Resource Manager-Bereitstellungsmodell können Sie eine JSON-Vorlage erstellen, um alle Ressourcen gemeinsam als eine einzige Anwendung bereitzustellen und zu verwalten.
