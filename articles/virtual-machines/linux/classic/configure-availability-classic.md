# <a name="how-to-configure-an-availability-set-for-linux-virtual-machines-in-the-classic-deployment-model"></a>Konfigurieren einer Verfügbarkeitsgruppe für virtuelle Linux-Computer im klassischen Bereitstellungsmodell
> [!IMPORTANT] 
> Azure verfügt über zwei verschiedene Bereitstellungsmodelle für das Erstellen und Verwenden von Ressourcen: [Resource Manager- und klassische Bereitstellung](../../../resource-manager-deployment-model.md). Dieser Artikel befasst sich mit der Verwendung des klassischen Bereitstellungsmodells. Microsoft empfiehlt für die meisten neuen Bereitstellungen die Verwendung des Ressourcen-Manager-Modells. Sie können auch [Verfügbarkeitsgruppen in Resource Manager-Bereitstellungen konfigurieren](../../azure-cli-arm-commands.md#azure-availset-commands-to-manage-your-availability-sets) .
