# <a name="app-service-on-linux-documentation"></a>App Service unter Linux – Dokumentation
App Service unter Linux ist derzeit als öffentliche Vorschau verfügbar und ermöglicht Kunden das systeminterne Ausführen ihrer Web-Apps auf einer Linux-Plattform. Dies ermöglicht eine bessere Anwendungskompatibilität für bestimmte Anwendungsarten und vereinfacht die Migration vorhandener Web-Apps, die an anderer Stelle auf einer Linux-Plattform gehostet sind, zu Azure App Services.

Kunden können Web-Apps unter Linux nur innerhalb eines für Linux aktivierten App Service-Plans erstellen, und dieser App Service-Plan kann nur Linux-basierte Web-Apps hosten. 

> Der Preis für die öffentliche Vorschau von App Service unter Linux beträgt 50 % des regulären Preises. Der Preis wird berechnet, indem die Anzahl der von Ihrem App Service-Plan genutzten Computestunden halbiert wird.
> 
> Während der öffentlichen Vorschau wird keine SLA bereitgestellt.
