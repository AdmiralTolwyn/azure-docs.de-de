# <a name="azure-china-21vianet-support"></a>Azure China 21Vianet-Support
Azure bietet viele Self-Service-Supportoptionen und Support über 21Vianet.

## <a name="self-help-resources"></a>Ressourcen für die Selbsthilfe
- In den [FAQ](https://www.azure.cn/support/faq) werden häufige gestellte Fragen zu allgemeinen Informationen, Abonnements, Abrechnung und Internetinhaltsanbietern (ICPs) beantwortet. 
- Der [Azure-Betriebsleitfaden](https://docs.azure.cn/zh-cn/articles/) bietet Lösungen zu aktuellen Themen.
- [Azure School](https://school.azure.cn/) bietet Selbstlernressourcen mit Kontakt zu Azure-Experten.

## <a name="assisted-support"></a>Unterstützter Support
- [Senden Sie eine Onlinesupportanfrage](https://www.azure.cn/en-us/support/contact/) (in der englischen und chinesischen Version verfügbar).
- [Durchsuchen Sie den MSDN-Forumsupport](https://social.msdn.microsoft.com/Forums/zh-CN/home?forum=windowsazurezhchs).
- [Verwenden Sie die Azure Cloud Assistant-App, um eine Anfrage zu senden](https://www.azure.cn/azure-cloud-assistant/).

## <a name="get-premier-support"></a>Premier Support
Wenn Sie unternehmenskritische Lösungen unter Azure ausführen, ist die Nutzung des Premier Support-Angebots sinnvoll:
- Bewährte Beratungsdienstleistungen zur Optimierung Ihrer Azure-Investition.
- Ein erfahrener Kundendienstmitarbeiter, der Sie bei der Verwendung der Azure-Dienste unterstützt.
- Reaktiver Support mit höchster Priorität, um die Kontinuität der Dienste zu gewährleisten.

Weitere Informationen zum Erwerb des Premier Supports erhalten Sie vom Microsoft-Kontoteam. 

## <a name="operation-and-support-through-21vianet"></a>Betrieb und Support über 21Vianet
Bei Problemen mit Ihrer Azure-Bereitstellung melden Sie diese 21Vianet über das [Azure-Portal](https://portal.azure.cn/), oder rufen Sie die Supporthotline unter 400-089-0365 an.
 
Eine *Service Request* (SR) wird je nach Schweregrad der Auswirkungen auf Ihr Unternehmen innerhalb weniger Stunden bearbeitet:
- **Schweregrad A: Schwerwiegende Beeinträchtigung des Geschäftsbetriebs.** Sie erhalten innerhalb von höchstens zwei Stunden eine erste Antwort, und ein Supportmitarbeiter wird den ganzen Tag über kontinuierlich bis zur Lösung des Problems daran arbeiten. Es wird erwartet, dass Sie geeignete Ressourcen für die Bearbeitung der Anfrage bereitstellen, bis das Problem behoben ist, und den Supportmitarbeitern, die Ihren Fall bearbeiten, genaue Kontaktinformationen zukommen lassen.
- **Schweregrad B: Mittlere Beeinträchtigung des Geschäftsbetriebs.** Sie erhalten innerhalb von höchstens vier Stunden eine erste Antwort. Ein Supportmitarbeiter wird den ganzen Tag über kontinuierlich bis zur Lösung des Problems daran arbeiten, es sei denn, Sie ziehen die Anfrage zurück. Es wird erwartet, dass Sie geeignete Ressourcen bereitstellen, die die Arbeit weiterführen, und den Supportmitarbeitern, die Ihren Fall bearbeiten, genaue Kontaktinformationen zukommen lassen.
- **Schweregrad C: Minimale Beeinträchtigung des Geschäftsbetriebs.** Sie erhalten innerhalb von höchstens acht Stunden eine erste Antwort. Es wird erwartet, dass Sie den Supportmitarbeitern, die Ihren Fall bearbeiten, genaue Kontaktinformationen zukommen lassen.
