# <a name="azure-china-21vianet"></a>Azure China 21Vianet

Microsoft Azure, betrieben von 21Vianet, ist eine Cloudplattform – entwickelt zur Einhaltung von gesetzlichen Bestimmungen in China –, die eine wachsende Sammlung integrierter Clouddienste enthält. Dies sind die Dienste, die Entwickler und IT-Profis zum Erstellen, Bereitstellen und Verwalten von Anwendungen nutzen. Azure China 21Vianet ist eine physisch getrennte Instanz von Clouddiensten, die sich auf dem chinesischen Festland befindet. Sie wird von Shanghai Blue Cloud Technology Co., Ltd. („21Vianet“) unabhängig betrieben, die auch die Transaktionen abwickelt.

Azure China 21Vianet bietet erstklassige Sicherheit und die <a href="https://www.trustcenter.cn/en-us/">Compliancedienste</a>, die für alle auf dieser Architektur beruhenden Systeme und Anwendungen entsprechend den in China geltenden Vorschriften erforderlich sind. Kommerzielle Clouddienste von Microsoft halten die Richtlinien und Datenverarbeitungsvorschriften ein, die von der Regierung in China festgelegt wurden.
