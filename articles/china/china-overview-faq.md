# <a name="azure-china-21vianet-faq"></a>FAQ zu Azure China 21Vianet
Erhalten Sie Antworten auf häufig gestellte Fragen zu Microsoft Azure, ausgeführt über 21Vianet (Azure China 21Vianet). Zu den abgedeckten Themen gehören Konten und Abonnements, Portale sowie Vorgänge.

## <a name="accounts-and-subscriptions"></a>Konten und Abonnements
### <a name="how-are-accounts-and-subscriptions-related"></a>Wie hängen Konten und Abonnements zusammen?
Ein [Abonnement](https://www.azure.cn/support/faq/#What-is-the-relationship-between-accounts-and-subscriptions) für Azure China 21Vianet funktioniert wie ein globales Azure-Abonnement. Sie können beispielsweise über mehr als ein Abonnement mit jeweils mehr als einem Benutzerkonto verfügen und Azure-Dienste aus jedem Ihrer Abonnements erstellen und nutzen. Derselbe Benutzer ist in mehreren Abonnements zulässig. In Ihrer Rechnung sind die Kosten für jedes einzelne Abonnement sowie die Gesamtkosten für alle Abonnements aufgeführt.

## <a name="portals"></a>Portale

### <a name="what-can-i-do-in-the-ea-portal"></a>Welche Möglichkeiten bietet mir das EA Portal?
Das [Azure Enterprise Agreement Portal](https://ea.azure.cn/) (EA Portal) ist eine exklusive Leistung im Rahmen eines Onlinedienst-Premiumvertrags (Online Service Premium Agreement, OSPA) für Unternehmen. Mögliche nächste Schritte:
- Hinzufügen und Verwalten von Abteilungen
- Hinzufügen von Konten
- Anzeigen von Nutzung und Kosten sowie Erstellen von Berichten

Hilfe zur Verwendung des EA Portals finden Sie im [Videocenter](https://www.azure.cn/video-center/).

### <a name="what-can-i-do-in-the-account-portal"></a>Welche Möglichkeiten bietet mir das Kontoportal?
Das [Kontoportal](https://account.windowsazure.cn/) wird nach und nach eingestellt. Die darin enthaltenen Funktionen sind jetzt im neuen [Azure-Portal](https://portal.azure.cn/) zu finden, das seit Juni 2017 verfügbar ist. Es wird empfohlen, das Azure-Portal für die Kontoverwaltung zu verwenden, aber für folgende Aktionen können Sie auch weiterhin das Kontoportal verwenden:
- Erstellen, Umbenennen, Verlängern und Kündigen von Abonnements
- Hinzufügen weiterer Abonnements
- Bearbeiten Ihrer Kontaktinformationen
- Anzeigen von Previewfunktionen, die vom Abonnement hinzugefügt werden

### <a name="why-is-my-new-enrollment-status-pending-in-the-ea-portal"></a>Warum steht der Status meiner neuen Registrierung im EA Portal noch aus?
Bei einer neuen Registrierung wird der Status zunächst als „ausstehend“ angezeigt. Sobald Sie sich mit dem Konto beim EA Portal anmelden, wird der Status in „aktiv“ geändert.

## <a name="operations"></a>Vorgänge
### <a name="whats-the-criteria-for-purchasing-a-subscription-to-azure-china-21vianet"></a>Welche Kriterien müssen erfüllt sein, damit ich ein Abonnement für Azure China 21Vianet erwerben kann?
Zur Einhaltung des [Anti-Terror-Gesetzes](https://www.gov.cn/zhengce/2015-12/28/content_5029899.htm) sind Onlinedienstanbieter verpflichtet, Benutzer vor der Erbringung von Dienstleistungen durch Authentifizierung des bürgerlichen Namens zu überprüfen. Wenn Sie sich für ein Azure China 21Vianet-Abonnement online oder über einen Microsoft-Vertriebsmitarbeiter registrieren, müssen Sie Folgendes angeben:
- Lokale chinesische Mobiltelefonnummer
- Chinesischer Personalausweis oder eine in China registrierte Gewerbeerlaubnis 

Wenn Sie ein Enterprise Agreement über OSPA erwerben möchten, sollten Sie vor Beginn des Vertragsverhältnisses eine Kreditprüfung mit 21Vianet durchführen. Sie müssen Ihre Gewerbeerlaubnis mit Informationen zum Stammkapital, der Bescheinigung des Finanzamts, den Finanzberichten der letzten beiden Jahre und anderen Details aktualisieren. Das Microsoft-Kontoteam kann Ihnen hierbei helfen, bevor Sie Ihre Anfrage an 21Vianet senden. Durch diese Anforderung wird sichergestellt, dass Sie den Dienst auch bezahlen können. 

### <a name="can-i-merge-my-azure-china-21vianet-account-into-my-existing-global-azure-account"></a>Kann ich mein Azure China 21Vianet-Konto mit meinem vorhandenen globalen Azure-Konto zusammenführen?
Ihr Azure China 21Vianet-Konto kann nicht mit einem vorhandenen globalen Azure-Konto zusammengeführt werden, da Azure China 21Vianet eine physisch getrennte Instanz von Clouddiensten auf dem chinesischen Festland ist.

### <a name="is-there-guidance-on-planning-the-migration-of-my-applications"></a>Gibt es Anleitungen zur Planung der Migration meiner Anwendungen?
Ja. Zur Ausführung über Azure China 21Vianet:
- Wenn die Anwendung oder die Workload bei einem anderen Cloudanbieter ausgeführt wird, können Sie diese [umgestalten und migrieren](/azure/china/china-how-to-refactor).
- Wenn die Anwendung in einer globalen Azure-Umgebung ausgeführt wird, können Sie diese [rehosten und migrieren](/azure/china/china-how-to-rehost).
