# <a name="welcome-to-azure-china-21vianet"></a>Willkommen bei Azure China 21Vianet
Von 21Vianet (Azure China 21Vianet) betriebenes Microsoft Azure ist eine physisch eigenständige Instanz von Clouddiensten auf dem chinesischen Festland, die von Shanghai Blue Cloud Technology Co., Ltd. („21Vianet“) – einer hundertprozentigen Tochtergesellschaft von Beijing 21Vianet Broadband Data Center Co., Ltd. – autonom betrieben und realisiert wird. 

Die Dienste basieren auf denselben Microsoft Azure-, Microsoft Office 365- und Microsoft Power BI-Technologien wie der weltweite Microsoft-Clouddienst, und sie haben vergleichbare Servicelevels. Vereinbarungen und Verträge für Microsoft Azure in China werden gegebenenfalls zwischen den Kunden und 21Vianet geschlossen.

Als erster ausländischer und gemäß den gesetzlichen Bestimmungen operierender Anbieter eines Public Cloud-Dienstes in China bietet [Azure China 21Vianet](/azure/china/china-overview-operations) erstklassige Sicherheit, wie dies im [Trust Center](https://www.trustcenter.cn/compliance/) beschrieben wird und wie es die chinesischen Bestimmungen für sämtliche Systeme und Anwendungen fordern, die auf seiner Architektur basieren. Azure umfasst die Hauptkomponenten Infrastructure-as-a-Service (IaaS), Platform-as-a-Service (PaaS) und Software-as-a-Service (SaaS). Zu diesen Komponenten gehören Netzwerk-, Speicher-, Datenverwaltungs-, Identitätsverwaltungs- und viele weitere Dienste.

Azure China 21Vianet unterstützt die meisten hervorragenden [Dienste](https://www.azure.cn/home/features/products-by-region) der weltweiten Azure-Umgebung, wie geosynchrone Datenreplikation und automatische Skalierung. Auch wenn Sie bereits globale Azure-Dienste für den Betrieb in China nutzen, müssen Sie möglicherweise [einen neuen Host zuweisen](/azure/china/china-how-to-rehost) oder einige bzw. alle Ihrer Anwendungen oder Dienste [umgestalten](/azure/china/china-how-to-refactor).

- [Anleitung für Entwickler](/azure/china/china-get-started-developer-guide)
- [Hilfe für die Migration](/azure/china/china-how-to-rehost) von Anwendungen und Diensten nach Azure, einschließlich des Global Connection Toolkits
- [Partner](/azure/china/china-resources-partners) mit Azure-Lösungsangeboten

## <a name="operations"></a>Vorgänge
Gemäß der chinesischen [Telekommunikationsverordnung](http://www.miit.gov.cn/n1146295/n1146557/n1146619/c4860613/content.html) (auf Chinesisch) benötigen Anbieter von Clouddiensten (IaaS und PaaS) wertsteigernde Telekommunikationszulassungen. Nur lokal eingetragene Unternehmen mit weniger als 50 Prozent Auslandsinvestitionen können diese Zulassung beantragen. Um dieser Verordnung zu entsprechen, wird der Azure-Dienst in China basierend auf den von Microsoft lizenzierten Technologien von 21Vianet betrieben.
