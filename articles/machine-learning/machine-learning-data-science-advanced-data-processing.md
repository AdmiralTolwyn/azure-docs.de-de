# Leitfaden: Erweiterte Datenverarbeitung in Azure 

Dieses Schema veranschaulicht das Vorbereiten von Daten für ein Azure Machine Learning-Experiment. Es zeigt auf, wie Daten verschoben, verarbeitet und bemustert werden, um sie auf das maschinelle Lernen vorzubereiten. Der hier beschriebene Prozess zur Datenvorbereitung ist ein Schritt im Ablauf unter [Erstellen erweiterter Analyselösungen in Azure](machine-learning-data-science-how-to-create-machine-learning-service.md). Im ADAPT-Handbuch werden die Data Science-Aufgaben beschrieben, die i. d. R. beim Erstellen und Bereitstellen von Vorhersagemodellen mit Azure Machine Learning durchgeführt werden müssen.
