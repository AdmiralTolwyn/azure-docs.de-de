# Machine Learning-Apps im Azure Marketplace

Azure Machine Learning bietet Datenanalysten eine Möglichkeit, ihr Fachwissen und ihre Kreativität gewinnbringend zu nutzen – ebenso, wie dies Entwickler von Smartphone-Apps tun. Durch das Veröffentlichen Ihrer Machine Learning-Dienste im Azure Marketplace können Benutzer Ihre veröffentlichten Cloud-APIs erwerben oder abonnieren, um sie in ihren eigenen Apps zu nutzen.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

Azure Marketplace bietet jedoch weitaus mehr als grundlegende Analyse-APIs. So haben wir beispielsweise einen Satz fertiger End-to-End-Machine Learning-Anwendungen erstellt, um speziellen Geschäftsanforderungen gerecht zu werden. Diese Machine Learning-Apps können genutzt werden, ohne dass besondere Kenntnisse in der Datenanalyse oder im Bereich des maschinellen Lernens erforderlich sind. Die Wissenschaft ist bereits in unseren Lösungen integriert. Benutzer können einfach ihre eigenen Daten eingeben und die Apps verwenden.

Zu diesen Apps gehören beispielsweise:

##Recommendations##
 Eine selbst zu verwaltende, aber extrem leistungsstarke Empfehlungs-Engine, mit der Sie persönliche Produktempfehlungen einfach Ihrer Website oder E-Mail-Aktionen hinzufügen können.

[Link zum Marketplace](http://datamarket.azure.com/dataset/amla/recommendations)

[Kurzanleitung](machine-learning-recommendation-api-quick-start-guide.md)
  
##Häufig zusammen gekauft##
Eine API, über die Ihre Kunden in Ihrem Katalog Elemente finden können, die häufig zusammen gekauft werden. Nutzen Sie das Kundenkaufverhalten, um "Häufig zusammen gekauft"-Empfehlungen zu Ihrer Website hinzuzufügen und die Anzahl der Kaufabschlüsse in Ihrem Onlinestore zu verbessern.

[Link zum Marketplace](https://datamarket.azure.com/dataset/amla/mba)

[Dokumentation](machine-learning-apps-frequently-bought-together-api.md)

##Erkennung von Anomalien##
Eine API, über die anomale Ereignisse in Zeitreihendaten (z. B. Servertelemetriedaten oder sensorbasierte Daten) erkannt werden.

[Link zum Marketplace](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[Dokumentation](machine-learning-apps-anomaly-detection.md)

Mit diesen APIs können Sie leistungsstarke mit Azure Machine Learning erstellte Dienste für das maschinelle Lernen nutzen, ohne über Vorkenntnisse in der Datenanalyse zu verfügen oder viel Zeit in die Entwicklung von Apps zu investieren. Geben Sie einfach Ihre eigenen Daten ein, und verwenden Sie die Apps.

##Textanalyse##
Mit dieser API kann unstrukturierter Text für Aufgaben wie Stimmungsanalyse und Schlüsselwortextraktion analysiert werden. Zur Verwendung dieser API sind keine Trainingsdaten erforderlich. Geben Sie einfach Ihre Textdaten ein. In die API sind moderne Verfahren der natürlichen Sprachverarbeitung integriert.

[Link zum Marketplace](https://datamarket.azure.com/dataset/aml_labs/anomalydetection)

[Dokumentation](machine-learning-apps-text-analytics.md)
