# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>Vorbereiten von Daten mit dem Azure Machine Learning Data Prep SDK für Python
 
Die Vorbereitung der Daten ist ein wichtiger Bestandteil eines Machine Learning-Workflows. Ihre Modelle sind präziser und effizienter, wenn diese über Zugriff auf bereinigte Daten in einem Format verfügen, das einfacher zu verwenden ist. Sie können mit dem [Azure Machine Learning Data Prep SDK](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) Daten in verschiedenen Formaten laden, zur besseren Nutzbarkeit umwandeln und diese Daten in einen Speicherort schreiben, damit diese durch Ihre Modelle aufgerufen werden können. 


## <a name="steps-for-data-preparation-with-azure-machine-learning"></a>Schritte zur Datenvorbereitung mit Azure Machine Learning

2. [Transformieren von Daten](how-to-transform-data.md )
3. [Schreiben von Daten](how-to-write-data.md )
