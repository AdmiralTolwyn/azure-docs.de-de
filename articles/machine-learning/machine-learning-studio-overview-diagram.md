# Übersichtsdiagramm der Azure Machine Learning Studio-Funktionen

Das Diagramm **Übersicht der Microsoft Azure Machine Learning Studio-Funktionen** bietet eine allgemeine Übersicht, wie Machine Learning Studio dazu verwendet werden kann, Predictive Analytics-Modelle zu entwickeln und diese in der Azure-Cloud anzuwenden.

[Azure Machine Learning Studio](https://studio.azureml.net/) stellt viele Machine Learning-Algorithmen zusammen mit Modulen zur Verfügung, die bei der Eingabe, Ausgabe, Vorbereitung und Visualisierung von Daten unterstützen. Durch Verwenden dieser Komponenten können Sie ein Predictive Analytics-Experiment entwickeln, dieses iterativ durchlaufen und dann dazu verwenden, Ihr Modell zu trainieren. Danach können Sie Ihr Modell mit einem Mausklick in der Azure-Cloud zur Anwendung bringen, sodass es zur Bewertung neuer Daten verwendet werden kann.

In diesem Diagramm wird gezeigt, wie alle diese Teile zusammenpassen.

> [AZURE.NOTE]Weitere Unterstützung beim Navigieren durch die und Auswählen der Machine Learning-Algorithmen, die in Machine Learning Studio verfügbar sind, finden Sie unter [Machine Learning – Cheat Sheet für Algorithmen für Microsoft Azure Machine Learning Studio](machine-learning-algorithm-cheat-sheet.md).

## Herunterladen des Machine Learning Studio-Übersichtsdiagramms

Laden Sie das Diagramm **Übersicht der Microsoft Azure Machine Learning Studio-Funktionen** herunter, und verschaffen Sie sich einen allgemeinen Überblick über die Funktionen von Machine Learning Studio. Drucken Sie das Diagramm in Tabloid-Größe (28 x 43 cm, 11 x 17 Zoll) aus, um schnell darauf zugreifen zu können.

**Laden Sie das Diagramm über diesen Link herunter: [Übersicht der Microsoft Azure Machine Learning Studio-Funktionen](http://download.microsoft.com/download/C/4/6/C4606116-522F-428A-BE04-B6D3213E9E52/ml_studio_overview_v1.1.pdf)**
