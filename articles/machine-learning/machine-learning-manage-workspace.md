# Verwalten eines Azure Machine Learning-Arbeitsbereichs 
Die Verwaltung Ihres Azure Machine Learning-Arbeitsbereichs (Azure ML) umfasst Zugriff, Bereitstellung von Produktions-Webdiensten und die Verwaltung Ihrer Azure ML-Webdienste. Außerdem finden Sie in der Dashboard-Registerkarte eine Übersicht über die Nutzung Ihres Arbeitsbereichs und weitere Informationen zum Arbeitsbereich.  

In der Registerkarte **WEBDIENSTE** können Sie Azure ML-Webdienste hinzufügen, ändern oder löschen.  

## Erreichen der Verwaltungsseite für den Arbeitsbereich ##
1.	Melden Sie sich mit Ihrem Azure-Konto an.
2.	Klicken Sie im Microsoft Azure-Dienstbereich auf **MACHINE LEARNING**.
3.	Klicken Sie in der Liste **Machine Learning** auf einen Arbeitsbereich.
