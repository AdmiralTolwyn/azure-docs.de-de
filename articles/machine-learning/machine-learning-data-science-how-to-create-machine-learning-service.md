# Erstellen von erweiterten Analyselösungen in Azure

In diesem ADAPT-Leitfaden (Advanced Analytics Process and Technology) werden die Datenwissenschaftsaufgaben beschrieben, die üblicherweise beim Erstellen und Bereitstellen von Vorhersagemodellen mit Azure Machine Learning durchgeführt werden müssen. Es wird gezeigt, wie die Azure-Plattform Aufgaben ermöglicht, wie das Erfassen von Daten aus verschiedenen Quellen, das Vorbereiten der Daten für die Verwendung in Azure Machine Learning und das Erstellen operationalisierter Modelle mit einem Azure Machine Learning-Experiment, die von Endbenutzeranwendungen (programmgesteuert oder anderweitig) genutzt werden können. Der Plan zeigt die wichtigsten Schritte eines umfassenden datenwissenschaftlichen Ablaufs. Es sind jedoch nicht alle Schritte erforderlich, und auch kann die Reihenfolge der Schritte kann je nach Speicherort, Größe und Komplexität der Daten variieren.

[AZURE.INCLUDE [machine-learning-free-trial](../../includes/machine-learning-free-trial.md)]

Der Schritt mit der Bezeichnung **Erweiterte Datenverarbeitung in Azure** ist ein Link zu einem zweiten Leitfaden, der den Prozess der Vorbereitung von Daten beschreibt, die in einem Azure Machine Learning-Experiment verwendet werden sollen. Fahren Sie nach Lektüre dieses zweiten Leitfadens mit dem vorliegenden Leitfaden fort: vom nächsten Schritt, in dem die verarbeiteten Daten in Azure Machine Learning Studio erfasst werden, bis zum letzten Schritt, in dem das Modell genutzt wird.
