# <a name="datamarket-is-now-deprecated"></a>DataMarket ist jetzt veraltet

[Microsoft DataMarket](https://datamarket.azure.com/datasets) ist jetzt als veraltet markiert, sodass auch die Machine Learning-APIs, die dort gehostet wurden, als veraltet gelten.

Sie finden diese Funktionen (und weitere) jetzt in den [Cognitive Services-APIs](https://www.microsoft.com/cognitive-services).

## <a name="text-analytics-api"></a>Textanalyse-API

Frühere Benutzer der DataMarket-Textanalyse-API sollten die [Cognitive Services-Textanalyse-API](https://www.microsoft.com/cognitive-services/text-analytics-api) verwenden.
Die Cognitive Services-Textanalyse-API unterstützt alle Szenarien, die auch die frühere Version von DataMarket unterstützt hat, sowie zusätzliche Szenarien, wie z.B. Sprachen- und Textgegenstandserkennung.
