# <a name="walkthroughs-executing-the-team-data-science-process"></a>Exemplarische Vorgehensweisen zum Ausführen des Team Data Science-Prozesses

Diese umfassenden **exemplarischen Vorgehensweisen** veranschaulichen jeweils die Schritte im Team Data Science-Prozess für bestimmte Szenarien. Sie zeigen, wie Cloud- und lokale Tools und Dienste in einem Workflow oder einer Pipeline zum Erstellen einer **intelligenten Anwendung** kombiniert werden. Die exemplarischen Vorgehensweisen sind nach verwendeter **Plattform** gruppiert. Das folgende Menü enthält Links zu den entsprechenden Beispielen:

[!INCLUDE [tdsp-walkthroughs-by-platform](../../includes/tdsp-walkthroughs-by-platform.md)]


## <a name="walkthrough-descriptions"></a>Beschreibungen der exemplarischen Vorgehensweisen

Es folgen kurze Beschreibungen der exemplarischen Vorgehensweisen auf der jeweiligen Plattform:

- [Exemplarische Vorgehensweisen zu HDInsight Spark unter Verwendung von PySpark und Scala:](data-science-process-walkthroughs-spark.md) In diesen exemplarischen Vorgehensweisen werden PySpark und Scala in einem Azure Spark-Cluster für Predictive Analytics verwendet. 
- [Exemplarische Vorgehensweisen zu HDInsight Hadoop unter Verwendung von Hive:](data-science-process-walkthroughs-hdinsight-hadoop.md) In diesen exemplarischen Vorgehensweisen wird Hive mit einem HDInsight Hadoop-Cluster für Predictive Analytics verwendet.
- [Exemplarische Vorgehensweisen zu Azure Data Lake unter Verwendung von U-SQL:](data-science-process-walkthroughs-azure-data-lake.md) In diesen exemplarischen Vorgehensweisen wird U-SQL mit Azure Data Lake für Predictive Analytics verwendet.
- [SQL Server:](data-science-process-walkthroughs-sql-server.md) In diesen exemplarischen Vorgehensweisen werden SQL Server, SQL Server R Services und SQL Server Python Services für Predictive Analytics verwendet.
- [SQL Data Warehouse:](data-science-process-walkthroughs-sql-data-warehouse.md) In diesen exemplarischen Vorgehensweisen wird SQL Data Warehouse für Predictive Analytics verwendet.
