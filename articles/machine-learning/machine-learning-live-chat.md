# <a name="get-help-from-machine-learning-live-chat-support"></a>Hilfe vom Machine Learning Live-Chat-Support
[Azure Machine Learning Studio](machine-learning-what-is-ml-studio.md) bietet eine intuitive Oberfläche zum Erstellen von Modellen für das maschinelle Lernen. Ein [Katalog](machine-learning-gallery-how-to-use-contribute-publish.md) mit von der Community eingereichten Beispielen erleichtert Ihren Einstieg. In einem [Forum](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning) können Sie außerdem bei Problemen Fragen stellen. 

Manchmal genügt es, nur eine kurze Frage zu stellen, um eine Hürde überwinden zu können. Suchen Sie in [Machine Learning Studio](http://studio.azureml.net/Home) nach dem Chat-Symbol in der oberen Navigationsleiste.  Wenn dieses Symbol angezeigt wird, bedeutet dies, dass ein Mitglied des Produktteams online ist und Ihnen in Echtzeit helfen kann.

![Azure ML-Chat](./media/machine-learning-live-chat/AzureMLChatNavBar.png)

Geben Sie Ihre Frage ein, und sie wird umgehend beantwortet!

![Azure ML Chat-Dialogfeld](./media/machine-learning-live-chat/AzureMLChat.png)

## <a name="dont-see-the-live-chat-icon"></a>Wird das Live-Chat-Symbol nicht angezeigt?
Der Livechat wird von Mitgliedern des Machine Learning-Teams betreut. Wenn Sie das Live-Chat-Symbol nicht sehen, liegt es daran, dass das Team aktuell nicht erreichbar ist, z.B. weil Sie es außerhalb der Geschäftszeiten versuchen.
