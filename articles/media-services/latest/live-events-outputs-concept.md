# <a name="live-events-and-live-outputs"></a>Liveereignisse und Liveausgaben

Mit Azure Media Services können Sie Ihren Kunden Liveereignisse in der Azure Cloud anbieten. Um Ihre Livestreamingereignisse in Media Services v3 zu konfigurieren, müssen Sie sich über [Liveereignisse](https://docs.microsoft.com/rest/api/media/liveevents) und [Liveausgaben](https://docs.microsoft.com/rest/api/media/liveoutputs) informieren.
