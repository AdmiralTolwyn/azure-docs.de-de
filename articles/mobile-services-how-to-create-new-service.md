# Erstellen eines neuen mobilen Diensts

In diesem Thema erfahren Sie, wie Sie mit dem [Azure-Verwaltungsportal][Azure-Verwaltungsportal] ein neues JavaScript-Back-End in Azure Mobile Services erstellen, das JavaScript für die serverseitige Geschäftslogik verwendet. Informationen zum Erstellen des Back-Ends für den mobilen Dienst sowie einer mobilen Client-App, die den neuen mobilen Dienst nutzt, finden Sie unter [Erste Schritte mit Mobile Services][Erste Schritte mit Mobile Services].

>[WACOM.NOTE]Sie benötigen ein Azure-Konto mit aktivierter "Azure Mobile Services"-Funktion, um dieses Lernprogramm auszuführen. Wenn Sie noch kein Konto haben, können Sie in nur wenigen Minuten ein kostenloses Testkonto erstellen. Einzelheiten finden Sie unter [Kostenlose Azure-Testversion][Kostenlose Azure-Testversion].
