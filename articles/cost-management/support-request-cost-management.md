# <a name="create-a-support-request-for-cost-management"></a>Erstellen einer Supportanfrage für Cost Management

Wenn Sie die gesuchten Informationen nicht finden können, können Sie eine Supportanfrage öffnen. Auch bei einem vermutlichen Problem mit einer Dienstunterbrechung oder einem Dienstfehler können Sie dies tun. Öffnen Sie dabei nur ein Supportticket für ein einzelnes Problem. So kann das gemeldete Problem schnell weitergeleitet werden.

## <a name="open-a-support-ticket"></a>Öffnen eines Supporttickets

1. Melden Sie sich beim Azure-Portal an (https://portal.azure.com).
2. Klicken Sie auf der oberen Navigationsleiste auf **Hilfe**.
3. Klicken Sie im **Hilfe**-Menü auf **Hilfe + Support**.
4. Klicken Sie im Menü „Hilfe + Support“ unter „Support“ auf **Neue Supportanfrage**.
5. Wählen Sie im Bereich „Grundlegende Einstellungen“ unter „Problemtyp“ die Option **Technisch** aus.
6. Wählen Sie unter „Abonnement“ eines der aufgeführten Abonnements aus. Das Abonnement, das Sie auswählen, wird nicht zur Weiterleitung des Problems verwendet.
7. Wählen Sie unter „Dienst“ **Alle Dienste** und dann **Cloudyn** aus der Liste aus.
8. Wählen Sie unter „Supportplan“ Ihren Azure-Supportplan aus, und klicken Sie dann auf **Weiter**.  
    ![Neue Supportanfrage – Bereich „Grundlegende Einstellungen“](./media/support-request-cost-management/support-request01.png)
9. Wählen Sie im Bereich „Problem“ einen **Schweregrad** für die Bestimmung der Antwortzeit aus.
10. Wählen Sie einen **Problemtyp** und dann eine **Kategorie** aus.
11. Geben Sie in das Feld **Titel** einen Titel ein, der Ihre Anfrage beschreibt.
12. Geben Sie in das Feld **Details** zusätzliche Informationen ein.
13. Wählen Sie für **Zeitpunkt, an dem das Problem auftrat** ein ungefähres Datum und Uhrzeit aus, und klicken Sie dann auf **Weiter**.  
    ![Neue Supportanfrage – Bereich „Problem“](./media/support-request-cost-management/support-request02.png)
14. Wählen Sie im Bereich **Kontaktinformationen** Ihre bevorzugte Kontaktmethode aus. Geben Sie dann Ihre Kontaktinformationen ein, und klicken Sie auf **Erstellen**.  
    ![Neue Supportanfrage – Bereich „Kontakt“](./media/support-request-cost-management/support-request03.png)

Wenn das Supportticket erstellt wurde, wird es zur Supportwarteschlange hinzugefügt. Die Antwortzeit variiert je nach Supportplan und Schweregrad (Auswirkung auf das Unternehmen) des Problems. Weitere Informationen finden Sie unter [Azure-Support: Umfang und Reaktionszeiten](https://azure.microsoft.com/support/plans/response/).

Den Status eines von Ihnen eingereichten Vorfalls können Sie unter [Alle Supportanfragen](../azure-supportability/how-to-create-azure-support-request.md#all-support-requests) überprüfen.
