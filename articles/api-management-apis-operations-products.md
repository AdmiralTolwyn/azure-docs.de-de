# Erstellen von APIs, Operationen und Produkten in der Azure API-Verwaltung

In der Azure API-Verwaltung werden APIs und deren Operationen zu Produkten hinzugefügt und können dort von Entwicklern verwendet werden, um Anwendungen unter Verwendung der APIs zu erstellen. Die Anleitungen in diesem Bereich beschreiben, wie Sie eine API erstellen, Operationen zur API hinzufügen, die API zu einem Produkt zuordnen und das Produkt für Entwickler veröffentlichen können.

## <a name="create-apis"> </a>Erstellen von APIs

Eine API in der API-Verwaltung besteht aus einem Satz von Operationen, die von Clientanwendungen aufgerufen werden können. Neue APIs werden im API-Verwaltungsportal erstellt.

Diese Anleitung beschreibt die Erstellung und Konfiguration einer neuen API in der API-Verwaltung.

-   [Erstellen von APIs][Erstellen von APIs]

## <a name="add-operations"> </a>Hinzufügen von Operationen zu einer API

Bevor eine API in der API-Verwaltung verwendet werden kann, müssen Operationen hinzugefügt werden. Diese Anleitung beschreibt, wie Sie unterschiedliche Typen von Operationen zu einer API in der API-Verwaltung hinzufügen können.

-   [Hinzufügen von Operationen zu einer API][Hinzufügen von Operationen zu einer API]

Eine API und deren Operationen können auch in einem Schritt importiert werden, entweder im WADL- oder im Swagger-Format.

-   [Importieren einer API-Definition mit Operationen][Importieren einer API-Definition mit Operationen]

## <a name="add-product"> </a>Erstellen und Veröffentlichen eines Produkts

Produkte in der API-Verwaltung enthalten eine oder mehrere APIs sowie ein Nutzungskontingent und Nutzungsbedingungen. Nachdem ein Produkt veröffentlicht wurde, können Entwickler das Produkt abonnieren und die APIs des Produkts verwenden. Diese Themen enthalten eine Anleitung zum Erstellen von Produkten, Hinzufügen von APIs und zum Veröffentlichen des Produkts für Entwickler.

-   [Hinzufügen und Veröffentlichen eines Produkts][Hinzufügen und Veröffentlichen eines Produkts]
-   [Erstellen und Konfigurieren erweiterter Produkteinstellungen][Erstellen und Konfigurieren erweiterter Produkteinstellungen]
