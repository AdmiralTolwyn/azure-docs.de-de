# <a name="upgrade-your-free-trial-or-microsoft-imagine-azure-subscription-to-pay-as-you-go"></a>Aktualisieren Ihrer kostenlosen Testversion oder Ihres Microsoft Imagine Azure-Abonnements auf nutzungsbasierte Bezahlung

Upgraden Sie Ihre [kostenlose Testversion](https://azure.microsoft.com/free/) oder Ihr [Microsoft Imagine](https://azure.microsoft.com/offers/ms-azr-0144p/)-Abonnement im Azure-Kontocenter auf [Nutzungsbasierte Zahlung](https://azure.microsoft.com/offers/ms-azr-0003p/).

1. Melden Sie sich beim [Azure-Kontocenter](https://account.windowsazure.com/subscriptions) an.
2. Wählen Sie im Bereich „Abonnementstatus“ das Banner **Jetzt aktualisieren**.
   
    ![Screenshot der zeigt, wo man klicken muss, um seine kostenlosen Testversion auf nutzungsbasierte Bezahlung zu aktualisieren](./media/billing-upgrade-azure-subscription/billpage.png)
3. Bestätigen Sie das Upgrade.
   
    ![Screenshot der zeigt, was Sie auswählen müssen, um zu bestätigen, dass Sie ein Upgrade Ihres Abonnements durchführen möchten](./media/billing-upgrade-azure-subscription/Upgrade.png)

 Wenn Sie ein Upgrade von einem kostenlosen Testabonnement durchführen, behalten Sie Ihr Restguthaben für die vollen 30 Tage nach Erstellung des Abonnements. Sie haben auch Zugang zu kostenlosen Diensten für 12 Monate.
