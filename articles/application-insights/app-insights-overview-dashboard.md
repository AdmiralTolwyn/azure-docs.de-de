# <a name="application-insights-overview-dashboard-preview"></a>Application Insights-Übersichtsdashboard (Vorschauversion)

Application Insights hat schon immer einen Übersichtsbereich mit einer Zusammenfassung enthalten, um Ihnen eine schnelle Bewertung der Integrität und Leistung Ihrer Anwendung auf einen Blick zu ermöglichen. Die neue Vorschauversion des Übersichtsdashboards ist jetzt noch schneller und flexibler.

## <a name="how-do-i-test-out-the-new-experience"></a>Wie kann ich die neue Oberfläche testen?

 Wählen Sie in Application Insights unter _Untersuchen_ die Option _Übersicht (Vorschau)_.

![Übersicht (Vorschau)](.\media\app-insights-overview-dashboard\01.png)

Hiermit wird das neue Standard-Übersichtsdashboard gestartet:

![Vorschauversion des Übersichtsbereichs](.\media\app-insights-overview-dashboard\02.png)

## <a name="better-performance"></a>Bessere Leistung

Die Auswahl des Zeitbereichs wurde zu einer 1-Klick-Oberfläche vereinfacht.

![Zeitbereich](.\media\app-insights-overview-dashboard\04.png)

Die Gesamtleistung wurde deutlich erhöht. Jede KPI-Kachel, die standardmäßig dynamisch aktualisiert wird, ist mit dem entsprechenden Application Insights-Feature verknüpft. Beim Auswählen von fehlgeschlagenen Anforderungen wird der Bereich _Fehler_ gestartet:

![Fehler](.\media\app-insights-overview-dashboard\03.png)

## <a name="application-dashboard"></a>Anwendungsdashboard

Im Anwendungsdashboard wird die vorhandene Dashboardtechnologie in Azure genutzt, um eine vollständig anpassbare einzelne Bereichsansicht Ihrer Anwendungsintegrität und -leistung bereitzustellen.

Wählen Sie oben links die Option _Anwendungsdashboard_, um auf das Standarddashboard zuzugreifen.

![Dashboardansicht](.\media\app-insights-overview-dashboard\0009.png)

Wenn Sie zum ersten Mal auf das Dashboard zugreifen, wird eine Standardansicht gestartet:

![Dashboardansicht](.\media\app-insights-overview-dashboard\06.png)

Sie können die Standardansicht beibehalten, wenn sie Ihnen gefällt. Sie können dem Dashboard aber auch Elemente hinzufügen oder Elemente löschen, um die Anforderungen Ihres Teams bestmöglich zu erfüllen.

Wählen Sie Folgendes, um zurück zur Übersichtsoberfläche zu navigieren:

![Schaltfläche „Übersicht“](.\media\app-insights-overview-dashboard\07.png)
