# <a name="preview-upcoming-changes-to-azure-application-insights"></a>Anzeigen einer Vorschau bevorstehender Änderungen für Azure Application Insights 

Für Application Insights werden immer wieder neue Features veröffentlicht. Wenn Sie sich eine Vorschau dieser Verbesserungen ansehen möchten, können Sie sich auf dem Blatt „Application Insights – Vorschau“ registrieren.  Das Entwicklungsteam stellt neue Features vor der Einführung für alle Benutzer zunächst im begrenzten Rahmen einer Vorschau zur Verfügung. 

Die folgende Abbildung veranschaulicht das Festlegen der Vorschaueinstellungen.

![Blatt „Application Insights – Vorschau“](./media/app-insights-preview/preview.png)

## <a name="set-preferences"></a>Festlegen der Einstellungen

Auf dem Vorschaublatt stehen folgende Optionen für die Vorschauanzeige zur Verfügung:

- Immer: Vorschauen werden angezeigt, sobald sie verfügbar sind.
- Automatisch: Es werden Vorschauen angezeigt, die Microsoft für Ihr Konto empfiehlt. 
- Nie: Es werden nur Vorschauen angezeigt, die Sie auswählen.
