# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Leitfaden für Azure Data Lake Storage Gen1-Kunden

Die Vorschauversion von Azure Data Lake Storage Gen2 verfügt über native Blobspeicherfunktionalität, z.B. Data Tiering, Lebenszyklusverwaltung und verbesserte Netzwerksicherheit – zusätzlich zum Hadoop-kompatiblen Dateisystem, das unter Data Lake Storage Gen1 verfügbar ist.

Wenn Sie Data Lake Storage Gen1-Kunde sind, empfiehlt Microsoft Ihnen, alle neuen und verbesserten Funktionen von Data Lake Storage Gen2 zu evaluieren und Feedback zu senden, indem Sie sich [am Testen der Vorschauversion beteiligen](https://aka.ms/adlsgen2signup). Data Lake Storage Gen1 wird ebenfalls weiter unterstützt. Data Lake Storage Gen1 und Gen2 können parallel genutzt werden. Ihre vorhandenen Ressourcen und Anwendungen, für die Data Lake Storage Gen1 genutzt wird, funktionieren weiterhin ohne Updates. Sie können einen Plan für die Migration zu Data Lake Storage Gen2 nach Ihren Wünschen entwickeln.

Wenn Sie derzeit Data Lake Storage Gen1 evaluieren, empfiehlt Microsoft Ihnen, am Test der Gen2-Vorschauversion teilzunehmen und die neuen und verbesserten Funktionen von Data Lake Storage Gen2 zu evaluieren. Falls Sie Blob Storage-Kunde sind, empfiehlt Microsoft Ihnen, eine Evaluierung durchzuführen, ob ein Hadoop-kompatibles Dateisystem für Ihre Analyseanforderungen geeignet ist.
