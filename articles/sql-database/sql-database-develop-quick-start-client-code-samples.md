# Codebeispiele für die Cliententwicklung und erste Schritte mit SQL-Datenbanken


Dieses Thema enthält Links zu Schnellstart-Codebeispielen, die Sie zum Herstellen von Verbindungen mit Azure SQL-Datenbank verwenden können. Die Beispiele decken Folgendes ab:


- Eine Vielzahl von Programmiersprachen.
- Windows und Linux als Betriebssystem, unter dem das Clientprogramm ausgeführt werden soll.
- Kurze Schnellstart-Codebeispiele sowie längere Beispiele, die vorübergehende Fehler mit automatisierter Wiederholungslogik behandeln.
- Codebeispiele, die relationale Resultsets in ein objektorientiertes Format konvertieren.


> [AZURE.NOTE]Mit Stand vom 19. Mai 2015 werden Codebeispiele für weitere Sprachen vorbereitet. Links zu diesen Beispielen werden in diesem Thema hinzugefügt.
