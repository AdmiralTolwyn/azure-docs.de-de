# Übersicht über die SQL-Datenbank

SQL-Datenbank ist ein relationaler Datenbankdienst in der Cloud, der auf der Microsoft SQL Server Engine basiert und integrierte, betriebsnotwendige Funktionen beinhaltet. SQL-Datenbank bietet vorhersagbare Leistung, Skalierbarkeit, Geschäftskontinuität, Datenschutz und nahezu keinerlei Verwaltungsaufwand für Cloud-Entwickler und Architekten für Cloud-Lösungen. Darüber hinaus unterstützt die SQL-Datenbank vorhandene SQL Server-Tools, -Bibliotheken und APIs, sodass diese problemlos in die Cloud verschoben werden können. Sie können eine SQL-Datenbank [innerhalb von Sekunden zur Ausführung bringen](sql-database-get-started.md), ohne virtuelle Maschinen oder Infrastruktur verwalten zu müssen. Die programmatische DBA-ähnliche Funktionalität ermöglicht effiziente DevOps, und aufgrund der Selbstverwaltung fällt nahezu kein Wartungsaufwand für die Datenbanken an. Dadurch können Sie sich auf die Entwicklung und Verwaltung Ihrer Anwendung konzentrieren.

SQL-Datenbank bietet Ihnen die Möglichkeit, einzelne Datenbanken ohne Ausfallzeiten und mit vorhersagbarer Leistung, die nach Bedarf einstellbar ist, zu [skalieren](sql-database-service-tiers.md). Sie können jedoch auch [elastische Datenbanken](sql-database-elastic-pool.md) verwenden, mit denen Sie eine Skalierung auf mehrere tausend Datenbanken vornehmen, gleichzeitig die Kontrolle über Ihr Budget behalten und die leistungsfähigen Verwaltungsfunktionen der [Aufträge für die elastische Datenbank](sql-database-elastic-jobs-overview.md) und die programmgesteuerten Funktionen der [Tools für elastische Datenbanken](sql-database-elastic-scale-get-started.md) nutzen können.

Die in der Branche führende Verfügbarkeit mit einer [SLA](http://azure.microsoft.com/support/legal/sla/) von 99,99 % stellt sicher, dass es für Ihre Anwendungen nahezu keine Ausfallzeiten gibt. Der integrierte Datenschutz durch Überwachung, Wiederherstellung und geografische Replikation stellt sicher, dass Ihre Daten geschützt sind. Die SQL-Datenbank bietet Ihnen die Möglichkeit, Ihre Daten für jeden Zeitpunkt, aus jeder Transaktion, für bis zu 35 Tage wiederherzustellen. Sie können Ihre Daten außerdem in eine Azure-Region Ihrer Wahl replizieren und ohne großen Aufwand eine auf Ihre geschäftlichen Anforderungen zugeschnittene Richtlinie zur geografischen Notfallwiederherstellung implementieren. Die [neueste Version der SQL-Datenbank](sql-database-preview-whats-new.md), V12, bietet eine nahezu vollständige Kompatibilität mit der SQL Server Engine.

Für die SQL-Datenbank sind die [Dienstebenen](sql-database-service-tiers.md) Basic, Standard und Premium verfügbar. Sie unterstützen geringe bis hohe Arbeitsauslastungen, sodass Sie zwischen den verschiedenen Ebenen wechseln oder diese mischen können, um innovative Anwendungsdesigns zu erhalten. Mit der Leistungsstärke und Reichweite von Azure können Sie Azure-Dienste mit der SQL-Datenbank kombinieren, um Ihre individuellen Anforderungen an modernes App-Design zu erfüllen, Kosten- und Ressourceneffizienz zu fördern und neue Geschäftsmöglichkeiten zu erschließen.

Beginnen Sie mit einer [kostenlosen Testversion](http://azure.microsoft.com/pricing/free-trial/), und [erstellen Sie Ihre erste SQL-Datenbank](sql-database-get-started.md) in nur wenigen Minuten.
