# <a name="view-and-update-an-azure-sql-database-server-and-its-settings"></a>Anzeigen und Aktualisieren eines Azure SQL-Datenbank-Servers und seiner Einstellungen

Sie können einen Azure SQL-Datenbank-Server und dessen Einstellungen über das [Azure-Portal]((sql-database-manage-portal.md), mithilfe von [PowerShell](sql-database-manage-powershell.md) sowie per REST-API oder C# anzeigen und aktualisieren. 

## <a name="view-and-update-an-azure-sql-server-settings-using-the-azure-portal"></a>Anzeigen und Aktualisieren der Einstellungen eines Azure SQL-Servers über das Azure-Portal
1. Öffnen Sie im [Azure-Portal](https://portal.azure.com/) das Blatt **SQL-Server**. 

    ![Neuer SQL-Server](./media/sql-database-get-started/new-sql-server.png)

2. Klicken Sie auf den gewünschten Server und anschließend auf dem Blatt des SQL-Servers auf die gewünschte Einstellung. 

    ![Blatt „SQL Server“](./media/sql-database-get-started/sql-server-blade.png)

> [!TIP]
> Ein Einführungstutorial für das Azure-Portal finden Sie unter [Erste Schritte mit Azure SQL-Datenbankservern, -Servern, -Datenbanken und -Firewallregeln mit dem Azure-Portal und SQL Server Management Studio](sql-database-get-started.md).
>
