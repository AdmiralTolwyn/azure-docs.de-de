# Versionshinweise

## 3.0.0 (17.02.2015)

-   Erste Version von Azure Mobile Engagement
-   Die "appId"-Konfiguration wird durch eine Verbindungszeichenfolgen-Konfiguration ersetzt.
-   Eine API wurde entfernt, um beliebige XMPP-Nachrichten von beliebigen XMPP-Entitäten zu senden und zu empfangen.
-   Eine API wurde entfernt, um Nachrichten zwischen Geräten zu senden und zu empfangen.
-   Die Sicherheit wurde erhöht.
-   Google Play und die SmartAd-Nachverfolgung wurden entfernt.
