# <a name="azure-stack-training-and-certification"></a>Azure Stack-Training und -Zertifizierung

*Gilt für: Integrierte Azure Stack-Systeme*

Die folgenden Trainings- und Zertifizierungsangebote sollen Ihnen helfen, Kenntnisse über Azure Stack zu gewinnen und Azure Stack professionell einzusetzen.

## <a name="training"></a>Training

- [Course 20537A: Configuring and Operating a Hybrid Cloud with Microsoft Azure Stack](https://www.microsoft.com/en-us/learning/course.aspx?cid=20537) (Kurs 20537A: Konfigurieren und Betreiben einer Hybrid Cloud mit Microsoft Azure Stack)
- [Configuring and Operating Microsoft Azure Stack](https://aka.ms/AzureStackMOOC) (Konfigurieren und Betreiben von Microsoft Azure Stack)
- [Erwerben Sie berufliche Kenntnisse, die Sie sofort in die Praxis umsetzen können. Azure-Lernpfade auswählen](https://azure.microsoft.com/training/learning-paths/)

## <a name="certification"></a>Zertifizierung
Azure Stack-Zertifizierungen werden derzeit entwickelt. Schauen Sie von Zeit zu Zeit nach den neuesten Updates.
