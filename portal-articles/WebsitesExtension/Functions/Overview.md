# <a name="azure-functions-documentation"></a>Dokumentation zu Azure-Funktionen

Azure-Funktionen ist eine serverlose-Serverdiensts, mit dem Sie Code bedarfsgesteuert ausführen, ohne explizit bereitstellen oder Infrastruktur verwalten zu müssen. Verwenden Sie Azure-Funktionen, um ein Skript oder einen Codeabschnitt in Reaktion auf eine Vielzahl von Ereignissen auszuführen. Informationen Sie zum Azure-Funktionen mit unserem Schnellstarts, Lernprogramme und Beispiele zu verwenden.
