> [!TIP]
> Der Inhalt in diesem Artikel gilt für den ursprünglichen Azure-Tabellenspeicher. Es gibt jedoch jetzt ein Premium-Angebot für Tabellenspeicher: die Azure Cosmos DB-Tabellen-API. Diese API bietet durchsatzoptimierte Tabellen, globale Verteilung und automatische sekundäre Indizes. Es gibt einige [Featureunterschiede zwischen der Tabellen-API in Azure Cosmos DB und Azure-Tabellenspeicher](../articles/cosmos-db/faq.md#where-is-table-api-not-identical-with-azure-table-storage-behavior). Unter [Einführung in die Tabellen-API von Azure Cosmos DB](https://aka.ms/premiumtables) erhalten Sie weitere Informationen und erfahren, wie Sie das Premiumangebot ausprobieren können.
