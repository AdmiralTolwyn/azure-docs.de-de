### <a name="c-class-library"></a>C#-Klassenbibliothek

Für die C#-Klassenbibliotheksentwicklung wird die Unterstützung für die Bindung durch [Installieren des Pakets](../articles/azure-functions/functions-triggers-bindings.md#local-c-development-using-visual-studio-or-vs-code) hinzugefügt.

### <a name="binding-extensions-in-functions-2x"></a>Bindungserweiterungen in Functions 2.x

Für die lokale Entwicklung in Azure Functions version2.x wird das Paket zum [Registrieren der Bindungserweiterung](../articles/azure-functions/functions-triggers-bindings.md#local-development-azure-functions-core-tools) verwendet.
