Für die lokale Entwicklung in Azure Functions version2.x wird das Paket zum [Registrieren der Bindungserweiterung](../articles/azure-functions/functions-triggers-bindings.md#local-development-azure-functions-core-tools) verwendet. Für die C#-Klassenbibliotheksentwicklung in Functions 1.x und 2.x wird die Unterstützung für die Bindung durch [Installieren des Pakets](../articles/azure-functions/functions-triggers-bindings.md#local-c-development-using-visual-studio-or-vs-code) hinzugefügt.

Beachten Sie, dass der Quellcode der WebJobs-SDK-Versionsnummerierung folgt: WebJobs-SDK-Version 2.x entspricht Azure Functions 1.x, d.h. Functions 1.x-Code befindet sich im v2.x-Branch des Repositorys. Master- und Entwicklungsbranches enthalten Code für Web Jobs 3.x, was mit Functions 2.x identisch ist.
