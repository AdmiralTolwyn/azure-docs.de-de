Zum Erstellen eines Premium-Caches melden Sie sich beim [Azure-Portal](https://portal.azure.com) an, und klicken Sie auf **Neu** > **Daten und Speicher** > **Redis Cache**.

![Cache erstellen](media/redis-cache-premium-create/redis-cache-new-cache-menu.png)

> [!NOTE]
> Sie können Caches nicht nur über das Azure-Portal, sondern auch mithilfe von Resource Manager-Vorlagen, PowerShell oder der Azure-Befehlszeilenschnittstelle erstellen. Weitere Informationen finden Sie unter [Erstellen eines Caches](../articles/redis-cache/cache-dotnet-how-to-use-azure-redis-cache.md#create-a-cache).
