Die Daten stammen aus Wikipedia (<a href="http://www.wikipedia.org/">http://www.wikipedia.org/</a>) und basieren auf Artikeln der einzelnen S&P-500-Unternehmen, die als XML-Daten gespeichert wurden.<p>Before uploading to Azure ML Studio, the dataset was processed as follows:<ul><li>Extrahieren der Textinhalte für die einzelnen Unternehmen</li><li>Entfernen der Wiki-Formatierung</li><li>Entfernen aller nicht alphanumerischer Zeichen</li><li>Konvertieren sämtlicher Texte in Kleinbuchstaben</li><li>Bekannte Firmenkategorien wurden hinzugefügt.</li></ul><p>Beachten Sie, dass für einige Unternehmen keine Artikel gefunden werden konnten, daher ist die Anzahl der Datensätze kleiner als 500.
