Sie benötigen ein Azure-Abonnement, um auf Azure Storage zuzugreifen. Wenn Sie noch kein Abonnement haben, können Sie ein [kostenloses Konto](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) erstellen, bevor Sie beginnen.

Der gesamte Zugriff auf Azure Storage erfolgt über ein Speicherkonto. Wenn Sie noch kein Speicherkonto haben, können Sie im [Azure-Portal](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM) ganz einfach eines erstellen. Sie können ein Speicherkonto auch mit Azure PowerShell oder der Azure-Befehlszeilenschnittstelle (Azure CLI) erstellen. Hilfe zur Erstellung des Kontos finden Sie unter [Erstellen eines Speicherkontos](../articles/storage/common/storage-quickstart-create-account.md).
