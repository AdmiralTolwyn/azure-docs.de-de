#### <a name="to-cable-for-power"></a>Um für die Stromversorgung zu Verkabeln.
1. Stellen Sie sicher, dass sich die Netzschalter an beiden Power und Kühleinheiten (PCMs) in der Position.
2. Schließen Sie die Netzkabel an beiden PCMs im primären Gehäuse ein.
3. Fügen Sie die Netzkabel an die Stromverteilereinheiten (PDUs) für das Gestell, wie in der folgenden Abbildung dargestellt. Stellen Sie sicher, dass die Verwendung der beiden PCMs separate Stromquellen.
   
   > [!IMPORTANT]
   > Um hohe Verfügbarkeit für Ihr System sicherzustellen, wird empfohlen, dass Sie unbedingt das in der folgenden Abbildung gezeigte stromverkabelungsschema einhalten. 
   > 
   > 
   
    ![Verkabelung des 2U-Geräts für die Stromversorgung](./media/storsimple-cable-8100-for-power/HCSCableYour2UDeviceforPower.png)
   
    **Stromverkabelung auf einem Gerät 8100**
   
   | Label | Beschreibung |
   |:--- |:--- |
   | 1 |PCM 0 |
   | 2 |Controller 1 |
   | 3 |Controller 0 |
   | 4 |PCM 1 |
   | 5 |PDUs |
4. Um das System zu aktivieren, bringen Sie die Netzschalter an beiden PCMs in die Position ein.
