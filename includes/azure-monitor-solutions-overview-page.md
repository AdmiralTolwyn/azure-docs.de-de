Die von dieser Überwachungslösung gesammelten Daten sind auf der Azure Monitor-Seite **Übersicht** im Azure-Portal verfügbar. Sie können diese Seite im Menü **Azure Monitor** öffnen, indem Sie im Abschnitt **Insights** auf **Mehr** klicken. Jede Lösung wird in einer Kachel dargestellt. Klicken Sie auf eine Kachel, um detaillierte von der jeweiligen Lösung gesammelte Daten anzuzeigen.
