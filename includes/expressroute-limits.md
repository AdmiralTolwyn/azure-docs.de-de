#### <a name="expressroute-limits"></a>ExpressRoute-Grenzwerte
Die folgenden Beschränkungen gelten für ExpressRoute-Ressourcen pro Abonnement.

| Ressource | Standardlimit |
| --- | --- |
| ExpressRoute-Verbindungen pro Abonnement |10 |
| ExpressRoute-Verbindungen pro Region pro Abonnement für ARM |10 |
| Maximale Anzahl von Routen für Azure privates peering mit ExpressRoute-standard |4,000 |
| Maximale Anzahl von Routen für Azure privates peering mit ExpressRoute Premium-Add-On |10,000 |
| Maximale Anzahl von Routen für Öffentliche Azure-peering mit ExpressRoute-standard |200 |
| Maximale Anzahl von Routen für Öffentliche Azure-peering mit ExpressRoute Premium-Add-On |200 |
| Maximale Anzahl von Routen für Microsoft Azure-peering mit ExpressRoute-standard |200 |
| Maximale Anzahl von Routen für Azure-Microsoft-peering mit ExpressRoute Premium-Add-On |200 |
| Anzahl der virtuellen Netzwerkverbindungen pro ExpressRoute-Verbindung zulässig |Siehe Tabelle unten |
