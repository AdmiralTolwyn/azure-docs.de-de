- Für den Zugriff auf den Data Box Edge-Dienst muss Ihr Unternehmen über ein Enterprise Agreement oder ein Cloud Solution Provider-Abonnement verfügen. Weitere Informationen finden Sie unter [Sign up for Microsoft Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/) (Registrieren für Microsoft Azure).
- Da Ihr Verwaltungsdienst in Azure gehostet wird, ist er durch die Azure-Sicherheitsfeatures geschützt. Weitere Informationen zu den Sicherheitsfeatures von Microsoft Azure finden Sie im [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Der Verschlüsselungsschlüssel für SDK-Verwaltungsvorgänge steht für Ihre Ressource unter **Geräteeigenschaften** zur Verfügung. Sie können den Verschlüsselungsschlüssel nur anzeigen, wenn Sie über Berechtigungen für die Resource Graph-API verfügen.
