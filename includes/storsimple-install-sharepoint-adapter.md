#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>Installieren des StorSimple-Adapters für SharePoint
1. Kopieren Sie den Installer an den Webserver-Front-End (WFE), die auch für die Ausführung der SharePoint-Zentraladministration-Webanwendung konfiguriert ist. 
2. Verwenden Sie ein Konto mit Administratorrechten auf dem WFE-Server anmelden.
3. Doppelklicken Sie auf das Installationsprogramm. StorSimple-Adapters für SharePoint-Setup-Assistent wird gestartet. Klicken Sie auf **Weiter** um die Installation zu beginnen.
   
    ![Setup-Startseite für StorSimple-Adapters](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. In den StorSimple-Adapter für SharePoint-Setup-Konfigurationsseite, wählen Sie das Installationsverzeichnis, geben Sie die IP-Adresse für die DATA 0-Netzwerkschnittstelle auf Ihrem StorSimple-Gerät und klicken Sie dann auf **Weiter**. 
   
    ![Setupkonfigurationsseite des StorSimple-adapter](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Klicken Sie auf der setupbestätigungsseite auf **installieren**.
   
    ![StorSimple-Adapter Bestätigungsseite zur Einrichtung](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Klicken Sie auf **Fertig stellen** um den Setup-Assistenten zu schließen.
   
    ![Setup fertigstellungsseite StorSimple-Adapters](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. Öffnen Sie die Seite SharePoint-Zentraladministration. Eine Gruppe von StorSimple-Konfiguration, die den StorSimple-Adapter für SharePoint-Links enthält, sollte angezeigt werden.
8. Wechseln Sie mit dem nächsten Schritt fort: [RBS konfigurieren](#configure-rbs).
