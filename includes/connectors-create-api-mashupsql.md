### Voraussetzungen
* Zugriff auf eine SQL Server-Datenbank

Bevor Sie Mashup SQL in einer Logik-App verwenden können, müssen Sie die Logik-App für Verbindungen mit Ihrem Mashup SQL-Konto autorisieren. Glücklicherweise können Sie dies ganz einfach in Ihrer Logik-App im Azure-Portal durchführen.

Hier finden Sie die Schritte zum Autorisieren Ihrer Logik-App für Verbindungen mit Ihrem Mashup SQL-Konto:

1. Um eine Verbindung mit Mashup SQL herzustellen, wählen Sie im Logik-App-Designer in der Dropdownliste **Verwaltete Microsoft-APIs anzeigen** aus, und geben Sie dann *Mashup SQL* in das Suchfeld ein. Wählen Sie den zu verwendenden Trigger oder die Aktion aus: ![Mashup SQL: Schritt 1](./media/connectors-create-api-mashupsql/mashupsql-1.png)
2. Wenn Sie noch keine Verbindungen mit Mashup SQL hergestellt haben, werden Sie aufgefordert, Ihre Mashup SQL-Anmeldeinformationen anzugeben. Mithilfe dieser Anmeldeinformationen wird Ihre Logik-App autorisiert, die eine Verbindung herstellen und auf Ihr Mashup SQL-Konto zugreifen soll: ![Mashup SQL: Schritt 2](./media/connectors-create-api-mashupsql/mashupsql-2.png)
3. Beachten Sie, dass die Verbindung erstellt wurde und Sie nun mit anderen Schritten in Ihrer Logik-App fortfahren können: ![Mashup SQL: Schritt 3](./media/connectors-create-api-mashupsql/mashupsql-3.png)
