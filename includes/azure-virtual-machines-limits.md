| --- | --- | --- |
| [Virtuelle Maschinen](../articles/virtual-machines/virtual-machines-linux-about.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json) pro Cloud-Dienst<sup>1</sup> |50 |50 |
| Eingabeendpunkte pro Cloud-Dienst<sup>2</sup> |150 |150 |

<sup>1</sup>in Service Management (anstelle von Ressourcen-Manager) erstellte virtuelle Computer werden automatisch in einem Clouddienst gespeichert. Sie können dieses Cloud-Diensts für den Lastenausgleich und die Verfügbarkeit mehrerer virtueller Computer hinzufügen. Finden Sie unter [zum Verbinden virtueller Computer mit einem virtuellen Netzwerk oder Cloud-Dienst](../articles/virtual-machines/linux/classic/connect-vms.md?toc=%2fazure%2fvirtual-machines%2flinux%2fclassic%2ftoc.json).

<sup>2</sup>Eingabe-Endpunkte ermöglichen die Kommunikation mit einem virtuellen Computer von außerhalb des virtuellen Computers Cloud-Dienst. Virtuelle Computer im selben cloud-Dienst oder virtuelles Netzwerk automatisch miteinander kommunizieren kann. Finden Sie unter [Einrichten von Endpunkten für einen virtuellen Computer](../articles/virtual-machines/windows/classic/setup-endpoints.md?toc=%2fazure%2fvirtual-machines%2fwindows%2fclassic%2ftoc.json).
