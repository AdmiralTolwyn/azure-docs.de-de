* **Azure Notebooks** – kostenlose, auf Jupyter basierende Notebooks in der Azure-Cloud.
    1. [![Azure Notebooks](https://notebooks.azure.com/launch.png)](https://aka.ms/aml-clone-azure-notebooks)
    [Importieren von Beispielnotebooks in Azure Notebooks](https://aka.ms/aml-clone-azure-notebooks).
    1.  Erstellen Sie einen Arbeitsbereich und seine Konfigurationsdatei (**config.json**) anhand der Anweisungen in [Konfigurieren einer Entwicklungsumgebung für den Azure Machine Learning Service](https://aka.ms/aml-how-to-configure-environment).
    1. Wählen Sie in der Azure Notebook-Symbolleiste `+New` aus, um Ihre Datei **config.json** demselben Ordner wie das Notebook hinzuzufügen.
    1. Öffnen Sie das Notebook.

* **Ihr eigener Jupyter Notebook-Server**

    Klonen Sie [das GitHub-Repository](https://aka.ms/aml-notebooks), und befolgen Sie die Anweisungen der **Infodatei**, um Software für Ihren Server zu installieren.
