
1. Melden Sie sich beim [Azure-Verwaltungsportal] an. Klicken Sie auf **Mobile Services** und dann auf Ihre App.

	![](./media/mobile-services-create-new-push-table/mobile-services-selection.png)

2. Klicken Sie auf die Registerkarte **Daten** und anschließend auf **Erstellen**.

	![](./media/mobile-services-create-new-push-table/mobile-create-table.png)

	Das Dialogfeld **Neue Tabelle erstellen** wird angezeigt.

3. Behalten Sie die Standardeinstellung **Jeder mit dem Anwendungsschlüssel** für alle Berechtigungen bei, geben Sie _Registrierungen_ in das Feld **Tabellenname** ein, und klicken Sie dann auf die Schaltfläche "Prüfen".

	![](./media/mobile-services-create-new-push-table/mobile-create-registrations-table.png)

  Dadurch wird die Tabelle **Registrierungen** erstellt, in der die Kanal-URIs gespeichert werden, über die Pushbenachrichtigungen gesendet werden.

Als Nächstes ändern Sie die App so, dass Pushbenachrichtigungen unterstützt werden.
