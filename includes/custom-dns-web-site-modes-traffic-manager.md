Es kann nur für Web-Apps in Azure App Service im Modus **Standard** ein benutzerdefinierter Domänenname für eine Website festgelegt werden, für die mit Traffic Manager ein Lastenausgleich durchgeführt wird. Bevor Sie den Modus einer Web-App vom Modus "Kostenlos" in den Modus "Freigegeben", "Basic" oder "Standard" ändern, müssen Sie zunächst die für Ihr App Service-Abonnement geltenden Kostenbeschränkungen aufheben. 

Weitere Informationen zu App Service-Planmodi, darunter auch Informationen zum Ändern des Websitemodus, finden Sie unter [Skalieren von Websites](../articles/web-sites-scale.md).
