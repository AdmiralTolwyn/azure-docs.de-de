## <a name="intent-recognition-from-a-file-with-events"></a>Absichtserkennung aus einer Datei mit Ereignissen

Der folgende Code zeigt, wie eine Absicht in der Standardsprache (en-US) auf kontinuierliche Weise erkannt wird. Mit diesem Code kann auf zusätzliche Informationen, z.B. Zwischenergebnisse, zugegriffen werden. Die Eingabe stammt von einer Audiodatei. Das unterstützte Format ist einkanalig (mono) WAV/PCM mit einer Samplingrate von 16 kHz.
