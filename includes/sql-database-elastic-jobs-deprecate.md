> [!IMPORTANT]
> Dieser Artikel bezieht sich auf die von Kunden gehostete Version von *Aufträgen für die elastische Datenbank*. Aufträge für die elastische Datenbank werden als veraltet markiert und durch neue in Azure gehostete **Aufträge für die elastische Datenbank** ersetzt. Verwenden Sie für neue Aufträge die [aktuellen **Aufträge für die elastische Datenbank**](../articles/sql-database/elastic-jobs-overview.md). Wenn Sie derzeit die älteren vom Kunden gehosteten Aufträge verwenden, finden Sie unter [Migrieren zu den neuen Aufträgen für die elastische Datenbank](../articles/sql-database/elastic-jobs-migrate.md) Anweisungen und Migrationsskripte, um schnell auf die neueste Version zu aktualisieren.
