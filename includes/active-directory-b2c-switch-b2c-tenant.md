Vergewissern Sie sich, dass Sie das Verzeichnis verwenden, das Ihren Azure AD B2C-Mandanten enthält, indem Sie oben rechts im Azure-Portal zu diesem Verzeichnis wechseln. Wählen Sie die Abonnementinformationen aus, und klicken Sie dann auf **Verzeichnis wechseln**. 

![Wechseln zu Ihrem Azure AD B2C-Mandanten](./media/active-directory-b2c-switch-b2c-tenant/switch-directories.png)

Wählen Sie das Verzeichnis aus, das den Mandanten enthält.

![Auswählen des Verzeichnisses](./media/active-directory-b2c-switch-b2c-tenant/select-directory.png)
