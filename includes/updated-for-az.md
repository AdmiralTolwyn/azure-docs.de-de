> [!NOTE]
>
> Dieser Artikel wurde vor Kurzem aktualisiert und beinhaltet jetzt Befehle aus dem neuen Azure PowerShell Az-Modul. Dieses Modul ist das neue PowerShell-Modul für Azure, das mit den PowerShell-Versionen 5.x und 6 kompatibel ist. Für AzureRM werden auch weiterhin Updates mit kritischen Bugfixes veröffentlicht, die neuen Features sind jedoch nur im Az-Modul enthalten.
>
> * Informationen über das Az-Modul und den fortgesetzten Support für das AzureRM-Modul finden Sie unter [Einführung in das neue Azure PowerShell Az-Modul](/powershell/azure/new-azureps-module-az).
> * Das Az-Modul ist bereits für Cloud Shell verfügbar, dabei ist AzureRM-Kompatibilität automatisch aktiviert.
>   Anweisungen zum Installieren des Az-Moduls finden Sie unter [Installieren von Azure PowerShell](/powershell/azure/install-az-ps).
> * Um die AzureRM-Kompatibilitätsbefehle zu aktivieren, führen Sie nach der Installation `Enable-AzureRmAlias` aus.
>   Weitere Informationen zur Kompatibilität finden Sie unter [Migrieren von Azure PowerShell AzureRM zu Az](/powershell/azure/migrate-from-azurerm-to-az).
