| Ressource | Standardlimit |
| --- | --- |
| Storage-Kontoverwaltungsvorgänge (Lesen) |800 pro 5 Minuten |
| Storage-Kontoverwaltungsvorgänge (Schreiben) |200 pro Stunde |
| Storage-Kontoverwaltungsvorgänge (Liste) |100 pro 5 Minuten |
