## <a name="create-an-azure-sql-database-logical-server"></a>Erstellen eines logischen Azure SQL-Datenbankservers
Führen Sie im Azure-Portal folgende Schritte aus, um einen logischen Azure SQL-Datenbankserver zu erstellen.

1. Stellen Sie eine Verbindung mit dem [Azure-Portal](http://portal.azure.com)her (sofern noch keine Verbindung besteht).
2. Klicken Sie auf **Neu**, anschließend auf **Daten und Speicher**, und suchen Sie dann **SQL-Datenbank**.
3. Klicken Sie auf **SQL-Datenbank** , um das Blatt „SQL-Datenbank“ zu öffnen.
4. Geben Sie Werte für die folgenden Servereigenschaften ein:
   
   * Servername
   * Serveradministratoranmeldung
   * Kennwort
   * Abonnement (nur relevant, wenn Sie über mehrere Abonnements verfügen)
   * Ressourcengruppe (neu oder bereits vorhanden)
   * Ort
5. Klicken Sie auf **Erstellen**. Im Infobereich wird auf den Beginn der Bereitstellung hingewiesen.
6. Warten Sie, bis die Bereitstellung abgeschlossen ist, bevor Sie mit dem nächsten Schritt fortfahren.
