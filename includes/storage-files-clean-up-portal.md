Wenn Sie fertig sind, können Sie die Ressourcengruppe löschen. Dadurch werden das Speicherkonto, die Azure-Dateifreigabe und alle anderen Ressourcen gelöscht, die Sie innerhalb der Ressourcengruppe bereitgestellt haben.

1. Klicken Sie im Menü auf der linken Seite auf **Ressourcengruppen**.
2. Klicken Sie mit der rechten Maustaste auf die Ressourcengruppe, und klicken Sie anschließend auf **Ressourcengruppe löschen**. Daraufhin öffnet sich eine Seite, die Sie darüber informiert, welche Ressourcen zusammen mit der Ressourcengruppe gelöscht werden.
3. Geben Sie den Namen der Ressourcengruppe ein, und klicken Sie dann auf **Löschen**.
