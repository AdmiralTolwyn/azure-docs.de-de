## <a name="clean-up-resources"></a>Bereinigen von Ressourcen

In den vorherigen Schritten haben Sie Azure-Ressourcen in einer Ressourcengruppe erstellt. Wenn Sie diese Ressourcen benötigt, die in der Zukunft erwarten nicht, können Sie sie löschen, durch das Löschen der Ressourcengruppe.
 
1. Von Ihrer Web-app **Übersicht** Seite im Azure-Portal, wählen die **MyResourceGroup** link **Ressourcengruppe**.
2. Stellen Sie auf der Gruppenseite Ressource sicher, dass die aufgelisteten Ressourcen handelt, die Sie löschen möchten.
3. Wählen Sie **löschen**, Typ **MyResourceGroup** in das Textfeld ein, und wählen Sie dann **löschen**.
