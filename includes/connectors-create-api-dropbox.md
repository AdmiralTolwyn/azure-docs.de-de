### <a name="prerequisites"></a>Voraussetzungen
* Ein [Dropbox](https://www.Dropbox.com/)-Konto 

Bevor Sie Ihr Dropbox-Konto in einer Logik-App verwenden können, müssen Sie die Logik-App für Verbindungen mit Ihrem Dropbox-Konto autorisieren. Glücklicherweise können Sie dies ganz einfach in Ihrer Logik-App im Azure-Portal durchführen. 

Hier finden Sie die Schritte zum Autorisieren Ihrer Logik-App für Verbindungen mit Ihrem Dropbox-Konto:

1. Um eine Verbindung mit Dropbox herzustellen, wählen Sie im Logik-App-Designer in der Dropdownliste **Von Microsoft verwaltete APIs anzeigen** aus, und geben Sie dann *Dropbox* in das Suchfeld ein. Wählen Sie den zu verwendenden Trigger oder die gewünschte Aktion aus:   
   ![Dropbox: Schritt 1](./media/connectors-create-api-dropbox/dropbox-1.png)
2. Wenn Sie noch keine Verbindungen mit Dropbox hergestellt haben, werden Sie aufgefordert, Ihre Dropbox-Anmeldeinformationen anzugeben. Mit diesen Anmeldeinformationen wird Ihre Logik-App autorisiert, eine Verbindung mit Ihrem Dropbox-Konto herzustellen und auf dessen Daten zuzugreifen:  
   ![Dropbox: Schritt 2](./media/connectors-create-api-dropbox/dropbox-2.png)
3. Geben Sie Ihren Dropbox-Benutzernamen und das zugehörige Kennwort zum Autorisieren Ihrer Logik-App an:  
   ![Dropbox: Schritt 3](./media/connectors-create-api-dropbox/dropbox-3.png)   
4. Autorisieren Sie die Logik-App für die Verwendung Ihres Dropbox-Kontos:  
   ![Dropbox: Schritt 4](./media/connectors-create-api-dropbox/dropbox-4.png)
5. Beachten Sie, dass die Verbindung erstellt wurde und Sie nun mit anderen Schritten in Ihrer Logik-App fortfahren können:   
   ![Dropbox: Schritt 5](./media/connectors-create-api-dropbox/dropbox-5.png)
