### <a name="prerequisites"></a>Voraussetzungen
* Ein [Dropbox](https://www.Dropbox.com/) Konto 

Bevor Sie Ihrem Dropbox-Konto in einer Logik-app verwenden können, müssen Sie die Logik-app zur Verbindung mit Ihrem Dropbox-Konto autorisieren. Glücklicherweise können Sie problemlos aus in der Logik-app im Azure-Portal vorgehen. 

Es folgen die Schritte zum Autorisieren der Logik-app zur Verbindung mit Ihrem Dropbox-Konto:

1. Wählen Sie zum Erstellen einer Verbindungs mit Dropbox, in der Logik-app-Designer **Anzeigen von Microsoft verwalteten APIs** in der Dropdownliste aus und geben Sie dann *Dropbox* in das Suchfeld. Wählen Sie den Trigger oder die Aktion an, die Sie verwenden möchten, müssen:  
   ![Dropbox Schritt 1](./media/connectors-create-api-dropbox/dropbox-1.png)
2. Wenn Sie alle Verbindungen mit Dropbox, bevor Sie erstellt haben, werden Sie aufgefordert abrufen, die mit Ihren Dropbox-Anmeldeinformationen. Diese Anmeldeinformationen werden verwendet, um Ihre Logik-app für die Verbindung autorisieren und den Zugriff auf Daten von Ihrem Dropbox-Konto:  
   ![Dropbox Schritt 2](./media/connectors-create-api-dropbox/dropbox-2.png)
3. Geben Sie Ihren Dropbox-Benutzernamen und Kennwort zum Autorisieren der Logik-app:  
   ![Dropbox Schritt 3](./media/connectors-create-api-dropbox/dropbox-3.png)   
4. Autorisieren Sie die Logik-app auf Ihrem Dropbox-Konto verwenden:  
   ![Dropbox-Schritt 4](./media/connectors-create-api-dropbox/dropbox-4.png)
5. Beachten Sie, dass die Verbindung erstellt wurde, und Sie können nun mit den anderen Schritten in der Logik-app zu fortfahren:  
   ![Dropbox Schritt 5](./media/connectors-create-api-dropbox/dropbox-5.png)
