### <a name="azure-active-directory-authentication"></a>Azure Active Directory-Authentifizierung

Bei Azure Spatial Anchors können Anwendungen sich auch mit Azure Active Directory-Benutzertoken authentifizieren. Sie können beispielsweise AAD-Token zur Integration in ASA verwenden. Wenn ein Unternehmen Benutzer in AAD verwaltet, können Sie ein AAD-Benutzertoken im Azure Spatial Anchors SDK bereitstellen. Auf diese Weise ermöglichen Sie für Konten, die zum gleichen AAD-Mandanten gehören, eine direkte Authentifizierung beim ASA-Dienst.
