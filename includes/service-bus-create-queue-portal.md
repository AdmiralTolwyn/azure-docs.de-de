Stellen Sie sicher, dass Sie bereits einen Service Bus-Namespace erstellt haben, wie gezeigt [hier][namespace-how-to].

1. Melden Sie sich an den [Azure-Portal][azure-portal].
2. Klicken Sie im linken Navigationsbereich des Portals auf **Service Bus** (Wenn Sie nicht sehen **Service Bus**, klicken Sie auf **Weitere Dienste**).
3. Klicken Sie auf den Namespace, in dem Sie die Warteschlange erstellen möchten. In diesem Fall ist es **nstest1**.
   
    ![Erstellen einer Warteschlange][createqueue1]
4. In der **Service Bus-Namespace** Blatt select **Warteschlangen**, klicken Sie dann auf **Add Warteschlange**.
   
    ![Wählen Sie Warteschlangen][createqueue2]
5. Geben Sie die **Warteschlangenname** und lassen Sie die anderen Werte mit ihren Standardwerten.
   
    ![Wählen Sie neu][createqueue3]
6. Klicken Sie unten auf dem Blatt auf **erstellen**.
