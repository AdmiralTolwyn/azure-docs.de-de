> [AZURE.NOTE]<a name="note"></a>Sie benötigen ein Azure-Konto, um dieses Lernprogramm abzuschließen:
  >
  > + Sie können [kostenlos ein Azure-Konto erstellen](/pricing/free-trial/?WT.mc_id=A261C142F): Sie erhalten ein Guthaben, das Sie zum Ausprobieren der zahlungspflichtigen Azure-Dienste nutzen können. Sie können das Konto selbst dann behalten und die kostenlose Azure-Dienste nutzen, wenn das Guthaben aufgebraucht ist. Ihre Kreditkarte wird niemals belastet, außer bei einer expliziten Änderung Ihrer Einstellungen und der Bitte um Kreditkartenabrechnung.
  >
  > + Sie können die [MSDN-Abonnentenvorteile aktivieren](/pricing/member-offers/msdn-benefits-details/?WT.mc_id=A261C142F): Sie erhalten jeden Monat über Ihr MSDN-Abonnement Guthaben, das Sie für zahlungspflichtige Azure-Dienste nutzen können.
