> [!NOTE]
> Der Docker Swarm-Orchestrator in Azure Container Service nutzt eine ältere eigenständige Swarm-Version. Derzeit ist der integrierte [Swarm-Modus](https://docs.docker.com/engine/swarm/) (in Docker 1.12 und höher) kein unterstützter Orchestrator in Azure Container Service. Wenn Sie einen Swarm-Moduscluster in Azure bereitstellen möchten, nutzen Sie das Open Source-Projekt [ACS Engine](https://github.com/Azure/acs-engine/blob/master/docs/swarmmode.md), eine von der Community bereitgestellte [Schnellstartvorlage](https://azure.microsoft.com/resources/templates/101-acsengine-swarmmode/) oder eine Docker-Lösung im [Azure Marketplace](https://azuremarketplace.microsoft.com).
