> [!IMPORTANT]
>
> Die Verwendung dieser Azure-Funktion mit PowerShell erfordert, dass Sie das `AzureRM`-Modul installiert haben. Dies ist ein älteres Modul, das nur für Windows PowerShell 5.x verfügbar ist und kritische Updates empfängt, aber keine neuen Funktionen mehr erhält. Wenn Sie das `Az`-Modul installiert haben, müssen Sie ebenfalls `AzureRM` installieren. So installieren Sie `AzureRM`
> 
> 1. Deaktivieren Sie AzureRM-Aliase in `Az` (sofern aktiviert) mit `Disable-AzureRMAlias`. Das Ausführen dieses Befehls ohne installiertes `Az` oder mit deaktivierten Aliasen ist problemlos möglich.
> 2. Befolgen Sie die [neuesten Installationsanweisungen für AzureRM](/powershell/azure/azurerm/install-azurerm-ps). Selbst wenn Sie `AzureRM` installiert haben, aktualisieren Sie auf die neueste Version.
