## <a name="azure-cognitive-service-subscription-types"></a>Azure Cognitive Services: Abonnementtypen

> [!NOTE]
> Abonnementbesitzer können die Erstellung eines Cognitive Services-Kontos für Ressourcengruppen und Abonnements durch Anwenden von [Azure-Richtlinien](https://docs.microsoft.com/azure/governance/policy/overview#policy-definition) deaktivieren, indem die Richtliniendefinition „Nicht zulässige Ressourcentypen“ zugewiesen und **Microsoft.CognitiveServices/accounts** als Zielressourcentyp angegeben wird.

Sie können über zwei unterschiedliche Abonnements auf Azure Cognitive Services zugreifen: Ein Abonnement für mehrere Dienste oder ein Abonnement für nur einen Dienst. Mit diesen Abonnements können Sie entweder eine Verbindung mit einem einzelnen Dienst oder mit mehreren Diensten auf einmal herstellen.

### <a name="multi-service-subscription"></a>Abonnement für mehrere Dienste

>[!WARNING]
> Zum derzeitigen Zeitpunkt unterstützen diese Dienste Schlüssel für mehrere Dienste **nicht**: QnA Maker, Speech-Dienste, Custom Vision und Anomalieerkennung.

Bei einem Abonnement für mehrere Dienste für Azure Cognitive Services können Sie ein Einzelabonnement und eine Azure-Ressource für die meisten Azure Cognitive Services verwenden. Die Abrechnung für die von Ihnen genutzten Dienste wird hierbei zusammengefasst. Weitere Informationen finden Sie unter [Cognitive Services – Preise](https://azure.microsoft.com/pricing/details/cognitive-services/).

### <a name="single-service-subscription"></a>Abonnement für einen Dienst

Ein Abonnement für einen einzelnen Dienst, z. B. maschinelles Sehen oder Speech Services. Ein Abonnement für einen einzelnen Dienst ist auf diese Ressource beschränkt.
