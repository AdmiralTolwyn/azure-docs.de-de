> [!NOTE]
>
> Dieser Artikel wurde aktualisiert und befasst sich jetzt mit dem neuen [Az-Modul von Azure PowerShell](/powershell/azure/new-azureps-module-az). Für AzureRM werden auch weiterhin Updates mit kritischen Bugfixes veröffentlicht, die neuen Features sind jedoch nur im Az-Modul enthalten.
> * Das Az-Modul ist bereits für Cloud Shell verfügbar, wobei Kompatibilität mit AzureRM gegeben ist.
> * Informationen zur lokalen Installation des Az-Moduls finden Sie unter [Installieren von Azure PowerShell](/powershell/azure/install-az-ps).
> * Wenn Sie das Az-Modul lokal installieren, können Sie Kompatibilität mit AzureRM durch Ausführen von `Enable-AzureRmAlias` ermöglichen.
