### <a name="retrieve-host-name-ports-and-access-keys-using-the-azure-portal"></a>Abrufen von Hostname, Ports und Zugriffsschlüsseln mit dem Azure-Portal

Wenn Sie eine Verbindung mit einer Azure Redis Cache-Instanz herstellen, benötigen Cacheclients den Hostnamen, die Ports und einen Schlüssel für den Cache. Es kann sein, dass von einigen Clients unter etwas anderen Namen auf diese Elemente verwiesen wird. Diese Informationen können Sie im Azure-Portal abrufen.

Zum Abrufen der Zugriffsschlüssel über das [Azure-Portal](https://portal.azure.com) navigieren Sie zu Ihren Cache, und klicken Sie auf **Zugriffsschlüssel**. 

![Redis Cache-Schlüssel](media/redis-cache-access-keys/redis-cache-keys.png)

Zum Abrufen des Hostnamen und der Ports klicken Sie auf **Eigenschaften**.

![Redis Cache: Eigenschaften](media/redis-cache-access-keys/redis-cache-hostname-ports.png)
