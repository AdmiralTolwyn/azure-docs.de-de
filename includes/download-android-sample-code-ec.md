Dieses Lernprogramm basiert auf der Android-App **GetStartedWithData**. Die GUI dieser App ist identisch mit der App aus dem Schnellstart für mobile Dienste in Android, nur werden die hinzugefügten Elemente bei dieser App lokal im Speicher abgelegt. Sie werden den Code zum Persistieren der Daten im Speicher hinzufügen.

1. Laden Sie das `mobile-services-samples`-Repository von GitHub herunter, indem Sie <a href="https://github.com/RickSaling/mobile-services-samples/tree/futures" target="blank">hier</a> und dann auf **Download ZIP** klicken.
2. Entpacken Sie die heruntergeladene Datei, und notieren Sie sich den Speicherort.
3. Klicken Sie in Eclipse auf **Datei**, dann auf **Importieren**, und erweitern Sie dann **Android**. Klicken Sie auf **Existing Android Code into Workspace** und dann auf **Weiter.**
   
     ![](./media/download-android-sample-code/mobile-services-import-android-workspace.png)
4. Klicken Sie auf **Durchsuchen**, navigieren Sie zum Speicherort des erweiterten GitHub-Repositorys, den Sie zuvor notiert haben. Navigieren Sie zum `GetStartedWithData`-Verzeichnis, und klicken Sie dann auf das `Android`-Verzeichnis. Klicken Sie auf **OK**, und stellen Sie sicher, dass das `GetStartedWithData`-Projekt aktiviert ist. Falls Sie das Projekt in Ihren Arbeitsbereich kopieren möchten (empfohlen), aktivieren Sie das Kontrollkästchen **Projekte in Arbeitsbereich kopieren**. Klicken Sie abschließend auf **Fertig stellen**.
   
     ![](./media/download-android-sample-code/mobile-services-import-android-project.png)
   
    Daraufhin werden die Projektdateien in den aktuellen Arbeitsbereich importiert.

> [!NOTE]
> Wenn Sie die Beispieldatei abrufen möchten, die in der vorherigen Version des Mobile Services Android SDK verwendet wurde, können Sie sie [hier][GitHub] herunterladen.
