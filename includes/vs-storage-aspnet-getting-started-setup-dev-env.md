## <a name="set-up-the-development-environment"></a>Einrichten der Entwicklungsumgebung

Dieser Abschnitt beschreibt die Einrichtung Ihrer Entwicklungsumgebung, z. B. Erstellen einer ASP.NET MVC-app, eine Verbindung von verbundenen Diensten hinzugefügt, Hinzufügen eines Controllers und die erforderlichen Namespace-Direktiven angeben.

### <a name="create-an-aspnet-mvc-app-project"></a>Erstellen einer ASP.NET MVC-app-Projekt

1. Öffnen Sie Visual Studio.

1. Wählen Sie **File-neu -> Projekt** über das Hauptmenü

1. Auf der **neues Projekt** Dialogfeld, geben Sie die Optionen, wie in der folgenden Abbildung verdeutlicht:

    ![Erstellen von ASP.NET-Projekt](./media/vs-storage-aspnet-getting-started-setup-dev-env/vs-storage-aspnet-getting-started-setup-dev-env-1.png)

1. Wählen Sie **OK**.

1. Auf der **neues ASP.NET-Projekt** Dialogfeld, geben Sie die Optionen, wie in der folgenden Abbildung verdeutlicht:

    ![Angeben von MVC](./media/vs-storage-aspnet-getting-started-setup-dev-env/vs-storage-aspnet-getting-started-setup-dev-env-2.png)

1. Wählen Sie **OK**.

### <a name="use-connected-services-to-connect-to-an-azure-storage-account"></a>Verwenden von verbundenen Diensten zur Verbindung mit Azure Storage-Konto

1. In der **Projektmappen-Explorer**mit der rechten Maustaste auf das Projekt, und wählen Sie im Kontextmenü der **verbundenen Dienst hinzufügen ->**.

1. Auf der **verbundenen Dienst hinzufügen** wählen Sie im Dialogfeld **Azure-Speicher**, und wählen Sie dann **konfigurieren**.

    ![Dialogfeld "verbundenen Dienst"](./media/vs-storage-aspnet-getting-started-setup-dev-env/vs-storage-aspnet-getting-started-setup-dev-env-3.png)

1. Auf der **Azure-Speicher** Dialogfeld Wählen Sie die gewünschte Azure-Speicherkonto, die mit dem Sie arbeiten, und wählen Sie möchten **hinzufügen**.
