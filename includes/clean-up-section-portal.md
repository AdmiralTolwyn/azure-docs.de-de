## <a name="clean-up-resources"></a>Bereinigen von Ressourcen

In den vorherigen Schritten haben Sie Azure-Ressourcen in einer Ressourcengruppe erstellt. Wenn Sie diese Ressourcen benötigt, die in der Zukunft erwarten nicht, können Sie sie löschen, durch das Löschen der Ressourcengruppe.
 
Wählen Sie im linken Menü im Azure-Portal **Ressourcengruppen** und wählen Sie dann **MyResourceGroup**.

Stellen Sie auf der Gruppenseite Ressource sicher, dass die aufgelisteten Ressourcen handelt, die Sie löschen möchten.

Wählen Sie **löschen**, Typ **MyResourceGroup** in das Textfeld ein, und wählen Sie dann **löschen**.
