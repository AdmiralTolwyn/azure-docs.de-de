## <a name="prerequisite-install-the-azure-cli"></a>Voraussetzung: Installieren der Azure CLI

Um die Schritte in diesem Artikel ausführen zu können, [installieren Sie die Azure CLI](/cli/azure/install-azure-cli), und [melden Sie sich bei Azure an](/cli/azure/authenticate-azure-cli). 

> [!NOTE]
> Wenn Sie über kein Azure-Konto verfügen, müssen Sie eines erstellen. Sie können sich [hier](../articles/active-directory/fundamentals/sign-up-organization.md) für eine kostenlose Testversion registrieren.
> Um die Schritte ausführen zu können, muss außerdem [jq](https://stedolan.github.io/jq/) oder ein anderes Tool oder eine Bibliothek für die JSON-Analyse installiert sein.
